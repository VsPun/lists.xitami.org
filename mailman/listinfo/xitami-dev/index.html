About Xitami-Dev
English (USA)

The Xitami-Dev mailing list is for discussion of changes to the Xitami software, and the technology it uses (SFL, SMT, etc). General questions about using Xitami should go to the Xitami Users maililng list. This list has now been archived.

To see the collection of prior postings to the list, visit the Xitami-Dev Archives.
