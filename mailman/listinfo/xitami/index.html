About Xitami
English (USA)

The Xitami Users mailing list is provided for the benefit of users of the Xitami webserver. It is intended for exchanging tips on using Xitami, and discussion about getting other programs to work with Xitami. This list has now been archived.

To see the collection of prior postings to the list, visit the Xitami Archives.
