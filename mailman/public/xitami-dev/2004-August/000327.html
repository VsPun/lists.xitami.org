In the mean time I got a hold of Peter Hintjens and he pointed out that RealiBase (in fact only the 'c' script from it) is necessary to build xitami on unix/linux. Not knowing where to find the script I was initially using an ages old version of script scavenged from sfl archive I downloaded long time ago. After downloading RealiBase archive and placing the script in the path (it is in the boom directory), build was straightforward. Simply running 'sh build.sh' will build with few warnings. I was able to build and run it on both RedHat and Gentoo boxes. 

NB: I was not able to completely build RealiBase, but I don't need it either.
