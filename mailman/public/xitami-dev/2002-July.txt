Sokrates 1.6 is available (http://runner.conacom.de)

We worked hard since the last release and can provide support for the
latest Java Servlet API 2.3 (see read.me) now.

Changes since last public release 1.3


05.07.02 1.6  
Added documentation
ServletContext#getRequestDispatcher(..).
ServletContext#getNamedDispatcher(..).
Information about operating system logged.
Attribute javax.servlet.context.tempdir points now to a java.io.File
with a tempdir.
Runner#check_semaphore() is a thread now.
update to API 2.3, not implemented are methods with Java 1.2 return
values (Map, Set).
update to API 2.2.
Session-ID is configurable in the sessionmanager config now
(session_id)
SessionCookie expires with -1 (browser session) again.
removed "queued_startup" parameter. ServletInstance uses LockObject for
synchronizing .init() and .run() which allows faster startup.
Fixed race condition in ServletInstance

16.06.02 1.5  
Speedups in MonitoredExecution (caching, thread reusing). 10 times
faster now.
Event system for shutdown/dispose.
Fixed problem with various un-closed streams (Properties).

14.06.02 1.4  
RunnerClassLoader: added local path to container classpath if classpath
is empty.
getServetName() & getRemoteHost() returning RDNS name (see:
reverse_dns=1/0 in container config).
added logscn_options.
ServletInstance#processError(..): uses rootcause of error now.
