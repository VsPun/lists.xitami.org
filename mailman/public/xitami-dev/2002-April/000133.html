In message <NEEFLDLKNGPKINAMMMLJIEANCAAA.cetti@keytech.it>, "Jonathan Cetti" writes:
> just wondering if there's a plug-in so to use the Xitami web server (and
>hopefully the servlet runner) within an Java IDE such as Jbuilder 
>(my need is to debug my application ....  maybe there's another IDE 
>integrable with Xitami debugging)

Considering that Xitami is written in C, I doubt that it would run all
that well in a Java IDE...

However if you want to debug a Java LRWP application (which is my guess)
then you should be able to run Xitami in one window, quite separately,
and then start the Servlet Runner (which is Java) in the IDE.  Some
random fiddling with runtime environment may be required, particularly
if you try to use the old iMatix beta ServletRunner rather than the
newer one that Bernd Hohmann has been working on.
