I don't actually recall when that FAQ entry was written, so I don't know

routers/firewalls are involved.

>What release has addressed this issue? Does the Pro version address's the

The Pro version is built from the same source at the current Xitami
releases, but with the HTTPS support added in.  So no, if it doesn't
work in the current Xitami releases, it won't work any better in the Pro
version.
