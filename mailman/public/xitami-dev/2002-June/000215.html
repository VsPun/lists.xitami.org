> a way to run Xitami on a machine(laptop) and operate as a server
> if not hooked up to a network or in other words as a stand alone
> machine(for development purposes)?

Install Xitami.

Open a browser on the same machine, open http://localhost , there is
your site (the default Xitami sample page, which you will edit or
replace).

*some* versions of Windows are reluctant even to open "localhost"
until they check with a DNS server. Stupid, but MS never understood
the internet. But my dial-up WinNT4 machine can surf itself all
night long without trying to dial-out.
