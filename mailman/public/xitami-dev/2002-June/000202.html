Hi,

In case anyone is wondering, the cvs.imatix.co.nz pserver anonymous CVS
archive has been removed, as the internal iMatix CVS repository is no
longer organised to provide a portion via CVS and is generally in a
state of flux as has been noted here in the past.  The cvs.imatix.co.nz
CVS was somewhat broken as a result of partial updates during some of
this transition so there didn't seem any point in keeping it around.

There may (or may not) be an announcement from someone else at some
later point as to how more recent (than the packages on imatix.com)
open source software can be obtained.
