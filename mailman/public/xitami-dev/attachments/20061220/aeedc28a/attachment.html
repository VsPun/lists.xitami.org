Turn on the debug option in the administration, just as you said.<br>
Xitami will then create additional log files in a directory called
"debug"<br>
After a few web page requests, you should see the following files:<br>
<ul>
  <li>debug.log</li>
  <li>header.log</li>
  <li>request.log</li>
</ul>
There may be others if CGI's, filters, or SSI is called.&nbsp; Not sure.<br>
