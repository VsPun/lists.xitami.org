Sendmail is a program that most CGI's assume you have already installed 
on your system. CGI are usually used Linux and Unix systems. Most Win32 
coders use a scripting language, like ASP or PHP, that already have a 
mail client installed. Webservers generally do not come with an e-mail 
client.

If you’re set on learning PERL or other CGI languages I would recommend 
using AnalogX's CGIMLS in place of Sendmail. Generally CGIMLS is more 
flexible then Sendmail that is my opinion. Of course, I know if you do a 
search, you will find a DOS/Win32 port of Sendmail.

AnalogX's CGIMLS: 
http://www.analogx.com/contents/download/network/cgimls.htm
Possible source for Sendmail: http://www.satoya.cz/download/sendmail/
