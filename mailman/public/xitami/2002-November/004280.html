At 11:23 PM 11/4/2002 -0600, meijin writeth:
>As a new web admin/page designer I'd like to ask all of you out there what
tools you recommend having. By that, I am including addons for Xitami,
admin/network tools, design tools, anything. I am a firm believer of
working smarter, not harder. To do this, you need a good set of tools.
>  
>One tool I was told of is Analog to do the log file analysis...only
problem was is I never could get it to recognize the Xitami log files.
>  
>Any suggestions?

IIRC, you can either tell Xitami to use the W3C Extended logging format or
use some sort of customized format (using the 'custom' option).  If you use
the custom option, look for the proper format of log files that Analog
needs (probably W3C) on google.com or some other search engine.  The
keywords 'log file tutorial' will probably also work.

Hope this helps!
