At 05:50 AM 11/21/2002 -0000, ASHWEB writeth:
>Is there a version of Xitami that runs on Win XP Pro on laptop?
>
>I have installed version v2.4d9 that I have been using successfully  for a
>long time to run/debug cgi scripts etc on Win98SE. When I run any script all
>I get is "...server error 500...."
>
>Any help gratefully received

Depending on what the CGI does, you may need to run the web server as
another user and/or grant access to the desktop.  To do this under XP, edit
the service properties from the service control panel applet
(Start->Control Panel->Administrative Tools (only accessible if you are in
"Classic View")->Services).  You will probably have to be logged in as
Administrator to change the service properties.  Only grant the web server
additional rights and/or a different login from system if no one else's
suggestions help.  It might help to clarify what exactly your CGI
configuration looks like.

Hope this helps!
