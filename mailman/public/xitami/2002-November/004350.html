There are utilities to spoof a IP.  But there needs to be a return route which can be traced.  
Other than that the best way to do this would be to set up dual systems on separate IPs.  
Have users connect to system A.  Then use IP forwarding From system A to system B with 
Xitami server running on system B.  System B's IP masked.  Only system A's IP shows up.
