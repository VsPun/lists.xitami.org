Hello Xitami Support,

I don't know whether this is the correct email address to use, but I
could not find a better one on the Xitami website... I just wanted to
let you guys know about Spyce:

    SPYCE is a server-side engine that supports simple and
    efficient Python-based dynamic HTML generation. Those who
    like Python and are familiar with JSP, or PHP, or ASP, should have
    a look at this engine. It allows one to generate dynamic HTML
    content just as easily, using Python for the dynamic parts. Its
    modular design makes it very flexible and extensible. It can also
    be used as a command-line utility for HTML pre-processing.

Why do you care, you say? Well, a Spyce user, Jim Madsen, made Spyce
work under Xitami, and contributed the interface code. I checked it
out, and it works very nicely. It's available at:
    http://spyce.sourceforge.net/contrib/index.html

Maybe you want to put it up with all the other similar language
add-ons, like PHP and Java Servlets at:
    http://www.xitami.com/addons.htm
