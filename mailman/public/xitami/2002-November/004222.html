Hi Bob,

I discovered what you did about my own router. Accessible from the LAN side
but not the WAN. However, I can check from a different dial-up account and
it's there. There is no loopback capabilities with home routers. Of course,
we could alway get an expensive Cisco router that would work, but I use a
less expensive way to to check to the server.

I use the host file. In it, I define all the servers like this:

192.168.1.2  www.domain1.com
192.168.1.2  www.domain2.com

It works for me.
