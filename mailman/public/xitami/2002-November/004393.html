PR> My current main server is AMD K6/2 350 and runs very solid in 256M

PR> Don't use any Win9x unless you really know Win9x well. Use one of
PR> the NTs (or of course OS/2 or Unix if you wish).

PR> On NT, NTFS is marginally more stable than FAT.

PR> I agree about not running user-programs on a webserver. Often this
PR> works fine, but sometimes not.

PR> Do not expect the tray-icon version of Xitami to stay up. It does
PR> for some people, but many folks report that it crashes/locks for
PR> them. USE THE NT SERVICE version for any long-running server. I do
PR> use the tray-icon jobbie for quick-checking stuff on my home
PR> machine, but not for any server I am going to walk away from.

>> Can you switch to the service without screwing anything up

PR> I have no idea; never did that. I believe all the CFG files are
PR> identical so it "should" work. But copy your CFG files to a safe
PR> place first. The Service Installer does not always work; drop to a
PR> command line and run the exe manually to see how to Install directly.

>> I recently purchased a "blower" ... supposed to move about 12 cfm
>> of air.

PR> If the machine is truly over-heating, 12CFM is like being in the
PR> desert and fanning yourself with a matchbook. Anything less than
PR> 45-60CFM in a case is wishful thinking.

PR> First find out if the machine IS hot. Set it up exactly as it
PR> normally runs (covers on(*), jammed in a corner, cat laying against the
PR> case vents, whatever) and let it run all morning. Shut-down, rip the
PR> cover off. Keep one hand on the metal case and use the other to
PR> carefully feel around. For a long-running server, I like everything
PR> so un-hot that I could keep my hand on it forever without pain. Once
PR> you know what's hot, you can then begin to think about a fix.

PR> However if Windows is staying up while Xitami locks, it is unlikely
PR> to be CPU overheating (hot RAM is possible).

PR> (*) Some machines run cooler covers-on, some run cooler with
PR> covers-off. No easy way to guess. I have some PPro-200s in service
PR> that are rock-solid covers-on, but cover-off will get flaky in 3 or
PR> 4 hours. So test it the way it normally lives.
