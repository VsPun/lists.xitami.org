I have an update here on my Xitami crashing problem.

I have disabled APM (Advanced Power Management) on the Web Server system and
disabled the hard drives going to sleep after an hour.  Since doing that
several weeks ago, Xitami hasn't crashed since.

The reason I suspected this was because, Xitami only ever seemed to crash at
night once the system would more likely be in a lower power state.

So far, so good.
