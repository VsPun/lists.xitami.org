TITLE:
Xitami POST Request Infinite Loop Denial of Service Vulnerability

SECUNIA ADVISORY ID:
SA10275

VERIFY ADVISORY:
http://www.secunia.com/advisories/10275/

CRITICAL:
Moderately critical

IMPACT:
DoS

WHERE:
>From remote

SOFTWARE:
Xitami 2.x

DESCRIPTION:
A vulnerability has been reported in Xitami, which can be exploited by
malicious people to cause a DoS (Denial of Service).

The vulnerability is caused due to a logic error in the handling of POST
requests. This can be exploited by sending a specially crafted request,
which causes the service to enter an infinite loop and stop responding
to requests.

The vulnerability has been reported to affect version 2.5 and prior.

SOLUTION:
Tri Huynh, Sentry Union.

REPORTED BY / CREDITS:
Use another product.
