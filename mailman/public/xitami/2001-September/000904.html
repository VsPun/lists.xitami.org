enspeck writes:
>Is there a way to auto-redirect all incoming requests for name whatever.com 
>to another IP/address?

Change the DNS?

If changing the DNS isn't appropriate you either want to send redirects
(a la Apache's mod_redirect) or you want to proxy the page (a la Apache's 
mod_proxy).   Xitami won't do either by itself, but you could add a
CGI/LRWP/filter to do most of what you want (all with a LRWP) in either
case.

But if it's the whole site for whatever.com I'd definitely suggest
changing the DNS as the first choice.
