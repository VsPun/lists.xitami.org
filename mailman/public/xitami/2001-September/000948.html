and can't seem to get anything to work.

  I am testing this all locally on private LAN - then once the sites are
working properly will deploy the server to the internet with valid
addresses.  My local DNS server is set up correctly and all the PTR and A
records authenticating properly.

  Could someone who is successfully doing DNS virtual hosting please send me
their xitami.cfg and their defaults.cfg files for reference.  As well, could
you give me a snapshot of the directory structure of the Xitami root
directory.  I think I recall how the directory structure is supposed to look
when configured correctly.

  How are the virtual site directories set up.  Like example A or example B,
I believe its supposed to look like Example A (which is completely logical
because it stores all of the Virtual Sites pages, files, scripts under a
single directory branch).  But when I use the Virtual Host Wizard it is
creating the directories similar to Example B (which doesn't make sense
because its seperating the virtual sites files under multiple directory
branches).

  Example A:
