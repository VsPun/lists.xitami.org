In message <000f01c1433e$4781b240$f4068418@a2000.nl>, "Maarten Koopmans" writes:
>And what about the source kit? I may want Pro but it will have to run on
>FreeBSD or Linux.  Is the source kit known to compile on FreeBSD?

The Xitami source kit has been compiled on FreeBSD (IIRC there's even a
FreeBSD port for it), but I'm not aware of anyone who has compiled the
Xitami/Pro source on FreeBSD.  In theory it shouldn't be too tricky,
a Xitami/Pro is basically the Xitami source, plus some code to interface
to SSLeay/OpenSSL which should be very portable.  The Xitami source is
freely available, but the Xitami/Pro extensions have traditionally been
a commercial product, hence fewer people have compiled Xitami/Pro.

If you're still keen mail me off the list (ie, directly to
ewen@imatix.com) and I'll bring the issue up with someone at iMatix HQ.
