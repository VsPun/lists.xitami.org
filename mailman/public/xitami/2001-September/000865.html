Basically, follow the 'readme' file that comes with the source code.  You
may need to install additional RPMs from the RedHat CD if you installed the
"workstation" setup.  Particuarly, gcc/g++/make are not installed by
default and need to be installed to compile the source.  You need to be
logged in as root or at least have sufficient permissions to install RPMs
as well as compile and set up Xitami to run on boot.

You also need to be able to run 'gzip' and possibly 'tar' on your system,
so if the basic system utilities RPM isn't installed (don't recall what
it's called), that needs to be done first.

Hope this helps!
