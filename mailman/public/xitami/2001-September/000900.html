In message <000a01c1385d$6e9706f0$6a759e3e@ednic>, Joubin Far writes:
>Where must i implement or copy HOSTS (DNS) file for using VIRTUAL HOSTS 
>in Standalone mode.  I use W2K operating system.

I'm not certain for Windows 2000, but for Windows NT, it's under
c:\winnt\system32\drivers\etc\hosts.  I'd guess that it's going to be
somewhere simillar for Win2K.  You'll generally find a hosts.sam file in
the relevant directory; you need to copy this to "hosts" in that
directory, and then add your own entries.  (The ".sam" file is a sample
that you can copy and modify to use; but it's named with that extension
so it doesn't get used by default.)
