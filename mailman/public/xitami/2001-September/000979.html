Paul yes you can do that. They can even have their own domain name, all of
them pointing at your IP. In Xitami, you then set up virtual hosts that
will redirect all requests for 'www.website1.com' to /websites/website1
and so forth. However, what sort of connection do you have? T1? T3?
Speed is key.

I suggest: Get a host name at www.active-domain.com for $9.95/yr and
use their controls to have the host name point to your IP. Use Xitami
Virtual Hosts to set up the web page directory. Easy as pie.
