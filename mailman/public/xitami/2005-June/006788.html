At 18:08 8/06/05, you wrote:
>One proviso on this.  My experience with XP is that it is extremly slow in
>dealing with large numbers of image files on a NTFS formatted volume.
>Converting the volume to FAT-32 gets a really dramatic performance
>improvement.
>
>No numbers on this, only observation of the way Windows Explorer displays the
>volume.

XP does a lot more than just displaying the files.
For instance, it creates an alternate stream for each image file, 
containing a thumbnail of the image. FAT-32 files don't have streams, so 
there is less processing involved.
FAT-32 is a leaner file system, but it's less robust, and slower for 
directories with lots of files. NTFS suffer less performance degradation 
when files are getting fragmented.

Besides, don't use windows explorer to time directory access.
To display my 20000 files, it takes nearly 5 seconds. Probably the files 
are read and sorted before being displayed. Take also into account that for 
each file an icon must be displayed.
