Hunter Carroll wrote:

>  i have perl and i set-up the cgi thing on xitami, but it still
>  doesn't work. i use windows xp home edition

So do I with Xitami  2.5c1. Run the default htm file that came with 
xitami & then look at the cgierr.log. If you can't figure what it is 
telling you post it & someone should be able to figure out what is 
wrong. The usual problem is file paths so be sure they are correct. 
Xitami & Perl have problems with the spaces that XP allows in folder names.
