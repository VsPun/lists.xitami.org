In your xitami directory, create a file called defaults.cfg that contains  
the following:  (If you already have a defaults.cfg, just add this text to  
it)

[Virtual-hosts]
     www.mydomain1.com = "mydomain1.cfg"
     www.mydomain2.com = "mydomain2.cfg"
     www.mydomain3.com = "mydomain3.cfg"

Add as many domains as you'd like here.


Now, create each of the cfg files for the domains.  In each cfg file, you  
should have (as a minimum):

[Server]
     webpages = "path/to/your/web/pages"

Save each of these files, and restart your server.
