>From the FAQ. If xitami is installed as a service then
net stop xitami
net start xitami
works.

You usually don't need to do this but just need to have a low server
refresh time because Xitami reloads its config file absed on this
time.

i.e. in the [Server] section of defaults.cfg put
  Refresh = "1"
so that the server reloads its config every second
