May I suggest you don't put the Xitami machine behind the router.
Point it to the gateway of the isp.
Your other machines, put these behind the router.
You really only need a router if you want a private ip network.

    |
  HUB
    |     \
    |       \
    |     Xitami Web Server

    Router

     |        \
     |          \
     |            \
     |              \
