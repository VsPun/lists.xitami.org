Hello !

To those still interested: there is an upgrade available for the log viewer.
Some users complained that there were screen problems under XP.
I don't use XP, so i can't debug them.
Downscrolling (scrolling in reverse order) could be the reason why there 
were screen problems, so this feature is removed.


Some tips if you have screen problems:

1
The application runs in the windows console subsystem: don't use a dosbox 
(cmd.com) to start the application, but start if from a GUI (windows 
explorer). Dos applications expects standard screens.

2
If you still have problems, set the cols and rows to standard values: 80 
and 25.



The upgrade uses an xilog.ini file where you can configurate the settings 
(run xilog without arguments for an help screen).

Executable: <http://web.verfaillie.com/webserver/xilog.exe>
Source:     <http://web.verfaillie;com/webserver/xilog.bas>
