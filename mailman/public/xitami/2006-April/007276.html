When you do the installation of Xitami it automatically creates a folder in 
the WEBFOLDER directory called PRIVATE
This is password protected.

I have note made use of it myself as I use different techniques that are 
more or less hidden from Xitami even, but study the user manual for 
password and security settings for this folder and you will find out how to 
do it.


You should be able to define a sub folder for this folder for each 
individual user you wish to grant specific access, and simply specify in 
the defaults.cfg or xitami.cfg and related aut files how users usernames 
and passwords are related to certain sub directories.

I hope this helps you a bit on the way.
