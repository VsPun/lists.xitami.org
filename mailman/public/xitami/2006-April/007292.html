If a browser displays a PDF then it has been downloaded. There may
possibly be some Adobe PDF DRM options that stop a PDF from being
saved - I haven't looked at the Adobe DRM for a while - but in any
case this isn't a Xitami problem because the same thing happens with
any web server

If you want to (say) display only a summary page or two then you have
to use soem tools to create one or two page PDFs or text documents
that you display and then say somehting like login to download. You
cna of course also limit these logins via IP address so that not
everyone can log in.
