>I do know that for includes to work such as:  <!--#include
>virtual="sitefoot.txt"--> it is going to
>look for the file you are including in the main web page directory of your
>server (i.e.
>/webpages) and it will not show if you place the file within a subdirectory
>of /webpages where
>you may have your template.html file.
>
>If you want to include a .txt file within a scipt what I use is:
>
>sub include {
># --------------------------------------------------------
># This is text to include. Use like an SSI.
>
>my $file = shift;
>open (FILE, "<$file") or die "Can't open $file ($!)";
>print <FILE>;
>close FILE;
>}
>## Then add this line wherever you want file included:
>
>  |;
>&include ("dbside.txt");
>print qq|
>
>I use this for many scripts to include a footer and side menu within my
>displays.   Especially
>for hosts sites which don't always allow SSI.
>
