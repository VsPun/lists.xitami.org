>> I finally got xitame, php, and mysql to run on a thumb drive.  No   
>> "installation" of any files onto the host computer needed.  Just plug  
>> it  in, run a batch file to start the programs, and start browsing.
>>
>> It seems most of the trick was to make sure that all .ini files use   
>> relative addressing, so that everything works regardless of what drive   
>> letter your computer assigns to the thumb drive.  Because of this, the   
>> directory structure of the thumb drive cannot be changed, although   
>> anything under xitami/webpages/ is up to you.
>>
>> I use it to demonstrate my php/mysql pages on localhost, but I'm sure  
>> the  computer will listen and respond on whatever addresses you put in   
>> xitami.cfg.  The total size of the install is about 50 meg plus  
>> whatever  size your database is, plus whatever session variables you  
>> might have  running.  Logging of the web server is turned off.   
>> Security is not a big  concern for my purposes.
