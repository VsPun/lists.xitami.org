[Alias]Easiest way is to simply set up a new host in Admin.  For =
example:
    >Virtual Hosts
    >New
    Create virtual host profile:  Anyname
    Enter DNS host name:  sub.yourdomain.com
    Web page root directory:  c:\websites\sub

That's it.  Don't select an IP address, just take the default of 'all =
addresses'.

You will need to have your DNS provider create an 'A' record for the =
sub.domain.com and point it to your server's IP address.  If you do a =
DIG or NSLOOKUP, you would see this entry:

    sub.yourdomain.com.      86400    A    123.123.123.123

Or, if your registrar (such as directNIC.com) will do DNS for you and =
wildcard all subdomains (*.yourdomain.com), then nothing further is =
needed.   =20
