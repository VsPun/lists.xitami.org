The Xitami build process changed.  It is still a single script 
(build.sh), but you run it at the main Xitami level, not in the 
xiopen project.

There is however a small problem in the last package, the build.sh 
script appears to be badly generated.  What it should say is simply 
this:

cd sfl
sh boomake build install
cd ..
cd gsl
sh boomake build install
cd ..
cd smt
sh boomake build install
cd ..
cd studio
sh boomake build install
cd ..
cd xiadmin
sh boomake build install
cd ..
cd xiopen
sh boomake build install
cd ..
