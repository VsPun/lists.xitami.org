Daniel Nowlin wrote:

>What I am attempting is to have GD create a thumbnail image of a jpg
>picture.  Any ideas will be welcomed.
>Dan
>  
>
You don't want to use GD then- IIRC Cg has a 256 colour palette so 
photos get trashed. What I did in this case was use qx() to run 
Irfanview (www.irfanview.com) on the relevant file. You use a line like

$rc = qx(i_view32 \\$path\\$img /resample=(${percent}p,${percent}p) 
/convert=\\$path\\small\\$img);

If you are discarding the results then its pretty trivial to delete the 
file once you have sent it....
