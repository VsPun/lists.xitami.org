Hmmm...

Our only differerence is:
        ISP --- Router --- Hub --- everybody else
                                     |
                                 Server

Router gateway = ISP
Server gateway = Router
I only have 1 assigned IP

Where does the router point to?  I see a possible conflict between your
router and the server:  unless you have multiple assigned IPs, they're both
using the same one IP, which may be possible if you are MAC cloning the
server's network card.

If using a single IP, then the server is receiving it's traffic thru NAT
services performed by the router, making the topology difference moot.

As an afterthough, this may not help.  Does DSL make you use the "hosts"
file?  Or is what you refer to "hosts" is the basehost.cfg file that is part
of Virtual domaining?
