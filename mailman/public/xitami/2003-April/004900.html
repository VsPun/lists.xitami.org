During a recent discussion about Xitami crashing when run from the GUI and
not the NT Service, I decided it was time to put together a Xitami NT
Service Monitor/Controller program for NT Service users.  The resulting
program, XiMon, is now available for download at:

http://www.shininglightpro.com/search.php?searchname=XiMon

It is a particularly useful tool for developers who constantly change the
server configuration and don't want to keep an extra window open to
shutdown and restart the Xitami Web Server.

Hope this helps!
