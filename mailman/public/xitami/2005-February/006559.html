Your configuration file is (probably) default.cfg and is found as

    e:xitami\defaults.cfg

The xitami.cfg file is for reference and lots of info but STRONGLY
recommended not be fooled with.

my php installation added this:

[Filter]
.php=C:\PHP\php.exe

and

[Mime]
Php="application/x-httpd-php"

to my config file.  That's all.  See if this helps.  Add and re-start
xitami.

There may be a php.ini thing that may need to be reset (its in the windows
dir).  A redirect something or other -

  cgi.force_redirect=0

some security thing that killed php function.  Changed it and it worked - no
need to know more.

Good luck,

Charles
Server: php is 4.3.10, xitami 2.5.c2, mySQL 4.0.15 on Win98SE.
