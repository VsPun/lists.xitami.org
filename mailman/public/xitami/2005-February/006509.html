Typically you see WS-FTP connecting correctly to the server (you see the 
hello message), but fails to retrieve a directory list.
Browsers use passive connections, while WF-FTP by default uses active 
connection.
You can set WS-FTP to use passive connections.

See
http://freedns.afraid.org/forums/viewtopic.php?t=2109
with an explanation WHY
