Are you sure you edit the file "hosts" and not "hosts.sam"?
Must always be a single line for each entry.
Maybe when Windows sees the keywords "localhost" or "mycomputername" it does not look into the hosts file at all. 
You can try to put entries into hosts when you use a name different from your computer's name:
127.0.0.1  anothername
127.0.0.1  sub.anothername
127.0.0.1  more.anothername
