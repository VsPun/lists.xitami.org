I just experimented with xp home and hosts entry's and xitami virtual 
servers/folders . .. kathy.local.lan   bob.local.lan    go to their own 
named folders fine once I from the console do the vhosts Wizard and then 
follow it up with the domain names in Configurations/vHosts.

If you're using a proxy server, and/or if aol or your isp uses proxy 
servers, the hosts file in xp is probably  doing nothing.     Seems theres 
also some spyware/adware & blockers that also that can trick you into an 
unknown proxy situation in xp rendering what you see in the etc/HOSTS file 
ineffective.

If you're using IE and proxy server configs might try checking the "bypass 
proxy server for local addresses" under connections.
