your hostnames in system hosts file (located in 
\WINDOWS\system32\drivers\etc\) to recognise hosts by the operating
system?
for example:
127.0.0.1      localhost john.locahost jack.localhost


In all windows versions for localhost on lan base domain names use the 
"hosts" file. The hosts file is the lookup table that each windows pc 
uses befor it querries a dns server. Note that you might want to use the

ip address of your network card and not the loopback ip of 127.0.0.1 and

copy the setting to the hosts files of the other pc's on your lan. So 
the can view the pages you are serving. This is all so needed if you are

running an e-mail server over a lan.
sample of hosts file
127.0.0.1  localhost
127.0.0.1  mycomputername
127.0.0.1  john.mycomputername
