My 2 cents:  If it ain't broke don't fix it. 

The mailing list has worked fine for a number of years.  Everything I 
need to know comes to my email, which I go through at least daily.  
There is no need to remember to go check a bulletin board, which based 
on history, may go days or weeks without any new posts.

 The advantage of Xitami is that it is simple low maintenance server, 
which you can almost install and forget.  The mailing list is generally 
pretty quiet (as it should be), if there were a high volume of problems 
(where a bulletin board was required to meet the information needs) I 
would be looking for a different Web Server.
