Actually the problem is not a problem but the developement of php past about
version 4.1 or so. The global variables were changed. I remember a way one
site worked around this, I'll explain and hope to make sense. At the top of
the page an if statement was included which check for the presents of the
$SERVER[PHP_SELF] variable, if it was the variable was equated back to the
regular PHP_SELF variable and the page worked fine.
Hope this makes some sense and I don't really remember who's code it was on,
used to many over the past year.
