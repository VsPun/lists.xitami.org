just to let all of you know. I had a few hassles getting the xitami to run
my php setup, but i found that after i edited the defaults.cfg to enable the
filters, it wouldnt take. I then went into my xitami web browser
configuration and found that it had listed the filter as c:phpphp.exe. didnt
make sense to me, as i had edited the file as per instructions. I then
edited the filter in the filters section, with the correct path, and
everything is running well. So thank you for your help, it was greatly
appreciated.
