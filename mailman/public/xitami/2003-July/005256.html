From: "Brian Fisher" <brianf213@msn.com>
> My ISP has recently blocked inbound port 80 traffic.
> 
> Is there a way to change the HTTP port to one other than port 80?


Yes. You need to change the PORTBASE config value.
For example, if you want to move your xitami HTTP port to
9090 but keep your xitami FTP on 21, you would need the
following in your defaults.cfg:


[Server]
    portbase=9010                       #   +80 = 9090

[Ftp]
    port=-8989                             #   + 9010 = 21
