<font size="2">If Xitami refuses to let you into the browser based configuration then you have to make some changes in the authorization configuration files. 
<p>Change to c:\xitami and use Notepad to create two files: 
<ol>
<li>defaults.cfg: <pre>[Security]
    filename=defaults.aut

</pre>
<li>defaults.aut: <pre>[/Admin]
    admin=verysecret

</pre></li></li></ol>
<p>Now use userid &#39;admin&#39; and password &#39;verysecret&#39;. (From the Xitami &#39;FAQ.&#39;) 
<p>Stop Xitami and then restart it so that it reads your changes. </p></p></p></font><br>
