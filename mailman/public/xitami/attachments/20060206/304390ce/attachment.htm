Defaults.cfg is an identical copy of xitami.cfg<br><br>
Just copy and paste it in the same directory, where also your xigui32.exe
is located.<br>
That should resolve any issues, in fact the need for a defaults.cfg is
only apparent if <br>
you ever need to reinstall xitami or if you are using virtual servers
hosting and have to <br>
manage more than one hosted section.<br><br>
In all other cases it is only a recommendation.<br><br>
The same for defaults.aut, otherwise it's in xitami.aut<br><br>
