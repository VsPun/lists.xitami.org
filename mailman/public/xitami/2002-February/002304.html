FYI, after doing a lot of research and reading numerous posts from the PHP-DEV
list.

This is not a problem that is specific to Xitami. It occurs when PHP is run as
a CGI process, which is how we use PHP with Xitami.

The solution is, as Warren pointed out, to explicity set $PHP_SELF. If the PHP
scripts are written in a modular fashion this is easy, but if they aren't it
can be a pain.
