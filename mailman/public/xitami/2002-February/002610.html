In message <3C7E7F03.9485BD28@altair7.com>, "Roberto R. Morelli" writes:
>When you create a virtual host and have it completly setup, how does one 
>deligate administration of that virtual host to the site owner ?

One doesn't.  There's no concept of "virtual site ownership" in Xitami;
the system administrator owns them all.  You can set up FTP accounts so
that other people can upload files to the directories used by a given
virtual host, but that is about as close as it gets.  (With some
trickery you could let them edit the virtual host configuration files in
an editor and upload them like this, but this would probably be a
security risk too.)
