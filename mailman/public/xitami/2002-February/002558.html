first you need to download and install php
http://www.php.net/downloads.php

next open the 'defaults.cfg'  file under the 'Xitami' folder and add

[Filter]
    .php = "C:\\PHP\\php.exe"
    .php3 = "C:\\PHP\\php.exe"
    .phtml = "C:\\PHP\\php.exe"

NOTE: I'm on windows 98
