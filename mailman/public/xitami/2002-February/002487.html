Hello all,

To get rid of all the log-entries created by
http-viruses, I have created a little windows-program
that processes logfiles and removes those 'dirty'
log-entries that clogup the xitami-logfile.

You can view all removed lines and kept lines in the
program because all lines are copied in the listboxes.

You can download the program 'logfilter' at:
http://hace.dyndns.org/files/files.html

Please first read the logfilter.TXT file for further
comments.

If you have questions or suggestions for the program
please sent those to my email address.

Hope this helps people who where searching for this
kind of software and couldn't find it (like myself
;-),

Thanks,
