> I cant seem to find my cgi-bin!  Its supposed to be at
c:\xitami\cgi-bin but  its not there. ... the 'test cgi' thingie
doesnt work..

The demo CGI files that used to come with Xitami were not real
useful and might compromise security. The CGI files were removed
from the latest release; the package builder saw that the CGI-BIN
folder was empty so didn't include it at all.
  
> Can I create my own cgi-bin folder?

Absolutely.

If you have no handy CGI program to try, put this text in a file and
save it as .\cgi-bin\test.bat :

@echo Hello!

Then open http://localhost/cgi-bin/test.bat and it should say
"Hello!" at you.
