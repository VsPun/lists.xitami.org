If you uninstalled and reinstalled Xitami, go back to the folder
and make sure that the Xitami NT Service account under
which Xitami runs ... has read and write permissions to
/Xitami/ and all child folders.
Furthermore, if PHP is installed, the /PHP/ folder has to 
have the same NT account permissions as does the Xitami
folder.
