>
> > if I have two names set up at say dyndns.  one named
> > frusterated.dyndns.com, and need-school.dyndns.com can I set up
> > xitami to use a different vhost for each site?
>
> Yes.
>
> Open myserver.org/admin, click Vhost Wizard.
>
> Click NEW in top left.
>
> "Create virtual host profile:" [frusterated ].cfg
>
> " enter DNS host name:" [frusterated.dyndns.com ]
>
> "Web page root directory:" [ /frusterated  ]
>
> "Create directories if necessary?" [X]
>
> Click CREATE at bottom.
>
> Do again for "need-school.dyndns.com".
>
> Wait a few minutes. Or restart Xitami. Or restart Windows. Sometimes
> Xitami seems reluctant to pick up new Vhosts, so don't panic too
> fast.
>
> I assume you already have frusterated.dyndns.com registered at a DNS
> server such as dyndns.com . Won't work until that happens.
>
> BTW: "frusterated" is not spelled the way I find in a dictionary.
> They like "frustrated". Does not matter in URLs, and I know you are
> just throwing out an example.
