Most cable ISPs are now blocking port 80 incoming, but not FTP (21). So
you'll need to change your port base for HTTP.

BTW, if you've set the computer running Xitami as DMZ it is wide open, and
you don't need to, and shouldn't, forward any ports to it. DMZ means all
incoming requests are directed to DMZ IP address.

As the other responder mentioned, a DMZ setup is inadvisable. Better to not
use DMZ and just forward the necessary incoming ports to the Xitami
machine's IP address.
