I've changed the list to be "members posting only".  

Anyone who has been posting from an address other than the one that they
are subscribed from should either (a) use the address they're subscribed
from for posting or, (b) subscribe the other address as well in a
"nomail" mode.  (What this means is that you'll be able to send mail
from both subscribed addresses, but the list will only send out copies
of the messages to one of the addresses.  This can be done from the
webpages.)
