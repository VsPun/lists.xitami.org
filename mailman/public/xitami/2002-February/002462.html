create a file in one of your published web directories with an extension
of .php (I usually call it info.php) then place the following in it.

<?php
     phpinfo();
?>

Make sure you have set php.exe up as the filter to process files files with
a
.php extension. Then point a browser at the url of the published file & you
should be rewarded with about 4 pages of html describing your php
configuration.

Good luck, Paul
