In message <034301c1bb36$1f35eb60$020aa8c0@Master>, "Russel Olinger" writes:
>Does Xitami use any files from the IIS Admin service? or from the FTP =
>publishing or the WWW Publishing (all parts of the IIS)

No it doesn't.  

But if you have IIS installed and Xitami installed, both with their
default configuration they won't both run at once.  (Both try to be the
"default webserver" and there can be only one.)  Try ensuring that IIS
is _not_ running and then start Xitami.
