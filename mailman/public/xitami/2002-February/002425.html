In message <NFBBLAJNNACAPEPIMNLMEEMHCBAA.Fontaine@Purdue.edu>, "Matt Fontaine" writes:
>That didn't work - How can I specify to pull the text file from another web
>server?  It's not stored on this machine.  

In that case server-side-includes will not do what you want; it can
handle only files that are actually on your machine.

You'll need something to download the file from the remote machine
(fairly easy in Perl, Python, etc), and then either do that regularly
(eg, with a time-triggered program) or on demand (by using PHP or Perl
or something like that to process your pages).

You're welcome to try to change PerlSSI to do something like this,
but note that the Xitami 2.5 beta includes SSI support internally.
