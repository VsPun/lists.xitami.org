To take out spaces
I've set up alias
temp=c:\temp


In autoexec.bat I've added
set path=f:\internet\php;%path%

I didn't know about the phperror.log. I downloaded the 'big' version of
php to have a look at the php.ini file. (I initially had the windows
installer version). I found out I can set display error!


The php.ini in the windows directory is now

; php.ini for PEAR tests
include_path=..

[mail function]
SMTP= localhost ; for Win32 only
sendmail_from= me@localhost.com ; for Win32 only

[Session]
session.save_path= F:\Internet\PHP\sessiondata    ; argument passed to
save_handler

[PHP]
error_reporting= E_ALL; display all errors, warnings and notices
display_errors = On
display_startup_errors = On
log_errors = On



With a test page
http://localhost/temp/Test Page.html

http://localhost/temp/Test%20Page.html
