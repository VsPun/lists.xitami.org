On Friday 22 February 2002 03:24 am, you wrote:
> How can I run Xitami in a boot up time with init-script? My system is
> Mandrake 8.1. Xitami is installed in /var/xitami.

This is what I did for my Mandrake 8.1 system,
put this in /etc/rc.d/rc.local:

echo "Starting xitami"
/var/xitami/xitami -s -w /var/xitami
#Syntax: xitami [options...]
#Options:
#  -w directory     Working directory for server
#  -r directory     Root directory for web pages (webpages)
#  -c directory     Directory for CGI programs (cgi-bin)
#  -b portbase      Shift HTTP port 80 by this value.  E.g. to run
#                   Xitami on port 5080, do 'xitami -b 5000'
#  -f directory     Root directory for FTP files (webpages)
#  -q               Quiet mode: no messages
#  -s               Server mode: run as background job
#  -S               Console mode: run as foreground job
#  -t               Trace all socket i/o operations to log file
#  -v               Show Xitami version information
#  -h               Show summary of command-line options
