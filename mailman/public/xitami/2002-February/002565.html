I am on a dynamic ip address, using dns2go to direct domain request to the
IP of my router. The router forwards to the SimpleDNS IP which translates
the request to an internal IP address for each domain. Right now I have 10
sites on 10 ip addresses and email hosting too. I have been using SimpleDNS
the whole time with out any problems. Now if there is another way to do it
without using a DNS server that's ok too. But the question was whether or
not there is a free or shareware DNS server out there and we answered that
question. Whether or not it's the best thing to use is another question
altogether.

And for all you Linux/Unix users, some of us just have never learned it so
it's not an option.
