'PUT' requires that your client app has the capability to invoke that
request: most browsers don't. It also has MAJOR security issues.
AND you need a script at the server that actually deals with the
put request. Using the upload mechanism with 'POST' and the
'multipart' mime type is probably much easier (although it would
still require a CGI program and has security issues!).
But why bother, when Xitami has such a nice FTP server built in?
Not even sure if Xitami supports 'put' requests!

Anyway, although the following article is about Apache, it DOES
contain some noteworthy information about using 'put' requests.
http://www.apacheweek.com/features/put

Hope it helps,

Regards,
Paul Claessen
