I created a simple perl script that would allow me to stream real player files using the free player. The script is:

#!/usr/local/bin/perl

$forminfo = $ENV{'QUERY_STRING'};
$forminfo =~ s/%([0-9a-fA-F][0-9a-fA-F])/pack("C", hex($1))/eg;

print "Content-type: application/vnd.rn-realmedia\n\n";
print "$forminfo\n";

You could also use the object tag in HTML and put it into a web page that way. If you could get me more information on specifically
what type of video file it is and if you want to display it in a different program or on a web page, I'd be able to help you more.
