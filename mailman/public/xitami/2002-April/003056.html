If you don't have perl, and you have a file, for example, video.mpv, you could create another file, stream.mpv, and put
http://www.domain.com/video.mpv in it. This is basically what the perl script does. The video.mpv is the file that you want to
steam. When someone clicks on stream.mpv, Xitami automatically sends the content type, video/mpeg, and when the media player opens,
it looks at http://www.domain.com/video.mpv and redirects to that file. Hope this helps for those without perl.
-Greg

I quickly tested it and this should work:

#!/usr/local/bin/perl

$forminfo = $ENV{'QUERY_STRING'};
$forminfo =~ s/%([0-9a-fA-F][0-9a-fA-F])/pack("C", hex($1))/eg;

print "Content-type: audio/mpeg\n\n";
print "$forminfo\n";

Just call the script like this: script.pl?http://www.domain.com/audio_file.mp3. If you want a video file, just change the content
type to video/mpeg.
