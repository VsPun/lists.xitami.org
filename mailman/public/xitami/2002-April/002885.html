Quote from the FAQ:

"
50: I want several 'sites' on the same IP address, on different ports =20

You can run several copies of Xitami, each in a different directory, and =
each on a different port. Xitami is small enough that this will not slow =
down the system. In each directory, create a suitable defaults.cfg file =
with the server:portbase set to a different value.=20

"

The two keywords here are 'directory' and 'port'.
You will need to install TWO instances of Xitami,
in different directories, with a different port base,
and start the servers from their own respective directories.

Paul Claessen
