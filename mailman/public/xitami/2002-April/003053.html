Hi,

If you are going to stream QuickTime videos there is a free, open source
streaming server called Darwin available on Apple
http://www.apple.com/quicktime/products/qtss/
I remember that it needs a perl engine to run at the same time but other
than this, the only thing you need to worry is your streaming server
bandwidth. I think you don't even need xitami to run the streaming server,
it can be a totally separate, machine. Of course you need a webserver to
store the webpages with the streaming links or embeds.

Good luck
Stefan
