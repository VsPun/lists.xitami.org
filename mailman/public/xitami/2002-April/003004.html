  I was able to get Virtual Hosts working with Xitami 2.4d9 production release.

  Since I have upgraded to Xitami 2.5b5 beta release, I have been unable to get Virtual Hosts working.

  Let me know which version you have and I might be able to help you out further.

  It's a pretty straight forward process:
      Register the Domain with a registrar to be used as the Vhost DNS name
      Give the Vhost a name
      Type in your Vhost DNS name
      Point to the dir where the files are
      Click Create
      Then go into the Vhost configuration to tweak the settings, such as the default html page, error logs, security, etc.
      Restart Xitami
      Should work
