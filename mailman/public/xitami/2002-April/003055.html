I quickly tested it and this should work:

#!/usr/local/bin/perl

$forminfo = $ENV{'QUERY_STRING'};
$forminfo =~ s/%([0-9a-fA-F][0-9a-fA-F])/pack("C", hex($1))/eg;

print "Content-type: audio/mpeg\n\n";
print "$forminfo\n";

Just call the script like this: script.pl?http://www.domain.com/audio_file.mp3. If you want a video file, just change the content
type to video/mpeg.
