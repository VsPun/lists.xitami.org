I think a lot of people don't realize Xitami is Open Source. Why not put it
on sourceforge.net, and really get it open for more development. Actually I
see an entry for Xitami on SourceForge here
http://sourceforge.net/projects/imatix-xitami/ but it doesn't seem to be all
that active.

I am pretty happy with the way my Xitami install is running on my Win2K pro.
It just chuggs right along, except I don't use the FTP portion. I have been
contemplating making the jump to Xitami Pro. My one complaint is that it
doesn't do ASP. I have it set up for PHP, Perl, and Cold Fusion but have
been asked for ASP.

I have dealt with IIS and Linux in the past, but I really like Xitami for
ease in maintainance and setting up new domains.
