Hello Dario,

as I mentioned I will try the iplist.txt in a different way.
My iplist.txt looks like this and it is working perfectly fine. I use this not only for my local 
addresses even for public ones.
here we go

webmask= !xxx.xxx.xxx.xxx		# excluding this particular IP
webmask= *						# all others allowed

I find this more convenient then putting more than one ip in a single line.
