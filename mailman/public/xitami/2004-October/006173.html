Unpack the service version in the same directory as Xitami.
If you followed the rules and didn't change xitami.cfg but created a default.cfg file, the console executables and config files will be replaced with the new files. Since the actual configuration is done in default.cfg and this file is not included in the distribution, your config will not be lost.
Don't forget to stop the application before the change.
