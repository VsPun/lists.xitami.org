Bala,

Xitami does not natively run ASP pages.  What your seeing is the server 
serving the raw, unfiltered ASP page.  Only PWS and IIS natively 
supports Active Server Pages.  

There are some emulator, such as the one put out by Sun/ChilliSoft and 
Halcyonsoft.  As of right now Xitami/iMatix does not plan on building 
any native support for ASP scripting into Xitami.

Cold Fushion, PHP, Perl and several can easily be setup on Xitami. ASP 
has been an extensive topic of conversion.  I sure you could review the 
archives and see why.
