>Content-type: text/html Content-length: 376 Last-modified: Thu, 07 Feb 2002
>14:34:14 GMT

Which are the bits of the HTTP header that Xitami supplies by default.
The header files that you're trying to use are for the HTTP header (eg,
expiry dates, etc), rather than for including text in (eg) the HTML
page.

If you want to include text in a HTML page then look at SSI (server side
includes) or PHP or similar.
