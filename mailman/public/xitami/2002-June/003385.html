yea - me too. everyone always talks about the gui.... but i couldn't get
vhosts to work using it. much easier/quicker to add some quick lines to the
config file.

 for people that need vhosts, all you do is add this.... then create the
"extranet.cfg" and "intranet.cfg" file..... using your own names of
course....

[Virtual-hosts]
    extranet.mydomain.com=extranet.cfg
    intranet.mydomain.com=intranet.cfg
