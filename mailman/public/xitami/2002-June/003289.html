In message <004a01c2036a$043f4ff0$2272e2d1@reimer>, "Trent Reimer" writes:
>I'm quite new to Xitami so forgive me if this is an ignorant question, but I
>was wondering if there was a way to hook up Perl scripts (or PHP scripts) to
>a LRWP in a manner similar to Apache modules. 

It wouldn't be _too_ difficult to write something like that (a few days
work I suspect), but AFAIK no one has written anything like that so far.
Making it have the same interface as mod_perl (for Apache) would be
useful, but fairly complicated, as mod_perl is hooked into Apache at a
very low level.

There are some examples of LRWP integration with Perl, Python, etc, in
the Xitami addons directory, however they all require changes to the
scripts.  (The Perl interface at least could be better encapsulated as a
Perl module if anyone felt like doing that.)
