At 08:14 5/06/02, you wrote:
>Does not windows (and I assume Linux) have automatic cache. I would think 
>that adding more memory would help?

Too much memory for Win NT/2000 is not always good.
Windows uses the L2 cache as index table to access memory. If the L2 cache 
is too small in comparison with the available memory, the index table will 
be put in main memory.
Windows states that you need roughly 1kb of L2 cache for 1Mb of RAM.
The L2 cache is a extra memory faster than normal memory. L1 cache is the 
internal cache of the processor.
Check your program manager <CNTRL><ALT><DEL> to look how much memory is in 
use. If you need virtual memory, buy some more memory, but don't do it if 
all of your available memory is not in use on a normal load.
