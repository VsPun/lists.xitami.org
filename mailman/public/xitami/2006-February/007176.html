Did you go into Device Manager, then to the LAN card(s) ->
Properties -> Power Management and uncheck "Allow the
computer to turn off this device to save power"? I just
found that this was enabled on my machine (XP SP2, a
desktop machine), so I presume this is the setting by
default, as long as the NIC drivers support it.
And maybe it is still worth removing Norton Antivirus for a
while, just for the test. If the server is not a file
server, then the lack of antivirus software for some time
won't do it much harm.
