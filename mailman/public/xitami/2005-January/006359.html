I forgot to mention:  Some bots intentionally follow disallowed pages.  So,
make sure only pages that are actually referenced on the main site are
included.  You can test for which bots try this by including a bogus,
non-existent page in the robots.txt file (requests for the bogus page will
generate 404's - should be easy to spot in the logs and in log analysis
tools).
