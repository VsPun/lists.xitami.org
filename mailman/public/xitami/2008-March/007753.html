Jeff McDonald wrote, On 2008-03-24 17:37:

> In all operating systems, Windows,Linux, Mac Os etc. each server is 
> given a port. One server = one port per computer. This means only one 
> web server on port 80 no matter how many IP addresses you have.

Sorry, but this is completely bogus. You can have one webserver on port 
80 per IP address. Provided the software itself isn't utterly broken, 
that is, or lacks the crucial function to bind to a specific IP. Plus, 
the operating system doesn't "give out ports" to applications. Rather, 
the application "claims" a port on a specified interface (which /might/ 
be "all interfaces").
