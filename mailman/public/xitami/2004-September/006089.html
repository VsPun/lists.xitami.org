looking at the source code I think Xitami can rename only files, not dirs
(it uses rename from stdio.h).

Try to create the new directory then rename all the files using full paths
to cause xitami to move files:

ren dir1 -> dir2:

make dir2
ren dir1/file1.txt  -> dir2/file1.txt
this should move file1.txt from the folder dir1 to the folder dir2
