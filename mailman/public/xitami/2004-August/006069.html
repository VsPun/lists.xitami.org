You shouldn't change anything in xitami.cfg and xitami.aut
Create a file called default.aut and if you have to make changes to xitami make them in xitami.cfg

Regards your admin entry: As I mentioned above, create a file called default.cfg
Put the folloing in there

[/admin]
   Your Username = "Your Password"

That is all. Make sure you have a space between = and " 
Leave the xitami.aut as is. That should do the job and you should be able to open up the GUI from 
your browser.
