An easy way would be to get your server script to find out if it is =
using a secure connection, and if not, to feed your browser a simple =
page telling it to load the secure one instead.

An example of how to do this using perl is as follows:

Insert the following line into your script, somewhere before the script =
sends any info to the browser.
Immediately after any 'use' or require' statements is a good place.

If your secure port is not 443, change the number below.
#
if ($ENV{'SERVER_PORT'} ne '443') { &redirectsecure; }
#

now insert the following procedure into the script, changing =
www.mysite.com to your own address (e.g. =
https://www.thesite.com/cgi-bin/page.pl
#
print "Content-type: text/html\n\n";
print "This is the secure page";
sub redirectsecure {
print "Content-type: text/html\n\n";
print <<"EOF";
<html>
<head>
<script language=3D"JavaScript">
window.location =3D "https://www.mysite.com";
</script>
</head>
<body>
</body>
</html>
EOF
exit;
}
#
