I want to make Xitami "anonymous" to illegal requests. One very importing is
to suppress some HTTP headers, in particular "Server".

"header-dir". I tried it with "header.php" for PHP-Scripts. This file
contains only one row

If I now retrieve a PHP script from my client, the HTTP-header "Server" is
already transmitted with the value "Xitami".
 
Does anybody have experience with HTTP headers and first of all an idea how
to solve this problem?
