On 03/05/04 15:34, jp@manninghammills.org wrote:
> > 2.4d10 is a simpler product, it has three sub-projects: sfl, smt, and
> xiopen.
> What should I use instead?

If you have unpacked the source archive correctly you should find yourself 
with a top level directory containing a few files, and several subdirectories 
containing the different projects that Xitami is built from.

If you don't find these subdirectories, you may have extracted a binary 
package instead.  Or, your source archive may be corrupted.  This is the 
start of the listing of the xiopen_2_4_src.tar.gz file, for instance:

$ tar -ztf xiopen_2_4_src.tar.gz
build.bat
build.sh
sfl/version.h
sfl/prelude.h
sfl/sflhead.h
sfl/sflbits.c
sfl/sflcomp.c
...
