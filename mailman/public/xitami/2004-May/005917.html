Hi Thomas

I don't know much about headers at all....but I know that the GSL 
scripting language that is supported by Xitami has a few "default" 
scripts, and one is the default "error" script.  It appears to be a file 
that formats the error messages that are returned by Xitami.  I took a 
look at that script, and I see information regarding the server type and 
version.

Don't know if changing that will help you, but the file is called 
xitami/templates/errors.gsl.  A bit more information about it can be found 
in xitami.cfg.
