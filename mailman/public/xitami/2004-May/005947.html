If you don't already have a default.cfg file, create one (in the install 
directory of Xitami)
This file contains the modifications to the xitami.cfg file (don't modify 
this file).


In the default.cfg file, create an entry:

[Security]
     Filename=defaults.aut


Now create a defaults.aut file with different entries for each directory 
and users
(It's possible this file is already present)

[/directoryname]
     username=password
     seconduser=secondpas

[/seconddirectory]
     user=pass
...
