> You can try in http://wwwgssr.uah.es/cgí-bin/contador.exe

The first "i" in "cgi-bin" in the URL you posted isn't a plain "i" but
has an accute accent over it. You probably want a plain "i" in the URL. 

http://wwwgssr.uah.es/cgi-bin/contador.exe "works" for me, EXCEPT it
DOWNLOADED your EXE program to MY machine! I don't want it, and you
probably don't want everybody to have it. 

Re-check your cgi-bin settings. CGI programs should go in someplace like
C:\Xitami\cgi-bin, _OUTSIDE_ the Web-root folder C:\Xitami\webpages. Set
the CGI options so files in that CGI directory can only be executed, NOT
downloaded directly.
