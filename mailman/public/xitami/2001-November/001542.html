Assuming y0u have one IP address for the server.

Set up Xitami or IIS to use a different port than the default of 80.
The to connect to that server you'd use this url:

http://<name or IP of your server:<port of server>
In my case I'm doing just that until I get my site moved off of ASP and onto PHP.  I run Xitami on port 8080 so I connect to it like this:  http://63.146.67.100:8080
