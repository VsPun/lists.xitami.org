try handediting the defaults.cfg file in c:\xitami . Under the [alias] 
section add the line you want
( images_esp=c:/xitami/images_esp/") note using /es not \es!

For example I just did this
Vnc = "c:/xitami/vnc_docs/"
to test and it worked fine
