I haven't seen this posted.

Xitami v2.5b5
Norton AntiVirus v4.05
WindowsXP/98/95

If I do a network virus check using Norton, any machine on the lan 
running Xitami will often (not always) lock up.  In WindowsXP only 
Xitami has to be restarted.  The local machine running Xitami and 
Norton never locks up.

I  noticed this when I began running Norton automatically at night and 
the next morning I would have to restart servers.  Without Xitami and 
Norton both runing there is never a problem.

There have been no viruses. (Knock on silicon.)

This version of Norton is one of the last forever free network capable 
versions.  The virus definitions have to be updated manually but free is 
better than a yearly fee.
