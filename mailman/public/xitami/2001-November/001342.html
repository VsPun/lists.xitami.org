Does your file have an extension of  '.ssi', '.shtm', or
'.shtml'? If not it won't work. If it does then a statement
as below or like <!--#include virtual="rexxdate.htm"-->
should include the referenced file. The
virtual="whatever.file" is the path relative to the html
page that contains it, or an absolute path, and they are
server paths, starting from, and within server-root.

It works for me. I am running v2.5b4 on NT 4.0 SP4. Maybe
you need to upgrade to a newer version?

Good Luck!
Dennis
