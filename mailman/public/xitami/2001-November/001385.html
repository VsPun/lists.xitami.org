I use Count from http://www.muquit.com/muquit/software/Count/Count.html

This is not a text only counter, though.  It is a CGI-based counter that
dynamically creates a GIF image for the hit count. It has a lot of options
and is very configurable.  I have been using it for a while and it seems
to work pretty well with Xitami 2.5b5
