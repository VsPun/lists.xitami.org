I'm having some troubles getting Xitami 'watertight'.

There is a little thing I have found out. Xitami is returning 2
different errors when trying to get files from areas where the user
shouldn't have access to.

When the file exists, it returns:
    403 forbidden

When it doesn't exist, it returns:
    404 not found

It would be better if it just returned '403 forbidden', else I could
write a program to find out which files there are on the server.
