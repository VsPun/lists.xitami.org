In your html file there is a string

<!-- #exec cmd="http://wwwgssr.uah.es/cgi-bin/contador" -->

First feel the difference between cgi and server side includes (ssi).

If you use ssi, then this string can be like a
<!-- #exec cmd="C:/web/cgi-bin/contador.exe" -->
As you see there is a LOCAL path not remote.

And you must configure your Xitami and system to operate with ssi.
