You can place your servlets anywhere but you have to put their path into
your CLASSPATH variable. And don't forget to run servletrunner of course.

In my case, i've created an directory called java under c:\xitami\ and I
located in this place the servletrunner and a directory called "servlets".
This directory is put in the classpath; and it works.
