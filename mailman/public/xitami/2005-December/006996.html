As far as I understand, your old user name and password are gone with
your old "defaults.aut" file. You can pick another name and password
then and create a new file.

The file is a normal text file. You can create it as "defaults.txt" with
the Notepad, type in the required entry, save it and then rename it to
"defaults.aut". 

To create the entry, give a linebreak at top of the file and then copy

 [/Admin] 

and then give another line break and then type in the formula
name=password and then give another linebreak.

But check first and make sure the "defaults.aut" does not already exist
in Xitami directory. If there is such file, then prefer to open the
existing file with Notepad and type in the above-described entry.
