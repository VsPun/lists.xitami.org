On 14/08/2005 10:55 am, Marc wrote:

> When i try to deliver XML files, the brower get the message '501 - not
> implemented', even if i provide a mime content-type in default.cfg. I had
> to use a custom extension to force Xitami to display the page.

This is probably because Xitami has, by default, a WSX filter defined for XML 
files, and this filter is failing for your specific XML files.  You can 
disable this by deleting the line ".xml=xixxml" in xitami.cfg.  It is in the 
[Wsx] section.
