The FTP protocol uses two ports.
One is the control port (normally port 21).
Data and the directory listing is transmitted over a second port. How this 
happen depends on the client: normal or 'passive'.

Basically, you need to switch the passive mode ON on the client (explorer 
uses only passive transfers, other FTP clients use normal FTP, but this can 
be changed).

In normal mode the server connects to the client for data transfers and 
most firewalls will block this on the client side. In passive mode the 
client connects to the server (the normal way in client/server applications).

See http://freedns.afraid.org/forums/viewtopic.php?t=2109 for more info.
