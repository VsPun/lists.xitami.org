I'm making an assumtion here so correct me if I am wrong ...
I am assumming you already have http://domain.com working with Xitami.

You can use a virtual host to do this if you like....
As long as you are looking for it to point to the same machine.
from the administration utility select virtual hosts, new, enter a config
name, enter the url something.domain.com on the DNS entry line
and point to the respective directories to point to.
