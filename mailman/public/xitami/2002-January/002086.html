No I don't think so (it's a PHP thing) -- but there are similar routines
in CGI.pm that work quite well.

As you noted in an earlier mesage, the main thing is that to read binary
files you'll have to tell Perl to access stdin in binary mode (there is
a function to do this).  Then it should work fine.  Although the CGI.pm
interface is easier -- just have a look through the CGI.pm documentation
for details on handling file uploads.
