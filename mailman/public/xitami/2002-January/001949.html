Assuming you are on a windows platform (I think so) then the only thing 
you need is Perl itself. I use activestate perl latest whatever version 
and xitami on both Linux and windows and it all works wonderfully.

Occasionally you will find a perl script that barfs, generally because 
it has an expectation of Unix/Linux and/or Apache (e.g. I saw one that 
tried to look for .htaccess files), but they are rare and generally 
there are alternatives that aren't so badly written.
