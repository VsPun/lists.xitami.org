At 08:20 PM 1/16/2002 -0500, Ron Ackerman writeth:
>I currently have 11 domains running and I am trying to set up a webcam using 
>WebCam2000. WebCam 2000 can be set up for any port and is currently using
port 
>8080. Xitami is set up for default port 80 and needs to stay there. I can
set up 
>the DNS server for webcam.<domain>.com. How should I set up Xitami to
alias port 
>8080 for that subdomain. Any ideas?

Well, you could always use a script to redirect to port 8080.  Here's a
quickie PHP script that will do the trick:

<?
  header("Location: http://webcam.<domain>.com:8080/");
?>


Hope this helps!
