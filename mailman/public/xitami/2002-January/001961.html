Basicly, xitami can run with any scripting language you want so comes
with all options. The catch is: you have to install the tool of your
choice yourself.

For example for the scripting language PHP (which you can get for free
at  www.php.net), you would just need to explain xitami to run php.exe
with all files which end with '.php', but only after you installed the
php.exe, of course...

Installing computer(scripting)languages is out of the scope of the
xitami manual. Refer to the computerlanguage of your choice to find
out how to install it on your system.
