Are you sure you're using php3 ??? php3 has been out of date for a LONG time
now; if so maybe you should download & install php4 (php.exe & friends) from
www.php.net. Also make sure that your system PATH variable contains the path to
the php.exe, perferably at the very beginning as Xitami seems to occasionally
have difficulty locating the main php dll if it is not on the path.

Paul
