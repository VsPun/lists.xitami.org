I'm not a specialist, but this is how I'm doing this. I currently host about
10 domains on a single IP using the xitami web server.

First of all you need to "point" you domain to the "209.5.45.142" IP in your
nameserver, or get your ISP to do this for you.

Then I use the xitami web interface to add a virtual host (click the
"virtual host" button and select "new")

1. Enter a name for the ".cfg" file at the top (any name)
2. Select "All addresses" in the IP Address field.
3. Enter your domain name in the DNS field
4. Enter the full path of the location of the web pages.
5. Click the "create" button at the bottom
6. Restart the web server

Finished!

hope this helps!!
