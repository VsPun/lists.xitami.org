I've been this route, tho at the time I using TinyWeb, had not learned of 
Xitami yet.  I would never try to do this again, it didn't work.

People didn't understand that they needed this web server running.  Some 
programs thought that it was a virus, or a Back-Orfice kind of spy program.

Some one else complained about the resources that the server where being 
used.  Even when all of the software was removed they STILL complained that 
it was the web servers fault that their resources where low.

I ended up writing my own "browser" that would read the HTML files, using 
the wxWindows tool kit, http://www.wxwindows.org/ .  Every one seems happy 
with that.
