You can make a cd rom that start window internet explorer automatically and 
loads an index page. (TCP/IP is not needed since only local files are accessed)
 From there on the users can navigate the CD like a normal internet site.
It will not be possible to execute programs, but IE has a FIND function.
There are also javascripts commands that let you select records in a file. 
Normally these commands are not used in an internet environment, since the 
whole file has to be downloaded anyway, but they function well for local 
browsing.

I regularly distribute CD-ROMS that way, and i never had complains that the 
files could not be read.
Of course the CD-ROM will only be accessible by windows PC's.

Simply make a "autorun.inf" file in the root of the CD with following content:

[autorun]
open=start index.htm

(assuming index.htm is also in the root of the CD)
