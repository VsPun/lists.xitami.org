I program Visual Basic and Macromedia Shockwave Director. 
I used to use ASP for stuff. 
Now I'm doing more CGI for communication when I want to read the server.
For posting to the server (from Shockwave)  ... I send an email to the server with the data encrypted.
A robot on my mail server unencrypts the email and posts data accordingly.
I like this because I can buffer a lot of heavy traffic on the email server,
and take my time doing the post. Plus I know I can't get hacked.
The robot has permissions into the database, but the email server can't see it.
