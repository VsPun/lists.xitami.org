> Does Xitami have ALL necessary software built into it to handle CGI scripts?

CGI is just a thin specification for an interface between a webserver and an
external program. It does NOT specify what that external program IS, so the
answer to your question is "No". 

I have CGI scripts written in DOS BAT file, and Visual Basic. unix shell
scripting is popular, but only on unix machines. There's also REXX and AWK and
one that sounds like a snake. 

The most popular tool for scripting today must be Perl, which as Francis said
you'll have to get and install. (Perl is much bigger and more often updated
than Xitami so it makes no sense to include Perl in an Xitami download.)
