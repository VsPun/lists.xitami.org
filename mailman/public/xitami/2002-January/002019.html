> Anybody know if Xifusion works to allow ColdFusion Server Pro and 
> Xitami to
> talk? 

Yep. You need to add it as a filter in xitami.cfg

Looks like this:

#  
-------------------------------------------------------------------------
_
#   FILTER configuration section                                         
    _
#                                                                        
    _
#   This section lets you define filter programs. To define a filter,    
    _
#   specify a file extension with a leading dot and the filter           
    _
#   command. The command may be with a full path, or without, if the     
    _
#   program is on the PATH.                                              
    _
                                                                         
    _
[Filter]                                                                 
    _
                                                                         
    _
.cfm=c:\cfusion\bin\XiFusion c:\cfusion\bin\cfml.exe                     
    _
