	Now, back to your PHP header problem below, the header line has to be the 1st

> 	You can do redirects with regular html, you don't need PHP.
>
> 	If you save this file to index.html:
>
>
> <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">
> <HTML><HEAD>
> <META content="text/html; charset=unicode" http-equiv=Content-Type>
> <META content="MSHTML 5.00.2614.3500" name=GENERATOR></HEAD>
> <BODY onload="window.location='index.php'"></BODY></HTML>
>
>
> 	and visit it with a browser, it will try to redirect to
> 'index.php' on the
> same
> host. Just put what you need in there, instead of 'index.php'.
>
