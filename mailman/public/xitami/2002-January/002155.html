The percent sign indicates the next 2 characters are for the hexidecimal
value of the character in the URL. For instance if you find a %20 it means
there is a space in the URL or one of the files you are pointing to....
Remove the space from the filename and it will fix it. However in your case
%2E is a period .... Your browser should have no problems interpreting this.
I don't believe this is a problem with xitami...
Just a shot in the dark, but check to see if pdf is a valid mime type in
xitami.. if not try adding it. see if this resolves the problem.
