Web browsers make more than 1 connection, regardless of the type of browser.
You can control this if you make changes to the registry. You can have as
little as 2 or up to 20 connections. More connections with a fast Internet
connection means pages load really fast. The difference is not that
noticeable with dialup.

I believe X-Tech Setup (a tweak program) gives you the ability to change the
settings.

Anyway, the counter counts http connections, not internet connections... if
that makes sense.
