You can technically use either the NT or the Win9x version on an XP system.
But, there is one thing you need to know...XP and the Xitami FTP functions
don't work well together. You're going to want to find a separate FTP Server
software to make up for it. I had good luck with
http://www.pablovandermeer.nl/ftp_server.html It's small, it's free, it's
easy to use and it comes with the source code.

Hope that helps!
