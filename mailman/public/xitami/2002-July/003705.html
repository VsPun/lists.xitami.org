Angelo Eatrides wrote:

>Hi all. 
>
>I need help with the xitamipro (evaluation version). 
>I need to assign a password protected directory on
>a virtual domain but I don't know how to define
>the directory in the defaults.aut. 
>
>I know [Private] will point to the private directory of
>the "webpages" dir but the dir I want to protect is above it
>off the main xitami root. 
>
>For example:  Xitami/example/private/
>  
>

You need to define an alias first and then password protetc the alias

i.e. in defualts.cgf have a line under alias called

Private = "C:/xitami/example/private"

And then in defaults.aut have (webmask is optional)

[Prvate]
    Userid = "password"
    Webmask = "ip.add.re.ss"
