Don't believe everything, sometimes not anything, a GUI shows you.
Just because a file is tagged executable does not mean that just
running it is going to do any good. Try running `more'. Utterly
useless until you give it something to do.

> how do I "run" Xitami

Open a unix command window (some call it a shell prompt). No, do not
ask me where a GUI unix hides the command line. Sometimes you have
to telnet to itself. Then issue these commands. Remember that unix
is case-fussy!

 cd /usr/xitami-24d9
 Xitami -h

See what it says. You probably have to start it with command line
switches to tell it what to do (which is why clicking pictures isn't
going to do any good).
