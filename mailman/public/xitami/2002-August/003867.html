I've used BIND on Windows NT and even Windows 98. Setup was 
somewhat involved, but it worked better than the DNS Server that 
comes with NT. One reason I tried it was that we had customers 
who couldn't access certain websites, yet they could when 
connecting to another ISP. Turns out it was a combination of poorly 
set up DNS zones and the poor operation of NT's queries. 
Apparently it wouldn't query secondary name servers like BIND 
would. I can tell you, however, that there is a distribution of BIND 
for Windows NT.
