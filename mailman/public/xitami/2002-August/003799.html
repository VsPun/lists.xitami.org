> I would like to find a good CGI script to create "Forums"/message
> board for my users to discuss art.

I have used WWWBoard from Matt's Script Archives successfully under BSD and
Linux.  It requires Perl to run.  It sets up a fairly nice little forum, and
you can customize it so that the forum interface is consistent with the rest
of your site.  Out of the box, it isn't moderated, but I think that can be
done with a little fiddling.

It is, of course, free (the price is certainly right), and you can find it
at:  http://www.worldwidemart.com/scripts.  This is actually an older link I
haven't visited in some time; if it doesn't work for you, let me know
privately, and I'll e-mail it to you.

Hope this is useful!
