> However the "mail" command in a PHP file always fails.

unix machines traditionally have a full mail server. Windows
machines traditionally do not. If they do, it is used so differently
than unix mail that it isn't going to work the way unix-oriented
scripts and script handlers expect.

I was shocked when a perl script on NT4 found my copy of
Communigate Pro (which has a unix-like interface) and sent mail.
Mostly I do NOT expect unix-mail calls to work under Windows,
without extensive script-hacking.
