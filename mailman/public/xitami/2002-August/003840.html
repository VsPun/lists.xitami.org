At 11:39 PM 8/16/2002 -0500, Neco Computers writeth:
>Can anyone tell me how to get xitami and MySQL server downloaded from
>http://www.mysql.com to work? I have downloaded version:
>mysql-3.23.52-win.zip and i am running Win2k.

What exactly are you trying to accomplish with Xitami and MySQL?  If you
want to do script-based stuff, try Nuclear Vision, PHP, ColdFusion, Perl,
or a number of others.  If you want to do something like database logs,
Xitami doesn't have that type of support.  You will have better luck with
Apache or ProtoNova or some other such server (IIS, maybe via ISAPI?).

If you clarify your intentions, then this list can help you out much
better.  A web server generally doesn't magically just know how to talk to
a database.

Hope this helps!
