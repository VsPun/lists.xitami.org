XiFusion v2.5 (a link to it is now available from the Xitami add-ons page)
will solve your problem.  It will handle both relative and absolute page
redirection (the latter is sent back to the browser for processing and
forwarding).  While XiFusion v2.5 is designed primarily for interfacing
ColdFusion with Xitami, it provides an excellent wrapper for PHP (v4.x) as
well.
