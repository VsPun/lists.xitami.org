Where ever you got your domain name (www.MyName.com) .. they should have a facility to 
point to a DNS server. They may have their own DNS server IP's as default. I use my SDSL ISP domain.  
I paid for 8 static IP's.
At Register.Com (where I got my domain name), I point to my IP domain name server and I point my Static IP address on one of my boxes static IP here as my web site. 
The router assigns internal IP addresses, as does my ISP. The ISP maps Internal IP's to External IP's, so that TCP/IP can travel on a private network on my side of the firewall. (We have 3 pc's here).
The outside world can get to my machine by my desk via external IP address or domain name.
Do get a firewall; better to run NT. WinXX is unsafe in the real world. WinXX must be behind a firewall. It can't be in the DMZ or you may get hacked.
Get rid of Windows Networking. Only use TCP/IP. NT ships "Wide Open". NT must be locked down. Win98 is wide open and can't be locked down.  If this bumms you out, call M$.
There are some good books out there on NT and TCP/IP and Firewalls. Very interesting reading.
Good Luck. Hope This helps.

> The First thing you will need to find out is if your DSL provider provides
> you with a static or dynamic IP address. If you have a dynamic address, than
> you will need to subscribe to a domain redirection service (bn3 is pretty
> good). If you have a static IP address you have the option of running your
> own DNS server (but with WinME stability could be an issue). If you choose
> to run a DNS server, I suggest you buy a book on setting up a windows DNS
> server as the process can be complicated and involve some steps that will
> not be that easy for someone new to hosting web sites. Personally, if you
> are new to this, I would go with a free redirection service like bn3 anyway,
> especially if you have a static IP address. With bn3 you can redirect
> http://www.yourdomain.com to your IP address and it will work great. If you
> have a Dynamic IP address, you really don't have a choice other than to go
> with a redirection service like bn3. If you would like to check out bn3's
> offering go to http://www.bn3.com
