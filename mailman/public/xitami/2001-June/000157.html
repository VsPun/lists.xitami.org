Note that this requires additional DNS entries, whereas
www.dummy.com/JohnSmith/ only requires that www.dummy.com be registered in
DNS.   
If you manage your own DNS, no problem. And the username entries can be CNAME
pointers to the one IP entry.

If you buy DNS service from somewhere else, adding hostnames to DNS may cost
you money.
