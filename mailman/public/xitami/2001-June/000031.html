In message <000601c0eb94$151bbb40$4db8d1d8@daniel>, "Daniel" writes:
>Is is possible to have 2 ftp servers to run at the same time and have Xitami
>to run HTTP service on port 80 and FTP service to run on another port than
>21?

Yes.  Change the FTP port setting:

[FTP]
  port=NNN

in your defaults.cfg.
