set up a vhost to the intranet ip or intranet host name to point to the
intranet site....then go to the xitami directory, you should see .cfg and
.aut files for the stuff you just created...you can then manually edit the
vhost's .aut (usually the name you gave when you configured it, so if you
made it use local.cfg..you'd edit local.aut) and it's really simple..you can
look at aut files....or in the webadmin..change defualts.aut to whatever the
filename is and edit that way.
