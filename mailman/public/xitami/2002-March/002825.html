This is actually a Mercury setup problem not Xitami, but since i'm also 
using mercury:

Goto Configuration, Core Module and change the name of the primary domain 
(tab general, internet name for this system).
Don't forget to enter all other domain names in Local domains, or the 
server will try to mail out messages destined for some local domain names 
and refuse some incoming mails.

Most important: UNCHECK "VALIDATION OF FROM FIELD WHEN PROCESSING MAIL" in 
Config, Core module, General.


You can also change your UA (user agent, that is your mail program) to use 
a different account whith the short domain name.
Normally mercury won't change From fields.
