	A fresh install of the PHP  InstallShield should configure it all right, except
the security problem.

	Then you go to c:\winnt\PHP.ini (your windows dir) and declare this:

cgi.force_redirect = 0

	and that should work. In fact, you must have this value already in the INI,
commented out. In this case it is using the default value of 1, but you want 0.

	From this point on, you're on your own.
