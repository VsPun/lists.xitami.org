    I think you're coming at this from a very difficult point of view.
You're trying to learn how a 'specific' application handles a situation with
out first understanding how 'any' application in general works.  Its great
to start the learning process, and Xiami is as good a place to begin as any,
but you need to learn the basic first before worrying about the specifics.
Get some good books on how Internet servers work, especially HTTP (i.e. Web
servers) and on CGI.  Once you understand these things, then learn how to
use your specific tool to implement them.
