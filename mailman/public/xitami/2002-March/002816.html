ites:
>problem - i have and intranet for the company. currently it is open to the
>internet. i want all employees to be able to access it. the problem is that
>i want to restrict public access to the site by using passwords. people in
>the office should be able to access the site as normal (no password prompt),
>but people accessing it from the internet should be prompted before access
>is given. [.....]
>any more ideas or help?

Configure two virtual hosts.  Configure one accessible from anywhere,
and one accessible only from the Intranet (by IP address ranges).  For
the one accessible from anywhere, configure it to require a password for
access.

Then tell the Intranet people to use the internal host name, and the
others to use the external host name.
