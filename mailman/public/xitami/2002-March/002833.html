I am including my default.cfg

Xitami uses two files xitami.cfg for main configuration and any differences
will go into the default.cfg

I think the reason is that you can create multiple configurations without
copying the whole thing.

make sure php is working.  Go to your command line and type 

php -v

that should show you the version.

That should be it.

All you need then is go to your /xitami/webpages directory
and put your test.php file(remember to put .php)
