27: How do I use a CGI program as a default page?
    Add the index.cgi file to the list of default pages.
    Make sure that index.cgi is 'executable'.
    Define the server:cgi-url as "/".
    Define the server:cgi-bin directory to be same as server:webpages.
    Define cgi:mixed-url to be 1.
    Define cgi:dirlist to be 1 if you want directory listings to be enabled.
