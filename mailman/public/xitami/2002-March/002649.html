Just wrote:
> 
> Question: Is there a way to name it to something other than localhost?

Create a hosts file and add entries for your machine(s).

> Question: If I want to install another Xitami in say PC#2 for this intranet,
> and leave PC#1's Xitami for all external calls?  how should I do it so it
> won't fight with PC#1's Xitami

Setup Xitami on PC#2 just like you did on PC#1. Since you won't be port
forwarding to it, the outside will never see it. Make sure there is a name for
PC#2 in your hosts file, use that name in the URL when trying to access the
local web-server.
