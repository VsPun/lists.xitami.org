kenjoy@kenjoy.com has been unsubscribed from the Xitami mailing list
because of this persistent vacation message being submitted to the
mailing list.  At present they can still resubscribe when they return;
but if it happens again they'll be banned from the mailing list.

Please folks, if you're going to configure a vacation message:
- make sure that it doesn't go to the mailing lists you are on (or 
  unsubscribe from the mailing lists when you go away, or just set them
  to "no mail" on the webpage)
- make sure that it only sends one message to each destination in a
  long period of time (eg, once a week or something)
- make very sure it doesn't respond to messages it generated itself,
  however they get back to it.
