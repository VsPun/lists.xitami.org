writes:
><?php header ("Location: test.htm")  ?>           doesn't work

This is a server-side redirect.  In theory it might work if the document
is in the root directory; otherwise it won't work with Xitami.  Try
/test.htm, or picking the document path of the existing document and
putting that before the file name you want.

><?php header ("Location: http://127.0.0.1/test.htm")  ?>           is ok

This is a client-side redirect and will "always" work, because the web
browser makes a new request.
