>I am looking through the Xitami SDK files.  I cant find any of them in my
>Xitami installation.....not even the .gif image files.
>Where does Xitami store the configuration files when it installs itself???

They're all compiled into the binary.  This is part of the reason it's
as big as it is.  The wsxrc program turns the GIF images into an include
file (xiadmin.h).

>If I wanted to replace a button, where would I go??

You need to change the button (from the SDK source), and then run 
wsxrc @xiadmin.rc, and then recompile Xitami.  If you've added images,
you need to edit xiadmin.rc (a text file, one image listed per line) and
add it to the end of that list.
