In message <DNEFIPOGFKEIMMJIHKLFKEOCCHAA.tinbb@hkplanet.com>, "alex" writes:
>    Can i install Xitami in my client machine without using the installer
>provided by the iMatrix ?

The installer is just an archive (I forget which format the embedded
file is, but I think it's a fairly common archive format), with a
self-extract executable bit at the front, and a tidy bit of "change this
to this" substitution going on in some of the installed files (eg, it
will set up a basic defaults.aut/defaults.cfg).

If you really don't want to use the installer you could either build
Xitami from source, or figure out how to pull the installer package
apart and manually get at the files.

Since the installer just puts stuff in a directory for you, and
configures a few files in that directory, I'm not quite sure why you'd
want to go to the trouble of avoiding using it if you don't want to
build from source.
