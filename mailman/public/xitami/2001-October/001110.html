In message <5.1.0.14.0.20011003004656.02673c60@pop.mail.yahoo.com>, Ketil Thorgersen writes:
>Is not chmod implemented in the xitami ftp server? When I try, I get a "502 
>- site not implemented" error. Whatever that means?

No, chmod isn't implemented.  IIRC it's done with a special "site"
command (for site specific extensions I think), and Xitami doesn't
implement any of those.  

Basically if you need a more fully featured FTP server you're probably
better using another one that comes with Linux, rather than the fairly
simple one in Xitami.
