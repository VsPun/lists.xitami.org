Zdenko Roh·Ë wrote:

> Hello.
> I have found out that you need append lines into defaults.cfg or
> to another apropriate config file (into Mime section):
>
> [Mime]
>      Wbmp=image/vnd.wap.wbmp
>      Wml=text/vnd.wap.wml
>      Wmlc=application/vnd.wap.wmlc
>      Wmls=text/vnd.wap.wmlscript
>      Wmlsc=application/vnd.wap.wmlscriptc
>
> Worked fine for me.
>                                  Bye
>                                          Zdenko Rohac.

Thank's, now it's working for me, too.

There is one (possible) bug in Xitami, as you have to define Mime-settings in
your virtual host's configuration file. It is not enough to define it just in
the default configuration file.
