In message <B7E0A3E9.248%yancy@print-imaging.com>, Yancy Shepherd writes:
>Hello all, I've got a little problem with my cgi. I'm tyring to use FormMail
>to submit forms vial the web. (I've used it on other xitami servers with no
>problems) When I post the form I get an error msg. saying the website cant
>be found. The url reads. http://site.com/cgi-bin/formmail.pl

Was that an example URL that is different from what is in the HTML?  Or
does it really say "site.com"?  In which case is your domain name
"site.com"?

Also if you're using IE, make sure you turn off the "friendly" error
messages, so that you can see what the webserver is really saying.
Debugging problems with the IE "friendly" error messages is much much
harder.  (Instead of getting a message telling you what went wrong, you
get a message saying basically "nope, that didn't work, guess again".)
