Well, I found Opera and Netscape were fine! For me IE is the really slow
one.
As for your Windows NT error, I'm not sure what this is but it sounds most
likely that it is trying to find Perl.exe and not succeeding. Have you
checked that the location of Perl.exe has been appended to the PATH
environment variable? To find out, simply go to the DOS prompt and type
PATH. In the list of paths you should see the location of perl.exe. If not,
this is probably your problem - although I'm not 100%.
I don't use NT currently, but I did before and it was OK.
