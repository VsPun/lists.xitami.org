Mihaela,

It sounds like you are having a CLASSPATH problem.  Make sure that your
servlet class is the classpath.  What I usually do is package my servlets in
a jar file and then use the "java -classpath" option to specify my jar file
when I launch the servletrunner.  I also do not use the .class extension in
my servlet.properties file.  Hopefully this helps.

~Nate Johnson
