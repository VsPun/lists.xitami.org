> Does a basic Hello World! perl script work through a browser? 
> 
>  #! fee/fie/perl
>  print "Hello World";

To be a real CGI script (i.e. one that will work) I believe that's

---- begin
#!/usr/bin/perl

# for Windows + ActivePerl the above should work anyway;
# if not, use #!c:/perl/bin/perl or whatever

print "Content-Type:text/plain\n\n";
print "Hello, World!\n";
---- end

since you need at least a Content-Type header and a blank line (so two 
consecutive newlines) before the content starts.
