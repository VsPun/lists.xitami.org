>HOW DO I GET THE CHANGES TO APPLY TO MY CURRENT INSTALLATION???

The basic idea is:
- make changes to the FDL file with a text editor
- run fdlgen xiadmin.fdl
- run fmlgen over each xiadm??.fml file that is generated
- rebuild Xitami from source (including your newly generated files)

You'll need to tools from the Xitami SDK and also a C compiler.  iMatix
uses Microsoft Visual C++ 6.
