The documentation for "Configuration" and "Setting-UP A Virtual Host" plus actually going through the motions of using the Xitami WBA was just the TICKET!!!! The terminology was straight forward and the required items were set.

Used WBA SETUP Virtual Host with NEW and set what I thought were the proper settings to isolate these DNS names to their respective directories for HTM, CGI, etc. files.

On the first try it did not work because I thought I had keyed in the following for the DNS names:

      www.abc.com
      www.def.com


When I took a look at the actual .cfg file with a text editor for these DNS names the following was listed:

     Www.abc.com
     Www.def.com

These were reset to www.abc.com and www.def.com and all worked as expected.

Maybe I did not wait long enough for the server to re-read the new cfg file(????).

Also, using WBA I could not find a selection that would allow resetting of the DNS names.
The text editor was used to actually modify the DNS names.
