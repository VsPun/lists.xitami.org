One of my client side applications downloads an xml file from the webserver.
However I was finding that it was failing. So I tried to it manually in a
browser.

example http://www.example.com/example.xml

and got:

Error 501 - The function you requested is not implemented 

After a bit of an investigation I found that I could fix this fault by
commenting out the line:

 .xml=xixxml

in the [Wsx] section of xitami.cfg

Everything seems to be fine now, but I'm concerned I've removed important
functionality from Xitami by doing so. Can anyone advise please?
