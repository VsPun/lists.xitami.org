Ok I think htis is a 2.5 feature not a 2.4 feature - no doubt someone 
will jump in and correct me if I am wrong - so if you are running 2.4.x 
none of this applies.

However if you are running 2.5.x then you should have a templates 
directory under you xiami directory. In that you will see a number of 
files which are GSL scripts that run when you do certain things such as 
try to look in a directory with no default html page (you can configure 
this at the bottom of the server config page in the wbe admin tool).

GSL scripts allow you to create anything you want practically for a 
directory listing and if you search back in the archives 
(http://lists.xitami.org/pipermail/xitami/) for messages from me 
containing the word template then you will find a number that talk about 
soem things I have done to customise them.

Hope that helps
