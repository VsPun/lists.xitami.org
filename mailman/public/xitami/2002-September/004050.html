Hello Ingo,

 While I'm not certain what you mean by "virtual servers", I am going
 to point you at the "virtual hosts" section of the help files in
 <Xitami Root>/webserver/xitami web pages.

 If I understand you correctly though, you are looking to install a
 server (web, mail, news or other) on a virtual host. This is not
 thoutoughly covered in the Xitami manuals, but you may get enough
 information to get you started from the Virtual Host section.
 Basically, you create a virtual host and a virtual cgi-bin directory
 in Xitami and then put your server in the cgi-bin directory.

 Hope this helps.
