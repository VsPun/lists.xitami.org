Use a workaround if nothing else helps.
Put the following in some ever available header or config file:

function scriptname($buffer) {
         $buff_array = explode("/", $buffer);
         $i = count($buff_array);
         $i--;
         $scriptname = $buff_array[$i];
         return $scriptname;
}

$php_self = scriptname($SCRIPT_NAME);

tony
