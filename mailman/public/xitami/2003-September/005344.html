NOTE FOR WINDOWS USERS: What was xi win32.exes (the actual server) is 
xigui32.exe in the 2.4.d10 release just posted.  This is change from the 
2.4.d7 version I was running and maybe other versions.  Since I start 
the server from a desktop icon all I had to do was change the shortcut, 
others may have to do something additional.

Also, neither of the 2.4 or 2.5 zip files released today appear to have 
the installer included.  A major reason that I'm running Xitami is the 
installer had it up and running in under 10 minutes for a newbie to web 
servers.  Without the installer features it is hard to recommend Xitami 
to other newbies.
