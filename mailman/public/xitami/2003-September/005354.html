"Daniel Nowlin" <nowlind@missouri.edu> wrote:

> Anybody know how I should update my 2.5 version to the new version?  I
> am running it on 2000 server as a service and want to keep it that
> way. 

In the install package you will find three binaries, the service 
version is called xisrv32.exe.  Place this file in your Xitami 
directory, then register it this as a service from a command prompt:

  xisrv -i

Stop the old service, start the new one, test that it works.
