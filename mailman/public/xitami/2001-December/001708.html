In message <006201c155d0$01190520$0300a8c0@aplusnet.com>, "Jeff Hamblin" writes:
>The DT is sending a list request as follows:
>ftp> ls \dtf-tests\*\*Job.type
>200 Command okay
>150 Preparing to transfer ASCII file
>226 Closing data connection

That's because you probably don't have a file called "\dtf-tests\*\*Job.type".

Xitami can't expand wildcards in the same fancy way that unix FTP
servers do (generally by calling an external ls program to do it).

The FTP server in Xitami is intended to be a simple one to complement
the HTTP server.  If you need more advanced functions, then disable the
built in FTP server and use another one.
