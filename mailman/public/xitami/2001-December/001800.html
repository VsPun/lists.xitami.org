  a.. Freechat http://www.sonic.net/~nbs/unix/www/freechat/
    a.. Con: Code (cgi-bins) in C, not in PHP - harder to change/customize
for your needs if required.

  a.. ARSC Really Simple Chat (http://freshmeat.net/projects/arsc/ )
    a.. Pro: PHP based.
    b.. Pro: Works on Mysql
    c.. Con: No name reservation, colors, instant messaging.
    d.. Overall: Does not have many features. Is a plain/simple chat.


  a.. PHP WEBCHAT http://www.csit-sun.pub.ro/~gygy/chat/
    a.. Con: Too simple, check the demo chatroom at the page above

  a.. WebChat http://www.webdev.ro/products/webchat/index.php
    a.. Con: Install failed to create database tables. "The used table type
doesn't support AUTO_INCREMENT columns"

  a.. MWChat http://freshmeat.net/projects/mwchat/ (demo at
http://www.appindex.net/applications/development_releases/mwchat/demo/ )
    a.. Con: Download is a beta. Expires after 1 week, wants to send email
do devel.
    b.. Con: Could not get it to work
    c.. Con: MWChat is copyrighted 2000-2001 Appindex.Net
  a.. phpMyChat http://www.phpheaven.net/
    a.. Pro: Very nice install, web-based, worked just fine
    b.. Pro: Supports custom colors, smilies, etc.
    c.. Pro: Works on MySQL, PostgreSQL or ODBC
    d.. Pro: Fairly nice GUI
    e.. Pro: In PHP, easier to change if needed
    f.. Con: Can't associate pictures with users, as far as I know

  a.. phpWebChat http://chat.nitco.com/
    a.. Con: Does not work under my test setup: Warning: crypt() is not
supported in this PHP build in C:/Xitami/webpages/phpchat/engine/room.php on
line 22
    b.. Con: Last update: 7/27/00

  a.. PHPOpenChat http://www.ortelius.de/phpopenchat/
    a.. Con: PHP safe_mode has to be set to Off. "Explanation: If the safe
mode are enabled we can't change php.ini variables during chat's runtime and
that is what we need in output.php, a script which never ends."
    b.. Con: Does not work under my test setup: Warning: crypt() is not
supported in this PHP build in
C:\Xitami\webpages\openchat\includes\check_permissions_inc.php on line 35

  a.. TAHC http://freshmeat.net/projects/tahc/
    a.. Con: GUI, didn't like it. See demo at
http://www.itcampus.dk/tahc/tahc.php . Haven't tried installing it because
of that.

  a.. PHProjekt
    a.. Pro: It is much more than just Chat, it is a full suite. More
integrated with the rest, same look-and-feel, nice to have something like
MyYahoo
    b.. Pro: Setup of this tool is a breeze. Web-based, it all just worked !
    c.. Con: The chat app flickers when it refreshes (background color
problem ?)
    d.. Con: No smilies or custom colors in chat
