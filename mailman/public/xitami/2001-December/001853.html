> I have the following problem using the web based administration on Xitami
> for Linux.  When I make setting changes via the web admin screens the
> settings do not stick.  If I manually modify the cgf files using the pico
> editor, the setting stick.  I'm using the save buttons, restarting the
> server, but no changes stay.


I remember having this same problem when first using Xitami. The solution,
in my case, was to click the Save icon (sometimes more than once if you are
on one of the sub-configuration pages) until the main Xitami Administration
page is displayed -- then click Restart Server. I found that clicking the
Restart icon on any of the configuration pages would lose the changes.
