On Sat, 2001-12-29 at 14:32, dotcom-fm wrote:
> Hello!
> If any can tell mi how I can set DNS based virtual host on one IP adress? My domain is www.limit.pl, what can I do to set www.sklep.limit.pl (on the same IP)? Of course my server is Xitami!!! (sorry my english)

In the latest beta you can use the virtual host wizard from the front
web page.  You can also do it manually in your config file like this:

[Virtual-hosts]
	www.sklep.limit.pl = "skelp.cfg"


The contents of skelp.cfg would look like this:
[Server]
	webpages = "c:\\Xitami\\skelpwww\\"

This [Server] section tells xitami that when a request for
www.sklep.limit.pl is recieved, the files are located in the director
c:\\Xitami\\skelpwww\\"


The wizard is the easiest way to get this setup though.
