In message <000801c17923$d5ffc550$01597ec2@tay>, test@emcom.co.uk writes:
>I run Windows 2000 Professional and have IIIS installed. Can I run =
>xitami and IIS together?

Yes.  Either on different IP addresses (tell each one to use only a
specific IP address), or on different ports.  

>If I can, how can I direct my browser to correct site/virtual site?

If you use different IP addresses, use your hosts file, or the DNS, to
map from the names to the IP addresses for each server.  If you use
different ports, include the port number in the URL.

You can run Xitami on a different port using the "portbase" command line
option: -b BASE (which shifts _all_ port numbers up by that BASE
amount).

You can run Xitami on only one specific IP address by adding:

[server]  
  ipaddress=aaa.bbb.ccc.ddd

in your defaults.cfg and restarting Xitami.
