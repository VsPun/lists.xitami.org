>I can not get cgi script to execute.[....]

>then I get a default IE5 page saying there was a problem
>opening this pg.

The default IE5 page is useless for trying to debug problems like this.  
You need to go into the advance options and disable the "friendly" HTTP
error messages, so that IE will show you the real error message, and you
actually stand a chance of figuring out what is going wrong.  The real
error message, plus cgierr.log, will often lead you to the problem.
