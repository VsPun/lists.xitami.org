Ah. If you are using a PC to get mail and send it to the printer, there
are standalone command-line MIME decoders. Most are very smart: you can
send them plain un-MIME text and they will give it back unchanged. One
old favorite is UUDECODE.

So after you get email from the POP server and put it in a file, use
SHELL to pass the file through a decoder

 Folder$ = "C:\spool\mail\"
 MessageNumber$ = "1234"
 InFile$ = MessageNumber$ + ".bin"
 OutFile$ = MessageNumber$ + ".txt"
 SHELL "command /c UUDECODE <" + Infile$ + " >" + Outfile$
 SHELL "command /c copy " + OutFile$ + " PRN"
