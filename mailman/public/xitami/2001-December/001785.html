running same version I am able to do this for sure
perhaps you need no quotes surrounding password
i have something like this

[/cgi-bin/subfolder]
    username1=password1
    username2=password2
    ..... etc.
I have a lot of 20+ users on this folder everithing works fine.

Also be sure to edit the right .aut file
If you use defaults.aut then make sure your config (xitami.cfg or
defaults.cfg) points to this file but not some other
[Security]
    filename=defaults.aut
