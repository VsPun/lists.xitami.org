In message <jn9t2u0l34luek81hhu1h00somm8okh32h@4ax.com>, Bruce writes:
>$cgiurl 
>The url portion of the address to the guestbook.pl file The file
>guestbook.pl needs to be placed in your server's cgi-bin and chmoded
>read/execute for users of your httpd. (chmod a+rx) 
>
>to a "guestbook.pl script being used under Xitami?

Put it in Xitami's CGI-BIN directory, make sure you have Perl installed,
and that the Perl executable is on your PATH (eg c:\perl\bin is in your
PATH).  If you're running on Windows then Xitami will treat things in
the cgi-bin directory that have a #! line as their first line that
mentions "perl" in it, as perl programs to run; the chmod is not
necessary under Windows (it's a unix/linux thing).
