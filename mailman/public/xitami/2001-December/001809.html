In message <nf682uc4pk2u73j59vf6hqo4il67fdeb42@4ax.com>, Bruce writes:
>Is there an FTP log somewhere in Xitami that will tell me which files 
>are being transfered from \ftproot\pub and when?

These are recorded in the same log as the HTTP transfers -- as far as
Xitami is concerned, it's all just files :-)

>What would be a good way to collect, store, and delete old log files in 
>Xitami. 

You can configure Xitami to rotate the log files daily (weekly/monthly),
and save them.  Then you can have an external program process them into
pretty reports (analog, webalizer, webtrends, et al), and something else
archive them later.
