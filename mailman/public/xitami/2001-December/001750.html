FYI: instead of using server side includes, you can also add a
<script src=http://www.server.com/cgi-bin/ezcount.exe></script>
The only difference in the exe file (compared to a exe-file that creates a 
complete page) is that the browser expects javascript, not html. Simply 
put: if you want to display "hello world" you will have to output 
"document.write('hello world');" [doubles quotes added for clarity].
I use a subroutine to add the javascript container to the desired output.
