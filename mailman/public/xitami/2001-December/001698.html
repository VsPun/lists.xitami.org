>My config looks like this:

Yup.  Beware that there are _two_ webmask settings in the configuration
file (used for different things); make sure you're setting the right
one.  (Based on the section that the setting is in.)

>Now, what goes in that text file???  How do I type in the addresses??
>Should I put in the [webmask] at the top??

No [webmask] at the top.  Just list the IP addresses (with wildcards),
one per line.   Put a ! at the start to negate it.  Make sure you have
something to allow all the addresses you want allowed; the default is to
deny everything once you have a webmask list.  (If there's no webmask
list it allows everything.)
