I'll help you with any questions you have.

At what level do you want to make the compression work?

Typically, you can make almost all the changes in just two places. 
First, there is the HTTP protocol driver, smthttp.c and smthttp.l (the
state machine).  Second, there is the HTTP library module, smthttpl.c.

If you want to modify the state machine you need to understand Libero,
the tool we use.  It's well documented on imatix.com.

If you want to modify the library, you need to understand how it
works, and the best way is to read the code, and try to see how the
various functions are called from the protocol driver.  The library
code is *horrible*, I know this.  My apologies.
