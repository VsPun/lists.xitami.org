Oh and you should take care to leave this entry in the file:

 [/Admin]
    name="your pass word"

separated before and after with new lines from the rest of the file,
in case your browser-based admin is enabled. Otherwise everyone who has
access to your [/] root directory will also have access to your /Admin
directory which happens to be under root.

You need to make a section like this in the aut file and test to see if
ok:

[/]
    Name=Password

There will be as many  Name=Password pairs as there are users, each pair
always on an individual line, and you are the one who is supposed to add
these to the file (by which ever method you see practical and safe).
