In message <006d01c103d7$eb837f00$617969d5@ben>, "Ben Wheare" writes:
>Any1 know how to use the errors. gsl file to produce a replia of the apache
>type. I.e. display a customized message, title e.t.c???

errors.gsl is basically a template file for what to output, with the
bits in $(....) being values that Xitami has set which you can
substitute into the HTML.  There's a list of which values are available
at the top of the file (in comments).

So essentially you need to edit the HTML in there to match what you
want, using $(...) to put in values that Xitami has from the request
where you want them.
