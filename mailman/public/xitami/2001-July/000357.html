>My website is composed from some static pages and web applications.
>I would like to use http protocol for static pages and secure https for
>applications. But I was not able to figure out how to force using https

There isn't a way to do this at present (although it's a good idea,
which has been suggested before).  Generally for most sites forcing
HTTPS doesn't gain you very much other than avoiding the possibility of
users accidentally sending things in plain text they might not want to
do so.  However that's usually balanced against allowing users to do so
if they're really not able to use HTTPS.
