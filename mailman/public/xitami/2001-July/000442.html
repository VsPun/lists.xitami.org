>I just upgraded one of my servers (WinNT 4-sp6) with the latest service
>version, 2.5b1.  

2.5b1 is NOT the latest service version; it's an old beta version.
2.4d9 is the latest release version of Xitami.  2.5b5 is the latest beta
version.

"2.5b1" has some known bugs so you might want to at least try 2.5b5
(which also has some known bugs, but fewer/different from 2.5b5).

>Doing some investigating, the POST parameters are not being passed.  I 
>looked through the xitami.cfg file but I'm not sure there is anything 
>there that I want to change without some help.

As a start try turning on CGI debugging ([CGI] debug=1), and have a look
at the .cgi and .cgo files.
