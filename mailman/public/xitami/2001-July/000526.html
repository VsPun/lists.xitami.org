Look for a file called HOSTS on your system, under Win98 it should be located
in your C:\Windows\system32 directory (I think, its been a while for me under
Win9x). There is a sample file called HOSTS.SAM that is an example file for
HOSTS if you don't have it there already. 

At any rate, add the following line to your HOSTS file:

127.0.0.1	localhost

HOSTS is always checked first by Windows when it is trying to resolve
hostnames. That should help you.
