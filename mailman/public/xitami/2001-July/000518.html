If you want to use relative 'Location:' headers like you have above, you
will need to use XiFusion as Xitami (latest stable release) does not
support server-side redirection, thus the 'Location:' must either be
absolute (header("Location: /fw/menu.php?menu=1&submenu=0");) OR you use
XiFusion as it will handle the server-side redirections (both relative and
absolute) correctly.
