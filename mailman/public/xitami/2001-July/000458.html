Try the "1st Class Mailserver" (http://www.1cis.com).  It sticks a fairly
non-intrusive footer onto all outgoing messages and is extremely easy to
set up.  However, I've never been able to figure out how to bring it down
without rebooting.  It consumes a very small amount of memory, and is
fairly fast.  There are other mailservers out there, but they take a while
to set up and not all are free.  Beware, the shareware version of 1cis is
SPAM-prone and offers no protection from outsiders.  If you want a more
permanent solution, then I recommend trying EzMTS (www.ezmts.org - the
freeware version).  You still need some CGI-based solution to actually talk
to the mail server, but at least you would have a test server to work with.
