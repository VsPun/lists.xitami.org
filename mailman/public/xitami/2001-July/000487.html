On Tue, 24 Jul 2001 15:38:11 -0500, Richard Donahue wrote:

>Does anyone have any recommendations (tips) when it comes to installing Php
>with Xitami? I'm using 2.4c3 Pro and can't get it to work. It either gives
>me a CGI error when trying a test .php file or it just gives me a blank
>page.

Dear Rich,

I have a win2k (formerly a Win98) system with Xitami and PHP running
very well and without trouble.

(1) Installed PHP for Win (download install file from www.php.net) in
      the "normal" directory c:\php

(2) In Xitami, I then edited the {Filters] section in the *.cfg files
for 
      the virutal hosts and defaults.cfg as follows:

     [Filter]
    .php=c:\php\php.exe
    .php3=c:\php\php.exe
    .phtml=c:\php\php.exe 

After that, I could call up any domain (virtual host) and use php files
with the above file extensions and they run without any trouble or
error messages.
(of course, the domains / virtual hosts need to be entered in the Win
HOSTS file to be recognized with "localhost"

God bless you with His grace and peace
