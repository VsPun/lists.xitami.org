In message <te0gktohtegce6a66ntulpjlt3nsbmha34@4ax.com>, jeff writes:
>Do I just run the Pro version, and have it handle everything,
>including Virtual Hosting? 

Sure.  Unless you've got a specific reason for having both Xitami/Pro
and Xitami running (some dramatically different configuration that
cannot be merged together), then Xitami/Pro can do everything that
Xitami will do "and more".  So you need only run Xitami/Pro: it'll
handle virtual hosting, HTTP/HTTPS/FTP, etc.  HTTPS virtual hosting is
tricky (needs to be IP address based) but if you're using HTTPS only for
a single secure order site that shouldn't be an issue.
