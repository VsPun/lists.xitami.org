Ron,

This is just an example...
1) defaults.cfg
[Virtual-Hosts]
www.douglasdemocrat.com=dd.cfg # site www.douglasdemocrats.com
www.orangekitty.com=ok.cfg     # site www.orangekitty.com
douglasdemocrat.com=dd.cfg # what if someone just types in
                           # douglasdemocrats or organgekitty.com?!
orangekitty.com=ok.cfg     # ....this handles that!

[Server]
Webpages=webpages

Now, do not put anything under webpages or the default pages you would like
served, say for my_business_domain.com (your
business!?).

2) dd.cfg
[Server]
Webpages=c:\douglasdemocrats (or /var/douglasdemocrats or whatever)

c:\douglasdemocrats -> all your files related to the democratic party of
douglas county
3) ok.cfg
[Server]
Webpages=c:\orangekitty

c:\orangekitty -> all files related to orang kitty web site

Now, remember if you IP address is 111.222.333.444, you _must_ have DNS
records setup (or request your ISP to do) such that both
these domains point to your server's IP address (111.222.333.444).  This is
the key for the whole setup to work!

Hope this helps.
