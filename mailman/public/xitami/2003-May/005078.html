Hello !

Use the ip address of the server, like
http://123.123.123.123

If this still don't work, then your network is not configured correctly.

Try:

tracert zeus

and

tracert 123.123.123.123

from the cmd prompt (or use ping).

This command works always ... if your network is configured correctly.

If the command works, but your browser is set to use a gateway, it will try 
to connect to the gateway.
In internet explorer, go to Tools, Internet Options, Connections tab, LAN 
settings, select "bypass proxy for local addresses" and in advanced select 
the local range.
