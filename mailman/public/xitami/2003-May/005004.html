I have myself have had good luck on my CGI server running Xitami 
on Win2000 using the console version.  I just have a .bat file to 
loop loading in case of a crash which so far seems pretty stable.
This in the bat will work:

:start
xidos32
goto start

I have also heard turning on FTP in Xitami can cause stability 
problems.  I know if I access Xitami FTP over my 100mbit LAN it 
crashes the server (not stable).  So just use WarFTPD or the like to 
keep out fragil Xitami FTP code.
