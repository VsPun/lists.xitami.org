Graeme Nichol wrote:

> Hey Alex
>
> How did you get Port 80 open again ?
>
> It closes my one laptop but not the other!

I had to set permission in the Computer Associates firewall to allow 
http on port 80 when Xitami is running.  When my ISP blocks http ports I 
reset Xitami to watch port 3080 .  If you have a router you need to set 
the router to recognize each computer & set permissions for both. Check 
your router docs for how to do that.
