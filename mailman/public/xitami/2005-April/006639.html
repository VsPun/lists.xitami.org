Hi there,

I have tried it. Unfortunately I was unsuccesful.
This is wat I did:
My OS is NT4 workstation.
Xitami version v2.4d9
I installed PHP. Not the newest version but an older one: 4.2.3. This
according to information obtained from the Squirrel Mail site.
I made sure this was in the php.ini file:
cgi.force_redirect = 0
cgi.redirect_status_env = CGI_REDIRECT_STATUS;
I added the following lines in the Xitami file defaults.cfg:
[Cgi-environment]
    Cgi_redirect_status=-

The only thing that did not work was the redirection in PHP under Xitami.
I posted this in the Xitami list as well but there was no usefull answer.
If you can make it to work, please inform me how you did this.
The entries in the php.ini and defaults.cfg were the result of email
correspondence about Squirrelmail and PHP under Xitami.
