> Is there a way to do a 404 pages for each virtual site in Xitami?

It is, unfortunately, a bit hard to make custom pages (I /think/ you 
can specify error pages for each virtual domain but I am not sure, if 
you can it's in the online help).

However: You can't use dynamic pages, either. My solution to this 
(which will also solve the virtual domain problem) is to use a 
frameset as error page. It maps to a "normal" php page, which can 
then check the "referrer" field for different sites, do some smart 
mapping/redirecting for "known errors" etc.

Check out (for example) www.9000.org/ewrwerw compared to 
www.9000.org/manual/to_mount_a_lens
