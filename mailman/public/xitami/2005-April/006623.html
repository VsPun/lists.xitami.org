I'm pretty sure it can't. The solution (if any) to this is probably 
to set up a filter that maps .exe to "cmd.exe /c" or something like 
that. I haven't done it in Xitami, but I vaguely recall that I did it 
in Netscape FastTrack some time back in the dark ages.

You may have to wrap it in a batch file, too (map extension .exe to 
startcmd.bat which contains
@echo off
cmd.exe /c %1
..and that should be it).
