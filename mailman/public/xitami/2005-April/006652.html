Check that the correct mime types are defined in the config files=20
(default.cfg).
Normally pdf should be present in xitami.cfg
If the mime types are not present, add them in default.cfg and restart =
the=20
srever


For pdf it's application/pdf
for mht message/rfc822

[Mime]
     default=3D*/*                         #   Default MIME type

     htm=3Dtext/html
     html=3Dtext/html
     txt=3Dtext/plain
     ...
     pdf=3Dapplication/pdf

add in default.cfg

[mime]
     mht=3Dmessage/rfc822


These are mappings of extensions used in the windows world to mime types =

used esewhere (first in mail messages to send "rich" content)
