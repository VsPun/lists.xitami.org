At 16:56 14/04/05, you wrote:

>I have implemented the cgi based Swish-eX that includes the Xitami 2.4d10 
>server.  All works great except that .pdf and .mht (Microsoft Internet 
>Explorer saved single file archive) will not display as a .pdf or .htm file.

Check that the correct mime types are defined in the config files 
(default.cfg).
Normally pdf should be present in xitami.cfg
If the mime types are not present, add them in default.cfg and restart the 
srever


For pdf it's application/pdf
for mht message/rfc822

[Mime]
     default=*/*                         #   Default MIME type

     htm=text/html
     html=text/html
     txt=text/plain
     ...
     pdf=application/pdf

add in default.cfg

[mime]
     mht=message/rfc822


These are mappings of extensions used in the windows world to mime types 
used esewhere (first in mail messages to send "rich" content)
