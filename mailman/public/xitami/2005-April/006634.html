I'm using Xitami for many years now, and while the server itself is not updated 
anymore, i still find ways to "upgrade" it myself.
For instance, i'm able to send "Content-Encoding: gzip" files instead of 
uncompressed files. Makes a lot of difference on a slow uplink like mine when 
the filesize exceeds 100k.

At this time i'm experimenting with XMLHttpRequest to provide dynamic content 
without page refresh. That's the way to make webpages behave like normal 
desktop applications. While users are entering data in forms, validation on the 
server is performed real time (fired by an onBlur form event).
