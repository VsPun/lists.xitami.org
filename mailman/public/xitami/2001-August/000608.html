> 
> BTW is there any that XITAMI can be configured to perform a specific
> action depending on a request such "Get /default.ida..." ?

PHP, REXX or some type of CGI scripting would help here. You'd have to
associate *.ida documents to the scripting engine.

> 
> Or how could create a page which would satisfy this GET request?

Create a stock HTML document and save it as 'default.ida' in the webroot for
each domain. Then in defaults.cfg add:

[MIME]
    Ida=text/html

It works here, try it:

  http://www.2cah.com/default.ida?NNNNNNNNNNNNNNNNNNNNNN
