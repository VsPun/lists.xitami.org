When you state that you 'create a virtual directory named 'greene', do you mean
simply a directory under 'Xitami\webpages' or are you creating a virtual host ?

There is a simple way to see if php is working. Open a DOS box, go to the
directory containing 'today.php' and type 'path to php.exe'\php today.php (eg
c:\xitami\php today.php). You should be blessed with the expected output from
your php script on the console.

I installed php way before Xitami so I didn't have the option of telling the
installer about Xitami, however the setup is just a matter of setting the Xitami
php filter & moving 2 files, php.exe & php4ts.dll.

In any case the virtual directory question may be the most important.

Regards, Paul
