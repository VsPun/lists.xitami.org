> BTW is there any that XITAMI can be configured to perform a specific
> action depending on a request such "Get /default.ida..." ?
>
> Or how could create a page which would satisfy this GET request?

You could create a PHP page called default.ida with something like:

<?

mail("webmaster@$HTTP_REMOTE_HOST", "Code Red Worm", "You have been infected
by the Code Red worm! Please download the patch from www.microsoft.com.",
"From: You <you@yourdomain.com>\n");

// please note this script is untested
?>

and add .ida = php.exe in the [Filters] section of defaults.cfg. Or you
could create a Perl script and set the filter up to run Perl instead of PHP.
