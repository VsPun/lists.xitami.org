I have been using LRWP for some time  under Visual Basic, using as freeware
DLL written by a person who seems to change web severs faster than I can
keep up. Sorry but  I cannot give you a current address for the software
download, since none of the ones I  have work anymore.

I do, however,  have the  DLL and some demo code that illustrates how to use
it. My application is a remotely tuned interactive radio directly from a web
page (html/javascript) You can see the value of LWRP at www.w4mq.com and
follow the links to the web page radio. Tuning rates vary from one/sec for
dial up connections to up to 10/sec for broadband. Try doing that with
CGI!!!
