If all you want to do is keep the contents of a text script private, why not 
use a simple script to call the actual working script which is kept in a 
different location not visible to the internet?   You can point to scripts and 
pass variables anywhere ~ to other local drives or other machines.  You 
and your scripts have complete access.  To the internet user only the 
calling script is visible.

You could also cripple the working script so that it only responds to local 
calls from your machine and bails out if not.

If working remotely you can pass a field variable to the machine so it 
recognizes you.

Not sure about your multiplatform and privacy needs but you should 
consider using programming software with compilers.
