If you add  #!/usr/bin/perl (the exact path to perl not neccesary)  to 
the first line of perl-scripts, xitami/perl should work with any 
virualhost.


You don't need these free mailing scripts; better do your own:


#!/usr/bin/perl 

use MIME::Lite;

$mailhost = "someMailAlias";
$From = 'someOne@somewhere.net';
$To = 'anotherOne@outside.net';
$Subject = 'mailing';
$text .= "$something\n";
 MIME::Lite -> send('smtp', $mailhost, Timeout => 60);
 $msg = MIME::Lite -> new(From => $From, To => $To, Subject 
=> $Subject, Data => $text);
$msg -> send;
undef $msg;
