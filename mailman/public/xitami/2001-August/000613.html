Don't put a webpages setting in defaults.cfg.  Put it in basehost.cfg,
or just define virtual hosts for everything.  

This is a known issue with 2.5b.  BTW you should probably be using 2.5b5
if you want to try the 2.5 betas, unless you don't care about the Windows
denail-of-service attacks.
