I have the following set up for my server:
xitami.cfg -> [Server] -> error-script=./templates/errmain.gsl (for main
server, you might be using defaults.cfg as recommended by Imatix people)
vhost01.cfg -> [Server] -> error-script=./templates/errvhost01.gsl (for
virtual host 1)
vhost02.cfg -> [Server] -> error-script=./templates/errvhost02.gsl (for
virtual host 2)
vhost03.cfg -> [Server] -> error-script=./templates/errvhost03.gsl (for
virtual host 3)
... etc.
You have to also specify in configs the use of error-schemas: [Server] ->
use-error-script=1
In this case templates directory should be under Xitami executable directory
(in your case //xitami).
