Hi Jill,

   Are you running any other CGI's successfully???  If not I'd check that
   you enabled the ENV variable SSI_INSECURE or config...

[Security]
    Ssi_insecure=yes

Ref: http://www.imatix.com/html/xitami/index6.htm#TOC64

> #exec cmd="command"
>    
> Executes a native command and includes the output of this command. Note that
>     the command runs in the main Xitami directory. This SSI command - which is a
>     potential security risk for your system - is only accepted if the environment
>     variable SSI_INSECURE is set to 'yes'. (Under Windows NT, in the System Control
>     Panel, under 'Environment'; under Windows 95 by adding 'SET SSI_INSECURE=yes'
>     to the autoexec.bat.)
