At 07:17 AM 8/19/01 +1000, Neville Franks writeth:
>I've installed Xitami and PHP4 on a Windows 2000 box and everything is
>working fine except I can't see an FORM POST fields in the PHP script.
>ie. When Submit is pressed the field values are all empty when the PHP
>script processes it.

I never liked this about PHP (or at least I could never figure it out), but
the HTTP_POST_VARS hash is a bit strange.  Basically, your variables are
converted from the hash into normal variables:

phpinfo() shows something like this -
HTTP_POST_VARS["MyVariable"] = MyValue
MyVariable = MyValue

Referencing it like this doesn't work as expected:
if ($HTTP_POST_VARS["MyVariable"] == "MyValue")
{
}

However, referencing it like this does:
if ($MyVariable == "MyValue")
{
}
