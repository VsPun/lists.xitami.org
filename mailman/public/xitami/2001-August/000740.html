Hi,

I joined the list a few minutes ago and I don't know the entire background
of your problem.  However, from the PHP script you've shown, it seems that
you are outputting HTTP headers rather late in your script.

If you want to output HTTP headers, they *must* be output before any other
kind of output is sent to the browser, including HTML tags and specifically
HTML <head> tag.  Your script should actually read:
<?php
header("Location:http://.../");
?>
<HTML>
...
</HTML>

(the rest of the HTML stuff is redundant as your page will have been
redirected)  For more information on usage of header() PHP function and
associated caveats, please refer http://php.net/header

Of course, what Ewen says about either specifying http://../ or the page
relative to the webroot (/menu.php) is correct, for the Location header to
work.
