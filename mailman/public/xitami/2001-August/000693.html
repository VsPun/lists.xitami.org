> my scripts which use input from user forms.  I tried putting in a long string of characters (like 300) to see if the text would wrap and they didn't.  

Wrap in the final output or wrap in the HTML Form? 

If you want to see it wrapped in the browser as the user is
inputting, Perl can't help because the text does not get to perl
until the user hits Submit. You may be wanting text wrap in the
browser. 

There is a WRAP=Off|Soft|Hard parameter for the textarea tag that
specifies wrapping for the form-user's convenience. It works
differently in different browsers and is generally pretty crude. 

 {textarea NAME="foo" ROWS="3" COLS="40" wrap=Hard}

See for example: 

http://www.blooberry.com/indexdot/html/tagpages/t/textarea.htm
