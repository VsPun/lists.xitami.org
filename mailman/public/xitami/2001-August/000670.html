Code Red Vigilante:

 http://www.dynwebdev.com/codered/

Basically it listens on port 80 (nothing else can be on port 80, of course,
so it mean you can't run a web server with this too). When it detects a Code
Red connection from an infected machine, it then uses the shell access of
the infected computer to pop up a window on the infected computer's screen
saying their computer is infected and giving info on how to get rid of the
infection.

Seems more constructive than some suggestions... :)
