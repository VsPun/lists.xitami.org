> I want to be able to have a seperate, simple URL for each of my
> sites on my home computer. If I type "http://qbono" I want it to
> load the site located at "e:\websites\qbono.com\", and if I type
> "http://xpxl" I want it to load the site located at
> "e:\websites\xpxl.net\", etc.

Find your "hosts" file, hidden deep under the WINDOWS folder. Mine
is at C:\Windows\system32\drivers\etc Open it with NotePad.

Add lines :

127.0.0.1       qbono.com
127.0.0.1       xpxl.net

Now (if Xitami is running with basic configuration) these URLs
should open your default page. (Only on that machine.)

To make them open different sites:

Use the http://localhost/admin tool to set up Virtual Hosts for each
site.

The fist box is a nickname and will be used as file-name for the
VHost CFG files. "qbono" is a good name here.

In "DNS Host Name", put "qbono.com"

In "Web page directory", put "e:/websites/qbono.com/" (note:
forward-slash is probably best here, even on DOS\Win boxes.)

Ignore the other boxes for today.

"Create", Save your way back to the main admin screen, and then it
may be best to re-start Xitami.

Test.

If it works, repeat for your other sites.

When the real qbono.com comes on-line, you better go into the
"hosts" file again and delete the pointer to 127.0.0.1 !!! Otherwise
your PC will forever go to your box, not the real server.
