This can easily be solved by 1 instance of Xitami set up to have 
the default host plus 3 Virtual Hosts (VHosts) for your 4 IPs.

When creating a VHost, select the IP that you want to use for it.
I only have 1 IP, so I haven't personally used the IP method to 
specify VHosts, but I am using the DNS method successfully.

Before you begin using Virtual Hosts, please refer to the list archive 
from last month as there is a bug in 2.5b5 and the solution (requires 
re-compiling the source) was discussed.  If you're using 2.4d9, then 
VHosts should work for you.

Good luck

Jon Warren
