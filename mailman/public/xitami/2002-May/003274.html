At 12:52 PM 5/30/2002 -0400, Ervin Kosch writeth:
>Just a couple of thought but how about these meta tag:
>/<META HTTP-EQUIV="expires" CONTENT="Wed, 26 Feb 1997 08:21:57 GMT">/  - 
>Have the PHP write the current date and time into the tag.
>/<META HTTP-EQUIV="Pragma" CONTENT="no-cache">/ - Also forces the 
>browser to not cache the content of the page
>
>I know there are PHP Header() function equivilats to these meta tags as 
>well.

Not all browsers pay close attention to META tags.  Using the header()
function to dump HTTP headers back to the browser is generally more
effective - especially through HTTP proxies that rely heavily on cache data
being correct.
