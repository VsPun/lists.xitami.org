NAT.  But I have run a lot of servers over NAT (FTP, WEB, Email, Finger, 
etc).  Basicly if the correct "incomming" Ports (this is normaly a config option 
on the Gateway/Server) on the external IP are directed to your local (NATed) 
IP then you can run most any server you like.  Hope that helps.
