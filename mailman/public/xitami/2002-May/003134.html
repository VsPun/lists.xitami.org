what version of w2k are you using (me, pro, server, etc). some version have
iis services running by default. you have to shut them down. i turn them off
by starting a MMC with the computer management snap-in. all services are
listed there - and you can start / stop them and specify which ones should
start automatically. make sure all iis related ones are shutdown and set for
manual startup.

i doubt the old version of xitami is running. open a command prompt and
telnet to port 80 (telnet 127.0.0.1 80). if there is a listener running on
that port - it may give info as to what it is. if xitami is running - it
will tell you.

ex. - i get the following....
HTTP/1.0 501 Not implemented
Server: Xitami
..
