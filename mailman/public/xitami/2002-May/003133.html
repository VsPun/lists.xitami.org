rc5 should be the last version before the final release (as rc4,
rc3, ... :-))

its more stable and faster than rc3.

fixes:
------

the classloader now shows a correct errormessage when subclasses
cannot be loaded. 

the classloader doesn't crash if the classpath is invalid
(typically when using single slashes instead of \\ on DOS
filesystems like windows or os/2)

again bugfixing in sendRedirect()

additions:
----------

RunnerClassLoader#getResourceAsStream()

ServletContext#getResourceAsStream(), see implementation notes

logfile_remove_classname removes the classname from the logfile
when using GenericServlet#log(...)

the path of "srvl_config" is now automatically added to
logfile_path and classpath - we can use one servletrunner for
multiple sites now.

http://runner.conacom.de/downloads.html

feedback is welcome.
