In message <NBEMJPOLIJHPGPFFHOLLIEIGGHAA.nberg@gandalf.xs4all.nl>, "Nico Berg" writes:
>Is this possible? I have adsl and 4 ipadresses The adresses all come in on
>one NIC: eth0. (I am using Linux).

Set [server] ipaddress in each of the servers, rather than changing the
port number.  The default is "all addresses", which is why you can run
only one.  That said, you can do IP address virtual hosting or name
based virtual hosting in a single Xitami -- why do you want 4 of them?
