No, I don't claim that at all; I was making an observation that they were 
different formats and that it would be most prudent to have configuration 
files Xitami likes.

What I could consistently get Xitami to crash on were my 16-bit CGI-scripts 
when I had "Use Standard I/O?" set the wrong way in the WBA under " CGI 
Properties.  It took me a long time to figure it out because some virtual 
hosts were set one way, while other vhosts were set the other way (it's a 
check-box).  Once I made it consistently the right way, no more "Abnormal 
Application Termination" errors.

Hope this helps,
