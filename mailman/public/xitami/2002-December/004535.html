Hello !

I'm in the repair business (not software).
When a computer works normally and then crashes, it is generally a hardware 
problem.
Software don't go defective after some time: the instructions that makes up 
the program don't get corrupted, but a defective memory chip can cause such 
problems (or a overheated processor, or a failing hard drive or an power 
supply with dried condensators or ...)

Of course it can also be a combination of some software that don't work 
well together. Is there some other software running on the server?
But since i'm using w2k and xitami and never experienced a crash, it can't 
be Xitami and/or windows alone: the error should be duplicated on my server 
since we both use the same software.
