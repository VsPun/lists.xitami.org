> > parser to use with Xitami.
> > http://home.fortech.lv/~andris/redirect/

> Do you know whether there is any difference in overhead between eg a
> perl cgi executable and a 
> perl filter activated as you describe?

The only difference is that cgi executes from a special cgi-bin location,
filter - from any other with specific file extension.

The reason I wrote a such tool is that www is a living media. Some pages
dissapeers, some moves to another locations. Many search engines keeps old, non
working urls for a log time. To avoid errors I use this redirector.
In a such way http://home.fortech.lv/uvis/ redirects to http://www.uvis.lv/ ,
http://www.safequipment.com/ to http://www.saftehnika.com/ ,
http://home.fortech.lv/~aparats/ to http://www.aparats.lv/ etc.
