> how can I run this web on my PC? I tried: http://127.0.0.1/myweb,

http://127.0.0.1 should open a main-page. Does it?

Virtual Hosts will not work locally unless you trick DNS.

The easiest way is to open your "hosts" file. On WinXP it is
C:\Windows\system32\drivers\etc\hosts, and I think NT4 had it the
same place. It has some #comments, and then a line for "localhost".
Add a line for each V-host, all pointing to 127.0.0.1. Like this:

 # Copyright (c) 1993-1999 Microsoft Corp.
 # This is a sample HOSTS file

 127.0.0.1       localhost
 127.0.0.1       myweb

Save the changed "hosts" file, then try http://myweb in your
browser.
