hello

found a fast workaround:

<http://web.verfaillie.com/invasion> normal access via adsl
<http://web.verfaillie.be?invasion> redirection to cable access

Users have to remember to use ? instead of /

<http://web.verfaillie.com/index/invasion> gives me a not_found since 
'index' is a executable and there is no directory or file 'invasion'

I'n already very pleased that i don't need a complete path like
<http://web.verfaillie.be/cgi-bin/index?invasion>


BTW language used is PBCC, sort of basic specific for console (and web) 
applications.
