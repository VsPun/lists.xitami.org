Under the logging section of the main configuration file there is an option:
"Translate IP address" - I would assume that is a DNS lookup on the IP
address.  If that is not enough then you could also toy with the "Log File
Format" - it gives you different options (maybe the extended one) that might
do what you are looking for.  And 3rd, the "log file format string" might
allow you to put a DNS/Hostname variable that writes the DNS name to the log
file.

And if all that doesn't work, then you can always write a script that parses
the more recent log file (not the current though because the current log
file is still open and controlled by xitami) and performs a DNS nslookup/dig
on the IP address and rewrites that data back to the log file.

good luck.
