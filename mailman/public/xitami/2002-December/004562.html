here is what I have in my defaults.cfg file

[Filter]
    .php = "c:\\php\\php.exe"
    .phps = "C:\\php\\php.exe -s"
    .phtml = "C:\\php\\php.exe"
    .shtm = "perlssi"
    .shtml = "perlssi"
    .ssi = "perlssi"
then under mimes

[Mime]
Php = "application/php"
Phps = "application/php"
Phtml = "application/php"

also in my autoexec.bat
SET PATH=C:\PHP\;%PATH%

this seems to work for me I am running Version 2.5b5 on XP home
should work from this but check the archives from the list you can get there
