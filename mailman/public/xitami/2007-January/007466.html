Well Ester, that's an "it depends". It depends on where you made the
changes. If you followed the suggestions and created the "default"
files, i.e. defaults.cfg, defaults.aut and so on, then yes it will
just pick-up the changes. If you changed the files that came with
2.5c1 i.e. xitami.cfg, xitami.aut and so on, then you'd best back them
up and copy them to defaults.<whatever> after installing from
xiopen_2_5.zip. The current version is xitami 2.5c2 BTW.
