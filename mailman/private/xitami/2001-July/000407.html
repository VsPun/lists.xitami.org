In message <001801c10bfc$7e707d50$0100a8c0@leiter1>, "Brian Leiter" writes:
>I know about @error/text-404.htm, the header and footer but for some 
>reason all I get is the STANDARD IE 404 error from internet explorer.

The @error/text-404.htm is how you get a custom error message.  But only
if the browser is willing to display it.  IE has its own ideas, and by
default displays a "simplified" (read: useless (IMHO)) error message
instead.  Any IE user that wants to see your customised 404 errors will
have to display this feature (called something like "simplified error
messages") in their IE configuration.
