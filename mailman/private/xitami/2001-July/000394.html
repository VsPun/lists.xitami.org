> take a perl script I have and make it so that I can pass form values to it. I know just about nothing when it comes to perl 

Xitami parses form values into the Environment with the prefix "POST_"
(this can be configured). 

Perl lets you read the Environment in an array called $ENV{}. Note the
curly-braces... in some fonts they look like parenthesis, but aren't!

If the form has fields named "name" and "gender", use this to print the
values: 

 print "Name            $ENV{FORM_name}\n";
 print "Gender          $ENV{FORM_gender}\n";

Instead of "print", you can assign to variables and do something else.
