a thought about reinstalling windows

windows 98 especially, but also win95, goes in search of any existing
version of windows on your hard drive ..in my experience it then cheats and
uses the settings from the previous install where ever it can.

assuming you reinstalled cause you had a problem the odds are this problem
still exists

the answer is to totally remove windows by deleting its directory and all
the system files before reinstalling the new version.
