Do you mean that when you type http://www.geocities.com/ (for example), you
want to go to your local Xitami server? If so, then create a file named
"HOSTS" in the Windows directory (or edit it if it already exists), and add
the following line to it:

127.0.0.1    www.geocities.com

You don't need to set up any virtual hosts, etc, in Xitami.
