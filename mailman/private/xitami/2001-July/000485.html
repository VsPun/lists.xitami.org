XiFusion will work with that version of Xitami, but requires another
wrapper for it to work correctly.  I'm experimenting with that particular
version of Pro as well, so here is a Perl wrapper for you to use (Requires
Perl, PHP, and XiFusion):

#!c:/perl/bin/perl

$ENV{"PATH_TRANSLATED"} = $ENV{"DOCUMENT_ROOT"}.$ENV{"SCRIPT_NAME"};
# You might need to modify the following line:
system "c:\\xifusion\\xifusion.exe c:\\php4\\php.exe";


In the Filters section, use something like this:
  .php=c:\perl\bin\perl c:\progra~1\xitamipro\nvmlwrap.pl

I know it isn't pretty, but it works.

Hope that helps!
