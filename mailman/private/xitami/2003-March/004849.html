> if ($debut=="")....

Try   print "$debut";

I bet you will find that it is always blank on one sever, and has a
value on the other server.

Your $debut is (or is not) imported from the URL Request. The user
can manually type a URL where $debut is any value. This could make a
script do something bad.

Older PHP imported the URL request anyway. Recent PHP does not
import the URL Request unless you ask for it. This can be changed by
the PHP administrator, or in each script.

Probably one of your servers has an older PHP, or the configuration
has been changed.

Read the PHP documentation for Register Globals:

http://php.benscom.com/manual/en/security.registerglobals.php

French?: http://linuxfr.org/2002/08/27/9393.html
http://php.benscom.com/manual/fr/configuration.php#ini.register-globals
