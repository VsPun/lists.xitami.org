> Is there a way to view Xitami access logs in real-time? i.e. to see the
> entries to the log files as they are written?
> 
> It is not possible to open access log files because they are in use by
> Xitami. Is there a way round this?


If you're on a Windows system, just use NOTEPAD
to view the log files. It doesn't try to put its own lock
on the file like, for example, WORDPAD does.
