Hi Andrew,

The absolute best (and free) reporting package would be AWStats.  I use this
as well.  This reporting package provides every detail I could ever want.
Only one minor problem though, it does not come with an "install program".
You will have to read the manual and invest 2 - 3 hours to get it to work
for the first time.  But, it's well worth the effort.

After I installed it, I created a batch file that windows runs automatically
at 1:00am every morning to generate the reports up until yesterday.  The
reports will accumulate data every day for the past 12 months.

Go to cnet.com and search for awstats and download it.
