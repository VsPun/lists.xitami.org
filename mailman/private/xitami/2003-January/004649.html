>> I am running 8 virtual domains all with separate logs in CLFX format. The
>> logs cycle monthly.
>> 
>> I recently noticed zero access reported in my log analysis software for a
>> period of about a week. However, I know the server was up for most of this

>Probably some disk access error, the garbage is the data in sectors that the computer assumed were containing access log data, while in fact there was nothing written at all.
>The computer keeps track of the position of data files and read the sectors that correspond to the file, but if some sectors were not correctly written, the read operation yelds 
garbage (the original content of those sectors).
