>I am trying to transfer an entire web site to cdrom.  [Some dynamic
>content] But can it do it, having the web site, xitami server and perl, 
>all living on a cd, without tcpip?

No.  Xitami, and pretty much every other webserver I'm aware of uses
tcp/ip as it's transport protocol, and hence won't work if it isn't
installed.

ISTR that Dr Dobbs did something a year or two back with a Java applet
that did the searching which would work without a webserver, for much
this sort of context.  Might be worth digging around their site to see
if you can find anything.

If you can live with Windows only it may be possible to do something
with an ActiveX control.

Then again I'm not even sure it is possible to install a web browser
like IE or Netscape without TCP/IP installed.
