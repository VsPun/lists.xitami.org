>  If domain = http://domain.com
> How do I set up something like this?
> http://something.domain.com

Ask your ISP's DNS administrator to set up an alias (or CNAME) of
something.domain.com that points to domain.com. 

A default installation of Xitami does not care what name is used to request
pages-- if a request comes to it, it serves its default page. 

If your real question is: how to make something.domain.com show a different
page than domain.com..... use the vHost wizard in Xitami's Admin screens.
