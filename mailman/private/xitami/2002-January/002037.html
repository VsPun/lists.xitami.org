All,

I just installed PHP 4.1.0 on my Xitami box, and I was getting the same
errors as the previous posters of this thread had mentioned: some error
message in the browser window about not being able to find the executable.

Well, I found a solution that may be of help. When I tried to run a sample
script at the command line, I got an error dialog box which said it was
unable to locate obdc32.dll. I checked around for this file on another
machine, found it, copied it over to c:\windows\system (this is a win98 box)
and tried the sample script again. Now it complained about missing another
DLL (its name escapes me right now.) I also copied it over and tried the
sample script one more time.

Bingo!

A check on the PHP site for these missing dll files indicates that these
missing files can be a problem - not very common, but it does happen. Other
users reported installing the free version of MySQL solved the problem - it
must have the right DLLs.

Anyway, its all working now, and I'm ready to start learning PHP. I just
thought I would share my experience.

-Warr
