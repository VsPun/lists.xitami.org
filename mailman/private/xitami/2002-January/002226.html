I take it you're using a 2.5 beta version (you don't say, but that
message shouldn't occur at all with anything earlier).  If so the issue
is that error messages are handled by a GSL template, precisely so you
can customise them.  But for some reason the template cannot be
processed.

Going out on a limb, I'd say you're probably trying to use Xitami 2.5b1
or Xitami 2.5b4 NT service (which is actually 2.5b1 in disguise).  Try
Xitami 2.5b5 (the latest beta) from http://www.xitami.com/
