    The following can connect at will.

    1. Friend using Roadrunner
    2. Mother using dialup

    5. From work using dialup

    The following cannot connect at any time. Ping and tracert do not work

    1. Friend using Roadrunner
    2. Sister using dialup
    3. From work using a T1
    4. Aunt using (not sure)
    5. TZO Tech Support

    I had people from the first list be able to view the website at the same
time that some people from the second were not getting anything.

    System details

    1.333 Ghz Althlon
    512 mb ram
    40 gb hard drive
    DSL connection through Earthlink/Mindspring. Handled by Covad.
    Linksys router (forwarding port 80 tcp to 192.168.1.5) I use Wallwatcher
to view the router log files.

    Windows 98 SE with all patches/updates from Microsoft site.
    IE 6.0 128 bit encryption with all patches/updates from Microsoft site.
    Norton Personal Firewall 2.0 with the latest and greatest.
    Norton Antivirus 2000
    TZO client
    Various family tree software

    Network settings. IP 192.168.1.5 subnet mask 255.255.255.0 - Gateway
192.168.1.1 (Linksys router) - DNS 207.69.188.186 and 207.69.188.185
(Mindspring DNS) Hostname - pongboy (my computer name)

    I have tested it with the firewall disabled and also uninstalled. Right
now it is installed since it didn't make any difference.

    I have uninstalled and reinstalled the Xitami software. It is a very
easy setup but not 100% sure that it is correct. I kept most of the defaults
but turned things off such as FTP etc.

    I do show them in the Wallwatcher log as an incoming connection and I do
see them as an active connection in the Xitami icon.

    But for some reason the people on the second list do not get any pages
served to them.

    I am by no means an expert in running a webserver but I do know a
little. My friend is an MSCE + Internet + a few others and he is stumped as
to why this is happening.
    I have had someone at TZO try to do some testing remotely but they can't
hit the site or my computer either.

    Any insight or a push in a certain direction to help me figure out the
problem will be greatly appreciated.

    Thank you in advance for any help/direction you can provide.
