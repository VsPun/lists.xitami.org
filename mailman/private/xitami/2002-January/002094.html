You can put in a page like www.ackerman-family.com/webcam.html with a meta
redirect to port 8080 something like this:
<META HTTP-EQUIV=Refresh CONTENT="0;
URL=http://webcam.ackerman-family.com:8080/">
Which would point at another copy of Xitami running on port 8080 and
handling the web camera and set that up as a virtual site or in DNS, either
should work. It would probably be easiest to do on a network setup with DNS.
There are many different ways to do this on a network.
