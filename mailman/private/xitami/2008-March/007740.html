For the moment addresses don't matter. You have one NIC. It has one Port 
80. If one Xitami server is listening on it then the other Xitami server 
can't listen on it. Simple?

So how do you get past this? Just run one Xitami server. But that leaves 
you with the problem of only being able to serve one web site and you 
want to serve two web sites, a live system and a test system. What to 
do, what to do?

Enter our saviour Virtual Hosts! How to do this is detailed in the 
configuration section of the documentation which can be reached at 
http://<your IP address or domain name here>/xitami or, if your Xitami 
is correctly set up at http://127.0.0.1/

I hope this helps. 8=))
