Download the zip. Unzip in two different directories. Edit the 
defaults.cfg so IP is different (for example, 127.0.0.1 and 127.0.0.2). 
Start the two servers. Does it work? If yes, great. If no, change LRWP 
and FTP port assignment to something different for each process. Start 
again. Now it works. Verify by editing the web pages.

It was very, very, very simple.
