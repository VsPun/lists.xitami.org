Unfortunatly I believe that the problem with Xitami on Windows isn't an
Xitami problem it's a Windows problem. I have been a long time Win OS user,
supporter, and technician and find this is more a Windows issue than
anything else. Especially if you have other apps that run on the same PC.
Windows is less forgiving for a lot of issues like Hardware, and badly
written software. Not saying that Xitami is badly written. I have been using
Xitami for a while and yes I crash at times too, but it's only while I'm
doing other things on the PC so I'm not sure it's really Xitami's fault. I'm
running Xitami, a mail server, a dns server, a web cam, monitor stocks, fax
software, cd burner, scanner, Norton AV,  Norton System Doctor and keep my
Outlook 2000 open all day (just to mention a few) all on the same system. I
probably have to reboot at least once a week. Is this an Xitami problem? NO!
most of the time either my DNS server forgets to server or some other apps
stops responding. I would be looking at a Windows or PC issue not Xitami.

My2cents
Ron Ackerman
