Mike,

I've been running the 2.4 versions (9x) for years. In fact, I'm using 2.4d9
on a Win98 machine with 64 megs of RAM.

I've never had a server crash, don't even know what it looks like. You're
running Win2000, which is basically a newer NT. Version 2.4d9 should still
run on it without crashes. If your OS was preinstalled, especially if on a
Compaq computer, I would say that's the culprit and not Xitami.

I know it's not what you want to hear, because it's a lot of work to do a
complete reinstall. I've done it myself many times on different machines and
I dread it each time, because I know I'll lose some piece of software or
data that I forgot about before the reinstall.

Just my 82 cents (inflation!).
