okay, here we go. Attached is my default.aut file exept the password. put yours in and you should 
be ready to go if your LAN addresses are 192.168.100.*

In case the attachment is stripped, create your file as desribed in the other mails and put the 
following lines in:(without the ---------) copy/paste 
only make sure you have a space between = and " otherwise it is not working for me here (2.5c2)
----------------------------
#*END

[/admin]
    username = "yourpassword"
    Webmask = "192.168.100.*"
---------------------------------

With this you should be able to open the console in your browser with :
1.  http://computername/admin
2. username password 
3. console window

I am using this webmask so I am able to connect from any machine in my small LAN
