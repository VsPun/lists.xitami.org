Stop xitami.
Look for defaults.cfg then put your defaults.aut in the same folder.
Mine is in  D:\xitami-25\app\defaults.aut

defaults.aut should look something like this;

#  Created at installation time
#
[/Admin]
    username = "password"[Private]
  
You can copy this using note pad,
change username & password to what you want,
save as defaults.aut
then start xitami,
type: http://127.0.0.1/admin & use your username & password to get into 
the console.
