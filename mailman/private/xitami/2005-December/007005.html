Default.aut here makes and modifies itself with a very little effort
and by making it in the console you have ip access restrictions, realm if 
needed, login/passwords.

Basicly mine;

http://192.168.0.11/admin    (from my laptop)  takes me to the console.

click Configuration button,     at top of page click the link     'Security'

Password file  name it,   defaults.aut   if its not existing.
Define button ...
  /admin  webpages/admin 192.168.0.12,192.168.0.11,127.0.0.1

Click /admin     username  password

pretty much everything is done with defaults except there are a few fill   
text boxes and add some checks marks along the way.    Takes moments.
