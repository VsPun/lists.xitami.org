I'll see if I can help.  Windows XP does not support chmod so no 
matter what server you are using it will say option not implemented 
or take the command and do nothing.  However if your script 
wished a file to be "read only" you can set that on a file by doing 
from a command prompt "attrib +r filename.ext" or you can use 
windows explorer right click the file and go to properties to chose " 
read only" check box.  Some scripts wish things like the config 
script set to read only to protect against being rewriten by a hacker.  
If that is the case setting it read only may end the error.
