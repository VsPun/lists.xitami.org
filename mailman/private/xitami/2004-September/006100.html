No, that message means that port 80 is opened by some other program, if you
have a good firewall you can see what application is using that port. you
can use zonealarm, kerio or the one you like best (and you MUST use one if
you connect to internet in these days). maybe, if you tried to launch many
times xitami, it should be that one is still open and running, and the
others you are trying to start find port 80 used; try "visiting" yourself
with a browser: type http://127.0.0.1 and you see what happens.
