What permissions do you want? 

Windows does not have the eXecutable flag so that is not needed (as it
would be on unix). Xitami uses rules similar to DOS to decide if a file
can be executed. First: Xitami will run scripts only in certain
directories, typically c:\xitami\cgi-bin. Then it looks for EXE, COM or
BAT extension (the closest DOS comes to an X flag). Xitami also examines
for the "MZ" header that marks a true EXE file (but COM files must be
loaded on blind faith). If the file appears to be text and isn't BAT, it
looks for "perl" in the first #! line and calls Perl either at the
specified place or by tracing PATH.
