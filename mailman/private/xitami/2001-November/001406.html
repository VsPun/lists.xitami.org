Carl my IP address is dynamic also.  I am also using TZO to resolve the 
dynamic IP issue and have been successful in running Xitami behind a 
router.  I never bothered to setup the DDNS feature in Xitami since I am 
already running the TZO client on my server.  Since I am on a network and 
behind a router I am also using a DNS server which allows me to 
access "MYDomain.com" from any pc on my network.  "Simple DNS' located at 
http://www.jhsoft.com/ offers a solution for this, remember also that if 
you're using a router you will have to map port 80 to the machine that 
you're running Xitami on.

Stephen.
