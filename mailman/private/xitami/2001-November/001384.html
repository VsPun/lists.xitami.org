I'm looking for a Page Hit counter, a simple text based one to show the 
number of times the page has been visited to the visitor.

I've looked through the archives and see this is a frequent question, but all 
of the links that I tried pointed to broken or non-relevant links (some of 
the messages where from 1999).

Also the URL given in the FAQ for this question does not work any more either.

All of the counter programs I've found so far require DOCUMENT_URI to be set
in the environment, and Xitami does not support this.

So how do you do a counter in Xitami?

> > All environment variables are available to CGI programs, if this
> > option is not disabled.
>

> on:
> 
> Running the 'testcgi' program does NOT show enviroment variable called

Okay, I've double-checked.  Xitami does not set the DOCUMENT_URI 
variable for CGI programs.  You'll have to find a web counter that 
does not use this.  I suggest you ask the discussion group to see if 
anyone has one they prefer.
