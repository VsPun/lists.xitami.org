On 29 Nov 2001 at 15:06, Cory Nelson wrote:

> OK, i'm trying to run a virtual host so that i can run two webpages off of
> one IP.  I can't find out how though.
> 
> Can somebody explain exactly how to set up a vhost, and if it requires
> running off of two ports, how do i specify which ports?
Are you trying to host 2 domains or two sub sites like this: www.yourdomain.com/siteA and www.yourdomain.com/siteB
To host domians you would use the Virtual Host configurations and setup a DNS based virtual host (vs an IP based) in the admin browser. To create the two subsites 
you would use the aliases configurations in the admin browser.
