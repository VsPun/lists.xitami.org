Check the following:

*.pl does not need to be bound to perl.exe
the path is irrelevant.
perl.exe must be in the path-statement of your computer so Xitami is able 
to find it.
if your script is called hello, it should work then.
if it is called hello.pl you need to call hello.pl of course (not even 
windows ignores the file endings, it just does not show them by default).
it is possible that you have to call the script from a html-page inside xitami.
if all of the above does not work, you should check your installation and 
maybe even check if you have saved the script as plain text or not.

Good luck,

Hugo
