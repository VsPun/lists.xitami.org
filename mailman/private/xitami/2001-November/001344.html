I've been using LRWP for a couple of years and it works fine.
To answer your questions:

1. Yes, you should certainly make the LRWP programs executable.  You will
need to run them.  They are not executed by Xitami.  They connect to Xitami
via a socket (port xx81).

2.  I don't think it matters where you put them.  Mine are in a directory
which also contains the Xitami directory.

3.  I don't know about WBA, but look in Xitami.cfg.  LRWP is probably already
enabled by default

If you try out the examples I don't think you will have any trouble.  The only
thing I had to do special was to modify lrwplib.c somewhat to make it more
"COBOL-friendly", since most of our stuff is COBOL.

Good luck.

Russell
