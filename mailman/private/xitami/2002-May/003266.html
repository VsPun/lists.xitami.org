At 08:53 AM 5/28/2002 -0700, Gunnar Swan writeth:
>Am I hearing you guys say that the client FTP can dictate the ports that
the server uses for FTP ?

The client can make suggestions to the server as to what mode to use.  It
all depends on the server implementation.  Try going to a DOS Prompt and
running straight FTP commands to Xitami.  Particularly, play with the PASV
command which toggles passive mode (if the server supports it).  For
transfers through a firewall you probably want it off.
