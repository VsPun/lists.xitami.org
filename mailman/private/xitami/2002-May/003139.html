Nico Berg wrote: 
> Because I asked for the options to compile PHP with for Xitami, nobody
> reacted. So, am I the only one using Xitami, Linux and PHP in one
> combination? I can't believe it.

I did some testing with this a couple of weeks ago.  If you're using Linux,
then run apache, since PHP was designed to be an apache module anyway. 
Xitami uses PHP as a cgi application, whereas apache runs it as a module and
parses PHP code much more efficiently.
 
> Is there someone who can tell me how to compile Xitami the right way so that
> I can use it with Xitami?

Xitami needs no special options at compile time to use with PHP.  When you
compile PHP, it has to be compiled as a cgi app, not an apache module.  The
PHP docs are quite specific about how to compile PHP.
