If you're concerned about FTP security here's something I've been 
using for a while and seems to work fine.

http://safetp.cs.berkeley.edu/

It is a secure wrapper for ftp servers and ftp clients that adds a 
secure layer to existing applications.

Setting the FTP server and the wrapper it's a bit tricky, you'll have 
to change xitami ftp port settings among other things but the 
documentation is clear.
The client install is a snap.

Everything will of course run slower because of the encryption but 
that's the price we have to pay. There are different levels of security 
though, at least the passwords are not sent in clear anymore.

Good luck
