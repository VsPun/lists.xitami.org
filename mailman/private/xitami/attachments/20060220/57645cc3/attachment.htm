I was sitting here reading through the entire online manual of Xitami and
found a very interesting entry I had not paid attention to
previously.<br><br>
The maual reads...<br>
<i>&quot;32-bit executable .exe files. The files do not actually need an
.exe. extension; Xitami will execute a file with any name so long as it
is in the CGI directory and 'looks' like an executable file (precisely:
it must start with the letters &quot;MZ&quot;) &quot;<br><br>
</i>If a CGI is a 32 bit exe-cutable and is not named
