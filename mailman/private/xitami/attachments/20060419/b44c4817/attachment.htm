Your DNS server should route the traffic<br>
your Xitami should be on a machine on the outside of your fire walled
router or a DMZ machine<br>
Then you can skip the double router step of the apache hack.<br>
All IP to Domain name is routed through, say BIND 9.3 or different DNS
solutions only.<br>
Xitami only handles Virtual host routes on that same IP to specific
directed domains, still routed with DNS albeit virtual.<br><br>
