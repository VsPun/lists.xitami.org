Would it be possible to have a Bulletin Board to post Xitami questions to?  
It would benefit the xitami user community in three ways:  1) users wouldn't 
continuously get emails filling up their mail boxes 2) there would be a 
history/record of all posts so people asking questions could do a search on 
their question before asking it and 3) responses to a question can be 
contained within a single post so the answer(s) is part of the recorded 
post.

In a sense, Xitami be building a knowledge base built by its own user 
community.

Heck, you could even link the Xitami help files right into the site as an 
additional reference.

Comments?
