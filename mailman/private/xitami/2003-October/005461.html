I think you've missed the point; a user should NEVER have to change
browser settings so as to be able to access any particular site: rather
the site's designer has a duty to write HTML which works with any
browser.  A browser's settings are supposed to allow the user to express
his/her preferences about presentation[1]: they are NOT there to excuse
the shortcomings of the web site itself.

Besides, making Mozilla use HTTP/1.0 protocols will block access to the
vast majority of other sites nowadays that host multiple virtual servers
at one IP address.

[1] E.g. some people are poorly-sighted, and hence want to override the
site's settings and use large fonts.
