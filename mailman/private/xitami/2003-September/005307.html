Hello !

There are two windows version "platform I" (win 9x, win Me) and "platform 
II" (win NTx, XP).
The difference between the two is that platform II has full support for 
services.
The windows 9x version will run on all platforms, the NT version that runs 
as a service needs platform II. This version is much more stable.
