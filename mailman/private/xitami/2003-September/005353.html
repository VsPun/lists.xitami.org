"H.W Overman" <euman@bellsouth.net> wrote:

> Im not sure you have given thought to Xitami running well on any other
> browser than IE6+. I have Netscape 7.1 and it takes virtualy forever
> for a single 6k page to load and my PHP site is all screwed up with
> the new version under Netscape. I would assume Opera will have the
> same issue as it too runs Mozilla 5+.

We use Mozilla mainly, MSIE in some cases, and we test regularly with 
Opera and other browsers.

I'm surprised to hear that your web site works differently with the 
new version of Xitami, but it is possible.  The timeout problem 
sounds like your page (PHP) is not providing a valid content-length 
header.  You might want to (a) disable keep-alives, (b) enable 
debugging and take a look at the log files to see exactly what 
headers are being sent back to the browser.

If you find something specific that Xitami is doing wrong, we'll fix 
it.
