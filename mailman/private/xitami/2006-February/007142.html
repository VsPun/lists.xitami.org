Hi Mark,
as php is running as cgi and using the php.exe I assume it will not
matter. I am still running php 4.3.11
To see if it's working create a file with a text editor like notepad with
the following, you can cut and paste

<?php
phpinfo();
?>

name it phpinfo.php. Make sure that you have the correct extension .php
not .txt
After that, call the file from your browser and if it's working correctly
you should get a purple page with lot's information.

>   In the defaults.cfg or xitami.cfg add the following settings after

>   =================================================================================
>  Hi,
>
>   here the settings for php.
>
>   [Filter]
>       .php = "D:/PHP/php.exe"
>
>   [Mime]
>       Php = "application/x-httpd-php"
>
>   In the filter section note the forward slahes. I had to use those,
> backslash didn't work for me.
