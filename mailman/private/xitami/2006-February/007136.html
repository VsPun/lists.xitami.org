Defaults.cfg is an identical copy of xitami.cfg

Just copy and paste it in the same directory, where also your xigui32.exe 
is located.
That should resolve any issues, in fact the need for a defaults.cfg is only 
apparent if
you ever need to reinstall xitami or if you are using virtual servers 
hosting and have to
manage more than one hosted section.

In all other cases it is only a recommendation.

The same for defaults.aut, otherwise it's in xitami.aut
