Some attempts to block such downloads can only be done in HTML code with 
the pragma meta tag for NOCHACHE however it doesn't work really well 
because of how different browsers read the code.

The best protection from download, is to not put the documents in the 
public directory at all.
If you need them online for your internal use, try keeping a copy only in a 
password protected folder and not in the public path.
