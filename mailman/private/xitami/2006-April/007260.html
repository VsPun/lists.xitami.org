  Are you newbie to perl, or just Xitami?  If new to perl, why not use PHP instead?

  I know, not what you asked, but was just wondering.

  For xitami, you can either set up a filter to run perl or php cgi scripts or put them in a directory that contains "cgi-bin" . By putting them in the "cgi-bin" directory xitami will try to execute it.

  Here is a link to the documentation for cgi
  http://www.imatix.com/html/xitami/index5.htm

  Or you can do it with filters. For example to run php programs I put this in my defaults.cfg file: If you don't have a defaults.cfg. Make one in the  Xitami base directory. Mine is "C:\xitami"
  [Filter]
      .php = "C:\\php\\php-cgi.exe"

  For perl use
  [Filter]
      .pl = "C:\\perl\\perl.exe"

  Note the TWO backslashes. adjust the filters to point to whreever you installed your perl.exe to.

  Here is the documentation on the filters section:
  http://www.imatix.com/html/xitami/index4.htm#TOC34

  You should also be able to execute scripts from a command prompt using  "perl somescript.pl" (I think) The output will go to your screen. When Xitami runs it,  it will capture the output and send it back to the broswer that requested it.
