Create (or edit if it already exists) a defaults.aut in the Xitami
directory (c:\xitami\ )

In it make sure there is a section
[/admin]
  userid = "password"

Where userid and password are what you want them to be. If you copy
precisely these two lines into your defaults.aut then you can access
the admin panel by
http://userid:password@localhost/admin/
