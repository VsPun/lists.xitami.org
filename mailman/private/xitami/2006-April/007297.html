If I understand you correctly you want people to be able to view these
PDFs but when someone clicks a link to one their browser prompts them
to download it instead of displaying the file. If that is the case
there are a few things to review.

1) The server needs to be setup with the correct MIME type. See
http://www.imatix.com/html/xitami/index4.htm#TOC45
The proper MIME type for PDF is application/pdf
Without that most browsers with the exception of IE on Windows (and
possibly Mac) will try to download the file becuase they don't know
what it is.
2) The browser viewing the site must have the ability to view PDFs.
This is a plug-in that is optionally installed when you install the
Adobe Reader on your computer and most browsers do not have it
installed. Modern browser if supplied the correct MIME type will
usually ask the user if they would like to attempt to open the file in
the appropriate file viewer though.

I hope that clarifies things a bit.  :)
