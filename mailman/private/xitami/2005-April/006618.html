Ok I have Xitami with PHP and ZEND on my thumb drive. I went through the PHP
files and stripped out everything I didn't think I would need. MySQL is a
different story, I think MySQL will have to run off of the computer you
attach the thumb drive too. Because I never got anyone to tell me what the
minimum install was for it. By the way I told PHP to put the session data on
c:\temp\ which might be a good idea for the Xitami log files as well. Unless
you want to turn off logging. Most Windows PCs have a c:\temp\ directory.

Oh yeah, my Xitami thumb drive with PHP and ZEND totals 15.9mb. It could
probably even be smaller once I tweak it a little more.

Ron

I use Xitami on a thumb drive.  I don't autorun it or anything, I insert
the thumb drive, and then run Xitami.

I move it around the office, and use it to demo web sites that I work on
to the folks I work for.  I don't have to worry about DNS like some others
would, since everything is http://127.0.0.1, and all of my web pages use
relative addressing.  Whatsmore, the computers are static IP, so I can hit
them from somewhere else with the IP in the URL.

Works like mud, i.e. smooooooooth.  Ron, I'd be interested in your
successes with php!
