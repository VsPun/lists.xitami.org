  Mine is strictly experimental and I've not had time to check it out very 
much, but I wanted to try to set up a simple "free" mail server on my xp, if 
for no other reason than to give my site, and those that help with my site, 
a "site-branded email addresses" so I went with the ArGoSoft mail server, 
and it seems to be right on!  Very simple set up, free version, small but 
powerful, and will work with current email client; ie: outlook ex. etc. You 
will need the Pro version for an IMAP module, but it's not much, and 
ArGoSoft is a fully functional SMTP/POP3/Finger server "for Windows" which 
is a major plus for me!   I hope this might be useful to someone, and 
although I've not had time to work with mine much at all as far as security 
etc; goes, if anyone wants to take a look at the webface or even try it out 
you can check what I've got so far at: http://www.sorcep.com:8010  and you 
can get your own or check it out further if you'd like at: 
http://www.argosoft.com/mailserver/

   thanks guys; take care!   harth
