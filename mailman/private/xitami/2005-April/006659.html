Hello !

A nice utility to check the headers of transmitted files:

http://afr_upd.verfaillie.be:20800/cgi-bin/gap?www.verfaillie.com	
or
http://web.verfaillie.com/cgi-bin/gap?www.verfaillie.com

replace the query string with the domain [:port] [directory] [file]

I wrote it first to check if my home server was responding (using the the 
corporate server as "client"), but it can also be used to show the headers.

More info:
http://freedns.afraid.org/forums/viewtopic.php?t=2077
