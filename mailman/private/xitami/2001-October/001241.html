You can't :-)  Windows doesn't support file permissions like that.  Any
program, and any user, can do anything they like under Windows.  So just
ignore the instructions about needing to change file permissions;
they're for Unix/Linux (and to a lesser extent Windows NT which uses a
slightly different model).
