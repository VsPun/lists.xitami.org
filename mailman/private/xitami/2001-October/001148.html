I know you have tried this once before, but..

Make a backup of just your configuration files. Shut Xitami down, delete the
contents of the directory you installed Xitami into. Restart your machine. In
your system32 directory (usually c:\winnt\system32) delete xiwinntc.cpl and
you'll have killed all the remnants of your previous install.

Install Xitami (2.4d9 NT service version) and copy your configuration files
back into the directory you just installed Xitami into.

I'm running the NT service version of 2.4d9 on both a Windows 2000
Professional machine and a Windows 2000 Server machine here without a hitch,
so I know it works.
