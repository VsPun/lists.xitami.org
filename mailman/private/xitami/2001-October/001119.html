>I'm having problems when I try to combine and alias with
>a filter to a php file extension.

They don't mix properly at all.  This is a known bug with Xitami.   At
present it's probably best to avoid mixing aliases and filters, and use
something like virtual hosts instead of the aliases (or just copy the
files into a suitable directory which doesn't require an alias).
