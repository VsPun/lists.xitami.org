> Linux server ... unable to execute Perl CGI scripts.

Does it have Perl? 

Where is it? 

What is the first line of your scripts? 

The first line of Perl scripts to run on Xitami cross-platform must be 

#! /usr/local/bin/perl

Or whatever the exact path to the Perl program is. 

With Xitami there is no need to change that line for Windows. Even
though Windows rarely has a /usr/local/bin. If the #! line has "perl" at
the end, and the directory does not exist, Windows Xitami will search on
PATH to find Perl.
