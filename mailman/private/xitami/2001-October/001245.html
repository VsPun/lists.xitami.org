I had this problem with Mozilla 0.95 on Linux; since Netscape 6.1 is a 
modified version of an earlier Mozilla build, it makes sense that you get 
the same problem. Disable Javascript in defaults.cfg by editing it with 
Notepad (I think German Windows calls it "Editor") and adding this:

------------------- begin
[Console]
    Javascript = "0"
------------------- end

The WBA should work nicely after that. It works for me, anyway.
