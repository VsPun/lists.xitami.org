>From what has been said on this board before, Xitami should work fine on XP.

If you are an avid Front Page user, then I am not sure that you will be
happy with Xitami.  Xitami is great, but it does not work with the Front
Page features:

1.) It does not support FP extensions such as THEMES.

2.) It can not update the site using the FP usual site update features since
Xitami will not open the site as an HTTP request.  You will have to publish
it locally, and then copy it over manually.
