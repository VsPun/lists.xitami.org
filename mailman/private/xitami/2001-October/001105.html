I think this is due to the disagreement between Xitami and PHP about how
the URL should be split up into extra path information and filenames,
etc.  ISTR last time it was discussed it was decided that Xitami was
wrong, but there hasn't been a proper release since then, so it's not
fixed.

The XiFusion addon (see http://www.xitami.com/addons.htm) apparently
includes a work around for this.
