    -----Original Message-----



    Well to be frank... you have a problem.
    Since xitami never did support asp code you will never be able to run an
asp page on your xitami server.
    I uses the Ultradev to develop asp apps myself but had to trade me
xitami for an PWS :(
    I have heard roumers about numbers of "asp-addons" event for the xitami
but have not so far found any that did work.
    If you want to stay with the xitami i recomend PHP as the language,if
not get pws (WARNING that 1 is EXTREAMLY unstable under W98) Better get W2k
and iis5 if you really want to work with aspcode.
