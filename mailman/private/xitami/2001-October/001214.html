> 	Can i not use http://www.website.classroom for example.

Get your server's IP address (192.168.0.1 or whatever - run winipcfg or ipconfig, depending on your Windows version).

On the client PCs, type in the URL http://<server IP address>/whatever/, for example http://192.168.0.1/ for the index page or http://192.168.0.1/xitami for the Xitami online docs or whatever.
