If you want Xitami to use the QUERY_STRING environment variable (as 
opposed to passing parameters in STDIN) then you should enable the 
relevant parameter

In the Web based admin this is the "Pass QUERY_STRING?" checkbox in the 
CGI properties page or in the defaults.cfg file have a line of 
form-query=1 in the [CGI] section

You may also wish to provide the URL arguments - in defaults.cfg this is 
enabled with a passargs=1 line.

Francis
PS RTFM -http://localhost/xitami/index4.htm  in the CGI section for 
other relevant info
PPS in 2.5b1 and 2.5b2 the ONLY way to get arguments to the CGI program 
was via QUERY_STRING. If you use perl's CGI.PM then you have to manually 
enable examination of the qeury string (its only semi documented in 
CGI.PM how to do this)
