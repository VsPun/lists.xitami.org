>I want to customize the graphics of the Xitami WBA interface.
>Since I can not find the HTML files responsible for all the pages (Admin,
>Configuration, VHost, ect.) then I am assuming they are dynamically
>generated.

They are dynamically generated in part; most of the text for the
webpages is in the xiadm??.h files.  This is generated from the
xiadmin.fdl file, into a bunch of fml files, which are then generated
into the .h files.

The necessary tools for doing this are available from the Xitami.com
website (http://www.xitami.com/addons.htm) in the SDK.  The rest of the
source is in the source kits also available from the Xitami.com website.

>Is Xitami using a Style Sheet to define color schemes??

No.  It's a really good idea, but the HTML for the admin screens
predates the adoption of a common style sheet language, so it hasn't
been updated to use them.

Updates (to the FDL file) to use a style sheet would be appreciated.
