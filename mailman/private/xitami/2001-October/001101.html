IIRC Xitami 2.5 (the beta version) includes support for registering with
dynamic DNS servers; it needs a ddnsdef.xml XML file which describes how
to register with the dynamic DNS service.  I suspect this may have been
omitted from the binary packages, but there should be a copy of it in
the source packages available from xitami.com.

>What can be the problem and how can I solve this? The problem is that every
>change I make in the html-admin pages, are not 'saved'. 

The dynamic DNS settings?  Possibly there is a problem with saving
those.  Or are all the settings not saved?

>Is the configuration stored in xml files? I thought it was stored in
>xitami.cfg?

No, the settings are stored in xitami.cfg, defaults.cfg, etc as well as
the aut files.  But some extra information is stored in XML files (like
the dynamic DNS registration information) because they're very flexible,
and there's already code to parse them in Xitami.
