In message <66137859576.20010930112121@iglobal.com>, xitami writes:
>As a very happy, long time user of Xitami, I accidentally upgraded/installed be
>ta 2.5b1 on our win2000 box (latest SP and patches).  

I'd suggest at least using 2.5b5 (the current beta release) if you do
want to use the beta version.  2.4d9 (the current release version) is
more stable.

>2. Can I load a production version *over* the beta without introducing
>disaster?

You're probably best installing the production version in another
directory and then copying over the configuration files, webpages, etc,
to keep things clean.  In theory it might work to install 2.4d9 over the
top of 2.5b5 but you'll end up with a bunch of extra config files that
don't get used, and it'll be a little confusing.
