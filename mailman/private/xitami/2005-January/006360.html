here some lines of my txt file which is working here at my place

webmask= !211.*
webmask= !216.239.244.98
webmask= !216.136.85.161
webmask= !217.236.57.66
webmask= !218.150.163.30
webmask= !221.232.128.2
webmask= *

# 195.92.95.61 netcraft
# 209.237.238.*  ia archiver

even the comment lines are okay in there.  Just make sure that the last entry of all lines 
"webmask" is set to webmask= * like in my sample. Xitami reads the file from top to bottom, each 
line separate, and to ba able to allow all other addresses except the ones with !  the entry has to
 
be there. In other words: first all disallowed IP's and then free entry. 
Hope this helps
