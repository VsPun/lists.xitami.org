When i try to deliver XML files, the brower get the message '501 - not implemented', even if i provide a mime content-type in default.cfg.
I had to use a custom extension to force Xitami to display the page.

Example, not working
http://afr_upd.verfaillie.be:20800/sitemap.xml

Example, working, using 'wrong' extension and content-type mapping
http://afr_upd.verfaillie.be:20800/sitemap.xmt



For those interested, i provide a webserver-test (to check that your server is responding to the internet):
http://afr_upd.verfaillie.be:20800/svt/svt.htm
Enter the domain name or ip of your server, the port number, eventually a resource file and select one of my servers.
