I don't know a WHOLE lot about web design, but I use Dreamweaver to do a 
lot of mine.  I have found that the following code:

<div id="Layer1" style="position:absolute; left:8px; top:11px; 
width:533px; height:146px; z-index:1"><img src="/global/clear.gif" 
width="400" height="100"></div>

Will place a gif in on a "layer" overtop of my image.  This gif is just a 
clear image, the same size as the picture.  I can see through this image, 
but my right-clicks register on the clear image instead of the picture 
behind it.  Therefore, any right (or left) click views/saves the clear 
image.

I haven't played with it a lot, and it doesn't address the web cache 
issue, but there ya go.  :-)

>> You can use a simple JScript code that disables the typical visitor from
>> doing a 'right-click' to save images like this:
>>
>> Paste this code into the HEAD of your HTML document
>>
>> <HEAD>
>>
>> <SCRIPT LANGUAGE="JavaScript1.1">
>>
>> <!-- Begin
>> function right(e) {
>> if (navigator.appName == 'Netscape' &&
>> (e.which == 3 || e.which == 2))
>> return false;
>> else if (navigator.appName == 'Microsoft Internet Explorer' &&
>> (event.button == 2 || event.button == 3)) {
>> alert("Sorry, you do not have permission to right click.");
>> return false;
>> }
>> return true;
>> }
>>
>> document.onmousedown=right;
>> document.onmouseup=right;
>> if (document.layers) window.captureEvents(Event.MOUSEDOWN);
>> if (document.layers) window.captureEvents(Event.MOUSEUP);
>> window.onmousedown=right;
>> window.onmouseup=right;
>> //  End -->
>> </script>
>> </HEAD>
>>
>>
