Hi Nick,
=20
take a look at your system folder in the systems32/drivers/etc
directory. I'm not completely sure about XP, but there should be a file
called hosts, which is sth. like the local SOA and has entries like
=20
127.0.0.1       mylocal.machine.net
127.0.0.1       vhost1.local.net
127.0.0.1       vhost2.local.net
=20
If you're using a proxy for your internet connection, you have to add to
add these domains (*.local.net and *.machine.net) to the "don't use
proxy for these sites" list in the browser you're using
(tools/connections/proxy settings or similar - I have a german version
here, so I don't know the correct english terms)
=20
good luck!
Thomas
