Have you tried WBSP (WhizBase Server Pages)?
It works with Xitami with no problems, web administrator can set different
setting for different virtual hosts, it is free for .org.xx and edu.xx
domains (for commercial use it costs 49.95 per domain) and it can be used
for many purposes (not just forums). Using WBSP you can create forums, CMS,
shopping carts, shareware libraries, mailing lists and many other things.
IMHO it is very good solution for building dynamic webs, and I believe that
anyone can create his first WBSP-powered database project in less than 30
minutes (with existing database, of course).
More info (and free download, with no forms and registration) can be found
at www.whizbase.com

Hope this helps.
