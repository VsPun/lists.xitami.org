At 23:44 18/07/03 -0700, Gunnar Swan <GunnarSwan@practicetopass.com> wrote:
>is not the cgi-bin sufficient ?

Using web admin:

Main screen "default - server properties": check the main CGI directory 
(should be something like cgi-bin, but you can enter an absolute path)

CGI screen "defaults - CGI properties": check that CGI is enabled.

That should do it.
Dont forget: if you change anything, SAVE it (on each screen) and at last 
RESTART the service.


Hope this will help.
