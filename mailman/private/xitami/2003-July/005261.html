Sorry if this is a stupid question, but did you set charset properly in
<head> section of your web page?
e.g.
<meta http-equiv="Content-Type" content="text/html; charset=[character set
code for your language]">

This, of course has nothing to do with Xitami, but it be the problem if you
did not set it. I use Xitami with non-international characters and I never
had any problems nor I had to configure Xitami to use them. And I also use
dynamic pages (web application) written in WBSP (www.wbsp.com) and still
have no problems with those characters.
