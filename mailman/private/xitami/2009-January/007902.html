At the time I was deciding which webserver to use, I knew Visual Basic
(whichever version at the time) inside out.

Support for VB EXE's as CGI applications seemed to be much more supported
within Xitami (with code examples - which actually worked!
http://www.clifton-fields.co.uk/cgi-bin/showprod.exe ). I'm not too sure if
Apache support them now, but Xitami is what I coded them in originally, it
does the job well and I for one am sticking with it!!

In answer to your question though Pieter, I think that the documentation for
making compiled VB applications work under Xitami was a major contributing
factor of me choosing Xitami.

If you need any help writing the doc for this new release - I'll sure lend a
hand!
