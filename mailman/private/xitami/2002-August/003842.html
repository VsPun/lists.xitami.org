At 12:54 AM 8/17/2002 -0500, Neil Finlayson writeth:
>Ok i want to use PHP to connect to the MySQL server to read a database that
>has customer details in it. Does that help? i have PHP installed correctly
>and that runs fine. I have a PHP script that was wrote to use a MySQL
>database but when i try to connect i get the error: MySQL Connection Failed:
>Can't create TCP/IP socket (10106).

You need to install MySQL as a service (just installing the software isn't
enough).  From the command line under NT (default install):

C:\>_
C:\>cd\mysql\bin

C:\mysql\bin>mysqld-nt --install
...Something about installing as a service...

C:\mysql\bin>_

Then try the PHP script again.  Note that this is an off-topic issue and
the request should have been placed elsewhere (hence the reason for the
initial confusion).

If you are running under 95/98/Me, then there is some way to run it in the
background, but I've never done it.

Hope this helps!
