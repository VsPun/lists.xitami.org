Hi Ayhan

XP has 2 hosts files.

C:\WINDOWS\SYSTEM32\DRIVERS\ETC\HOSTS
and
C:\I386\HOSTS

I think that windows reads C:\WINDOWS\SYSTEM32\DRIVERS\ETC\HOSTS on 
stratup then reads C:\I386\HOSTS for each new conection. I keep both the 
same. Here is a sample.

#------sample--------
#local only
127.0.0.1       localhost 
127.0.0.1       privetpage.test
#Default IP & Host Names for pc and laptops
192.168.1.7     PC1              
192.168.1.9     PC2
192.168.1.3     Laptop1
#Web sits on LAN
192.168.1.3     domain.test
192.168.1.3     john.domain.test
192.168.1.3     jane.domaine.test
192.168.1.9     janes.desk
192.168.1.9     webwork.janes.desk
#Favorites made faster
216.109.118.71  www.yahoo.com
#-----end sample-----
