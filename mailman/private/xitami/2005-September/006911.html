You could just have index.html (the default page) forward the user to the
script by putting 
	<meta http-equiv="refresh" content="0; url=/cgi-bin/upload.cgi">
in the header.

Might be a work around if for some reason, the default page can't be a
script. Maybe Xitami checks if the
file actually exists before referring the user to it.
