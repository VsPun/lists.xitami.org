Do you have the same problem if you try to download the file form a local

If this fails too, then the problem will probably be the server computer.
Xitami is known to have a weak ftp server: it's included, but don't rely on it.

Are you allowed to use your ADSL connection for running servers?
Here, most adsl connections don't have a fixed ip and well known server 
ports are blocked.
Did you try to use your webserver with an alternate port? (anything between 
10000 and 32000 is fine)

Just some ideas to begin with.
