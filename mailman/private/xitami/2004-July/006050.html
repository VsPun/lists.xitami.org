Hello Tom,

I'm not familiar with XP, I run NT4, but as far as I have seen/heard XP has the ability to run 
services compared to 98SE. If you are working with the installer package, you have actually 2 
options to start xitami automatically
a: put the file xitamigui32 in the autostart folder; with user logon it should start automatically
b: install xitami as a service with the file xisrv32
for b: do the following
    1.copy the file xicpl32 to the Windows sytem directory
    2. open a dos box
    3. enter the following command after the prompt : xisrv.exe -i <enter>
    you should get a message which said that the xitami service is installed
    4. define the start options of the service in start > settings > service

That should be it. I hope you get xitami started.

Johannes
