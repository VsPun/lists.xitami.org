I run my own DNS server on the same machine which resolves different domains
to different IP addresses. A good easy to use DNS server is DNS Plus and if
you are on a dynamic IP you can use DNS2GO.com or 2myDNS.com. So as my ISP
changes my IP, daily I might add, DNS2GO and 2MYDSN update their DNS Server
for my new IP Address. So if someone enters www.mydomain.com, or
www.myotherdomain.com or 123.mydomain.com. it will naturally go to the IP
address of my computer. Now what does my computer do with it. That's where
DNS Plus takes that domain name and resolves it to an internal IP which
Windows 2000 Pro can have hundreds of virtial IPs. So it goes like this.

Enter www.mydomain.com > 2MYDNS.com says go to my PC at 200.111.222.111 >
DNSPlus gets it and says go to 192.168.1.12 > which my Xitami default.cfg
says this 192.168.1.12 = "mydomain.cfg" and thus viola the www.mydomain.com
index.html page displays.

If someone enters 123.mydomain.com then DNSPlus may send it to 192.168.1.13
which the Xitami default.cfg says 192.168.1.13 = "123mydomain.cfg"

I know there are other ways to do this but this works well for me. I can
host many VHosts.
