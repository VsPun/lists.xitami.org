Ok guys... one step closer to the sorting out all of the nasty bits, odds
and ends.

Many asked before me, I'm sure, if we can run ASP on Xitami without
having to hassle through IIS and ASP.DLL and so forth.
Everyone here surely knows that VBScript and VBS for Applications is the
root to ASP 3.0
Now, many also say, Yes we can if you pay this and that much for a
parser.


Well... we're one step closer to the FREE way of doing it.
At least it seems to work on my Win 2000 Pro box with nothing but Windows
Script Engine 5.6 freely available from MS web site.
Download and install the Scipting engine, unless you already have it.

-----------
Add the following to your MIME specifications in the DEFAULT.CFG file

[Mime]
# >> Added by MRJ Design
    Vbs="application/x-vbs"
    Vbs="text/vbs"
    Vbs="text/vbscript"
# << Added by MRJ Design
# If you want to make it appear as if ASP is being enabled on your domain, you could
# also add a line saying
    Asp="text/html"
#It's cheating but looks kinda cool.



====================================================================
Now create a file named TEST.HTM and add the following lines.........................................
====================================================================

<HTML>
<HEAD><TITLE>A Simple First Page</TITLE>
<SCRIPT LANGUAGE="VBScript">
<!--

Sub ConvertTemp()
   temp = InputBox("Please enter the temperature in degrees F.", 1)
   MsgBox "The temperature is " & Celsius(temp) & " degrees C."
End Sub

Function Celsius(fDegrees)
   Celsius = (fDegrees - 32) * 5 / 9
End Function

Sub Button1_OnClick
   MsgBox "Mirabile visu."
End Sub

Call ConvertTemp
-->
</SCRIPT>


</HEAD>
<BODY>
<H3>A Simple First Page</H3><HR>
<FORM><INPUT NAME="Button1" TYPE="BUTTON" VALUE="Click Here"></FORM>
</BODY>
</HTML>

====================================================================
End adding here, and restart/reset your Xitami...
Surf into your page http://127.0.0.1/TEST.HTM
====================================================================

Now, if this doesn't look and feel like VB to you, then I don't know...
