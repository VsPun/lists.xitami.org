I'm also using Xitami for years. It's the only server that runs nearly 
out-of-the-box.
When i install the server at some customers, i know that it will work, and 
if not, it's not webserver-related.

We use it at our company (2 servers) and at home on a dynamic IP (where i 
use it for remote monitoring and controling of my central heater). It 
displays a real time high res graphic of the temperature (outside, 
basement, living, sleeping room, hot water, ...)
Before going home, i log in to my server to switch the heating on. Not cold 
enough to start the server yet.

Tried Sambar, compared to Xitami it has lots of features, but it's not so 
easy to configure.
