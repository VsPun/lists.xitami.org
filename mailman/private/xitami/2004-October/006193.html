Here's one solution to make this work in Xitami and ASP and this time I 
mean PURE ASP.
It has ONE drawback and that is you have to specify a port number in the URL.
=================================================================================
1) Install Xitami and all things as you like them set up.
Now go back to the installation disk for Windows 2K, 2k3 or selected flavor.

2) Install IIS 5 or 6 of your choice.

3) Enable ASP pages and set permissions to enable ASP to execute as needed.
Script permissions with read permissions.

4) Now set IIS to open on Port 79 and specify the home page directory to be 
your Xitami/webpages/ directory

5) Point ASP queries to Yourdomain.com:79/YourPage.asp and voila'... it 
works :)
=================================================================================
