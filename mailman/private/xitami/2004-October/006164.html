Hello !

Any cgi program has the same right as the webserver.
Under windows nt5, you can create a specific user with limited rights and 
run the webserver as this user.

First create a new user with the correct access rights, then change the 
Xitami rights to that of the user (start/programs/administrative 
tools/services, and locate Xitami somewhere at the end of the list, right 
click on the log on as and change to the new user)

You should not allow other people to run cgi programs, unless you secure 
your server.
It's never a good idea to let others upload programs to the cgi directory. 
It will always be a security risk.

marc
