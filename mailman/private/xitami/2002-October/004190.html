we're running into problems due to some items in a conf file that is needed
when run under apache... is there an equivalent for each of the settings in
the the following?:  The ScriptAlias equates to [CGI-Alias], we think the
Alias item equate to [Alias], but the SetEnvIf and SetEnv are questionable:



Alias /dev60html/ "c:\ORACLE\DEV6i\tools\web60\html/"   
Alias /jinitiator/ "c:\ORACLE\DEV6i\JINIT/"   
Alias /forms60java/ "c:\ORACLE\DEV6i\FORMS60\java/"   
Alias /dev60temp/ "c:\ORACLE\DEV6i\tools\web60\temp/"

ScriptAlias /dev60cgi/ "c:\ORACLE\DEV6i\tools\web60\cgi/"


# Forms and Reports CGI needs to have ORACLE_HOME set
SetEnvIf Request_URI  "rwcgi60"   ORACLE_HOME=c:\ORACLE\DEV6i
SetEnvIf Request_URI  "rwcgi60"   RW60=c:\ORACLE\DEV6i\report60
SetEnvIf Request_URI  "ifcgi60"   ORACLE_HOME=c:\ORACLE\DEV6i
SetEnvIf Request_URI  "ifcgi60"   FORMS60=c:\ORACLE\DEV6i\forms60

# Forms CGI needs location of formsweb.cfg file
SetEnv   FORMS60_WEB_CONFIG_FILE
c:\ORACLE\DEV6i\forms60\server\formsweb.cfg
