I don't know if this was answered.

If PHP's register globals is off (now the default), the link you used can
only be called from within PHP. This is to prevent PHP variable pollution.
If you must have this type of link called from outside PHP, you have to go
into the php.ini file and turn register globals on. I highly recommend you
read about "register globals" on the PHP website.
