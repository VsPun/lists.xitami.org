>> Good day all... will CSS work with Xitami (Win) ?
>> Is so, what directory does the CSS file go into?

 > Works great, because it's just another file for Xitami
 > to serve.

 > For example, create a folder named "stylesheets" just
 > below your webroot, then put this in the header of
 > pages that use the stylesheet:

<head>>
 > <link rel="stylesheet" type="text/css" href="/stylesheets/yourstyles.css">
 > ...
 > ...
 > </head>

 > Put the appropriate CSS stuff in "yourstyles.css" and
 > that's all there is to it.


 > -Jeff
