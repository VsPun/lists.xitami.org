Works for me in MSIE 6.0 and in NetScape 4.7.

Do you have it embedded in a public web page that we can see? It may
work different inside an IMG tag than called directly.

NS47 says "has the following structure:

     http://www.furtopia.org/cgi-bin/counter.cgi?0
          Image: http://www.furtopia.org/digits/4.gif
          Image: http://www.furtopia.org/digits/7.gif "

And for the first image:

Location:  http://www.furtopia.org/digits/4.gif
 File MIME Type:  image/gif

So either you are sending a correct MIME header, or NS is making the
right guess based either on extension or the header of the actual
file.
