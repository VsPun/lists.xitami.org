Just go into the 'Configuration' menu from http://localhost/admin, and =
then it should show a page with many text boxes. The first three are the =
ones that you want to focus on. The first one called 'Main HTML =
directory' is your webpage directory. Change the one called 'Main CGI =
directory' to the 'Main HTML directory'. Next, change the 'CGI URLs =
start with' to /. Finally, press Save at the top left and you'll be =
brought to the main menu. The last step is to press Restart in the top =
right corner. It should say 'Server is restarting...' and then you can =
start using your cgi files in your webpage directory, or any sub-folders =
in it, once you see that message.

If it still doesn't work, try right-clicking on Xitami and press =
terminate. Then you need to start Xitami again.

I hope that works for you! Email me again if still have problems.
-Greg

  It depends on what you mean by a whole lot of sites. If you are just =
using one configuration, but you want many directories that you want to =
have cgi access, you just put the cgi-bin directory to your webpage =
directory. If you have virtual hosts, instead of clicking on =
configuration, you need to click on virtual hosts, then on the =
configuration that you want to change, and then you can change the =
cgi-bin directory. If you mean something different, just email me. Hope =
that helps!
