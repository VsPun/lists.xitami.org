In message <2C61CCE8A870D211A523080009B94E43071C78F9@HQ5>, "Joshi, Sunit" writes:
>1)Create a virtual host called PyCgi so that it points to E:\\Sunit\\XMLFiles\\
>PyXml\\Scripts
>2)Be able to point to http://localhost/PyCgi/PyTest.py and runmy python scripts

You want a CGI-BIN directory.  It's going to be called "PyCGI" in the
URL, and have the appropriate directory as its on-disk location.  You'll
also need to enable index-pages-in-CGI-directory.

Something like:

[server]
  cgi-bin=E:/Sunit/XMLFiles/PyXml/Scripts
  cgi-url=/PyCGI

[cgi]
  mixed-url=1

You may also want to define some more default filenames to apply when
accessing a directory.
