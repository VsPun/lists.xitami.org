To me, it sounds like you're allowing anonymous ftp connections. You can =
set up ftp users in ftpusers.aut, or by pressing Define... next to User =
configuration file under FTP. Web Browsers by default try anonymous ftp =
first. You can try ftp://admin@domain.com and that should tell the =
browser to use the name admin and then it should ask for a password.
-Greg
