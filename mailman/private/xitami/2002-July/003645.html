It sure is. I am running a web server with 8-10 domain names at www.trushosting.com. 
First you either need to signup for DNS services. I would suggest www.myserver.org. All 
the help you need is on their site. Second register a domain name, I use 
www.DomainWizardry.com. When you sign up for a myserver.org account your will be given 
name servers, e.g. ns.myserver.org etc. When you signup for your domain name you will 
need to add those name servers. That is pretty much it. There are alot of other ways to 
doing this including running your own DNS instead of myserver.org, but it is much more 
complicated.

Hope this helps,
