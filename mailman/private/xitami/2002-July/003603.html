>> Xitami has a known crashing problem with heavy server usage when the FTP
>> server is enabled.  Xitami also has a serious DoS/DDoS issue with large
>> numbers of CGIs running under Windows (and slows to a crawl with heavy CGI
>> usage under Linux).
>> 
>> The solution is to get a dedicated FTP server.  The FTP server in Xitami is
>> not meant for heavy-duty use and tends to crash the server.  Xitami without
>> FTP is very stable, but you need to be picky in the CGI arena (e.g. PHP)
>> till that gets fixed.
