Even if I log into the admin-pages, go to logging and
press the 'cycle now' button, I get the following
error (shown in red in top of page)

"Log file is not active"

What does this mean? 

In my xitami.cfg I certainly have:

[Accesslog]
#   Specifies whether logging is enabled or not. 
    enabled=1
#   Specifies the name of the log file. 
    filename=access.log
    local=1
    format=CLFX
    custom=
    translate=1
    cycle=daily
    cycle-how=rename
    cycle-time=00:00
    cycle-day=0
    cycle-size=60
    cycle-lines=0
    cycle-arg=ac%yy%mm%dd.log


In other words: how is it possible the accesslog does
not cycle anymore?
