Sounds like xitami is either not marked as executable or your path
information is set up incorrectly.  I'm going to assume the latter as it is
more likely (you can edit your .bashrc file later to add the proper
information to the path and change other default bash settings).

Try ./xitami from the usr/xitami-24d9 directory.

If xitami is not marked as executable (if it isn't, there is probably
something wrong with the build), try chmod 700'ing the file.

Hope this helps!
