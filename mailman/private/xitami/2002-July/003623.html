When you installed Xitami, you were prompted for some install folder
(usually c:\xitami or c:\program files\xitami). Inside the install folder
there is, among other things, a subfolder called webpages. Anything you
place there can be accessed from the entire world, which include your public
library. For example if you create a page called test.html and place it
there, you can access it remotely using http://YourServerIP/test.htm
I suggest that you rename your test page as index.html which enables you to
access it only by http://YourServerIP
you'll probably find out later why.

I guess this proves that you can further create your own webhost.

Good luck.
