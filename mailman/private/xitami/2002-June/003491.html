Create an NT user account
Go to service panel, assign this user to Xitami
On the C:\Xitami Folder right click and set permissions.
The folder where your web pages are, the PHP EXE and your PHP scripts, set permissions to this Xitami user.

A good book on NT security is ... " Securing Windows NT/200 Servers"
Works for NT Workstation too.
It goes step by step, avoiding all the theory that fills pages.
ISBN 1-56592-768-0
Published by O'Reilly
Author Stefan Norberg
