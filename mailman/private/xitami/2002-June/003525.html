I wouldn't install it on the USB drive, but that's just me.

You can change the path of the \xitami\webpages\ directory by opening the
xitami.cfg (located in the xitami directory) and assigning the server
locations to your e:\ drive. This way it recognizes the server as your e:\
drive instead of the default location.

If you do this there is no need to redirect your users between drives.

That's just my thoughts.
