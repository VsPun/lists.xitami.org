> eliminate your headaches of getting to XitamiPro

What? I thought XitamiPro is a native WinNT/2000
service application? I hope so -- I just paid for it a
couple of days ago but haven't had the chance to
set it up yet.

=== quote from readme file ===

    Open a console box and cd to the Xitami/Pro directory.  To install the
    Xitami/Pro service, type:

        xiprosrv -i

    To run it without installing it (nice for testing), type:

        xiprosrv -d

    To uninstall it, type:
    
        xiprosrv -u

    Finally, copy the xiprocpl.cpl control panel into your Windows system
    directory if you want interactive control of the service.
=== end quote ===

Sounds like a service app to me....
