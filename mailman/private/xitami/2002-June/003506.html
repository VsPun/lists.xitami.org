Hmm...I didn't think Xitami built with anything but Visual Studio/Visual C++.

Actually, the target is _almost_ correct.  The correct file should be
wsock32.lib for Borland products.  If you don't have wsock32.lib in the
lib\PSDK directory of your Builder install, you can create one with the
command-line 'implib' utility Borland provides (although, only the latest
Builder products comes with the fastdef utility which optimizes the
resulting .lib files).

Hope this helps!
