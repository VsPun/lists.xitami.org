Shining Light Productions is having a **HUGE** software release party and
you are all invited to come and download XiFusion v2.6!  XiFusion allows
Xitami and ColdFusion to work together (and helps PHP work better)!

However, when Shining Light Productions says, "party," the Company really
does mean a party!  Not only is XiFusion v2.6 being released, but 5 other
applications and Open Software Debugging Functions are being released as well!

While you head over and grab XiFusion v2.6, be sure to check these out as
well:

Nuclear Vision v1.0 (An easy-to-use, HTML-style scripting language)
GetWebPg (Grabs webpages via the command-line - optionally displays headers
too)
SMTPSend (Sends e-mail via the command-line)
POP3Get (Retrieves e-mail via the command-line)
QuickFTP (A one-shot command-line FTP client)


Also, if you are a CGI developer (or do any type of C/C++ programming), be
sure to download the Open Software Debugging Functions!  They will aid you
in pinpointing application crashes as well as find memory leaks and profile
your code!

Cool toys await you at the Shining Light Productions software release party!
