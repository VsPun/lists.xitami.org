If you're running PHP I suggest upgrading that to latest set.
We just had another site go offline becuase someone infected and took over
SMTP on a server with PHP and a basic forum.
If you run volic forum you should get their latest patch for this as well
and upgrade your MySQL files.

Apparently the system can be taken over and start running smtp bulk mailing
from infected machines.
Not sure if it applies to your system but worth a checkup/notice.
