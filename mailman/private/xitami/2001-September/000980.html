What version of OS/2 Xitami are you using?  I also had crashes about every 2 days (give or take).  I found 
the following 2 items:

1) There is a patched version of Xitami on Hobbes (http://hobbes.nmsu.edu) bos225b4p2.zip which fixes 
some problems in (what is now) the current OS/2 Xitami release.

2) Setting the EMX env var in config.sys to:   SET EMXOPT=-c -h256 -!0x800
(This fixes some file handle issues in the OS2 EMX runtime)

With both these in place I've been up and running for 4 days now with no abends in Xitami (A minor record 
for me )
