This is fixed by the same name/nameurl fix that ewen mentioned fixing 
the double extension issue.

You need to modify the httpdir.gsl (or better copy it, modify the copy 
and use the copy instead of the original) in the templates directory.

Line 89 is in the original form:

<A HREF="$(urlbase:)$(nameurl:)"><IMG SRC="$(icon)" 
BORDER=0>$(name:)</A></TD>

You should change this to :

<A HREF="$(urlbase:)$(name:)"><IMG SRC="$(icon)" BORDER=0>$(name:)</A></TD>

Personally I think this is the correct behaviour. Nameurl seems to be a 
bit too escaped to be useful for anything
