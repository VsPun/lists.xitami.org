In message <Pine.OSF.3.95.1010904180450.23203A-100000@www.bit-net.com>, John Hol
t writes:
>Can Xitami deny serving outside links to my files?
>In other words, I don't want other web sites to link to my images and suck
>up my bandwidth. I only want my pages to link to the images and such.

You can "sort of" do this by having a program (eg, filter, CGI, LRWP,
etc) that checks the HTTP referer header to see if the thing that told
the browser to fetch the file was on one of your sites.  (Xitami itself
cannot do this directly.)  However the HTTP Referer can be faked fairly
easily, so it's not very strong security, and some things don't send it
at all so may be locked out.
