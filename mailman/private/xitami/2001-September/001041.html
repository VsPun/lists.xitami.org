 Joubin.

Paulo is right. If you uncheck the preview pane you must doubleclick the =
message header before it opens. If the preview pane is on, the message =
will pop up immediatelly in it. This is not a good thing if you want to =
delete a message. You have to click it first. If you have selected to =
show preview pane the message will appear there as soon as you click it =
once. With preview pane off you can delete message without opening it. =
This way it is possible to get rid of messages that look suspicious.
