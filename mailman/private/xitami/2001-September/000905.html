you can use an empty web root for whatever.com and configure to use gsl
error-script which will produce an error page with redirect header included.
in this case request for any page on whatever.com will produce 404 error and
request for / (i.e. root of whatever.com) will produce 403 error. Both
should generate an error page with redirect header.
