what I've done, is set up aliases to point to all of
my drives.  For example, the alias /a points to a:\
/c points to c:\
/d points to d:\
/e points to e:\
/f points to f:\
/g points to g:\

All aliases work fine except /d.  The only thing I can
figure out is that since xitami is installed on that
drive, it can't see past it's root.  If I make that
alias point to d:\xitami , then it will list the
directory fine.  Also, if I create a mirror of the d:\
drive, and map it to something else (/h -> h:\ for
example) then it will work fine.  I'm thinking that
this may be the only solution.  But if anyone else has
an answer, I'm listening!
