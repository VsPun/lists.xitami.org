We (programmers in a windows environment) use the word 'thread' for 
multiple processing.
http://www.powerbasic.com/support/technote/multithreading1.asp

Streams on NTFS volumes are something else than extended attributes, they 
are just an alternate file content. The main file content (the one you see 
when you open the file in a word processor) is the unnamed stream.
http://www.powerbasic.com/support/forums/Forum4/HTML/003677.html

Both links contain very intersting info!
