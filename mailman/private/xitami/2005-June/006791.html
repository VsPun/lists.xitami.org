I hold  alot of users/folders under Win2k3 NTFS.  NTFS uses a btree (I 
believe) so it's pretty good on bulky directories.  You can increase 
performance two ways through windows.  Use a directory tree index scheme 
to limit your folder/files per directory or (and I think the bigest 
performance gain for you) would be windows disk Cache.  Windows disk 
cache comes from "unused" memory so the more free memory you got the 
more file/directory tables will be able to be held in memory.  This lets 
the system bypass hard drive tables and access directly from RAM which 
is much faster.  My main server has 1 gig of ram and I will upgrade it 
to 2 before long.  Win2k3 also has faster file access then win2k.
