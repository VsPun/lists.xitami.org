Okay, here's a shot in the dark:

I've been able to get alias to work.  My configuration:

(Defaults.cfg)
[Alias]
photos="D:\Photos"

*D: is a mapped network drive, not the server.

(reference)
src="photos/skyline/album/index.html"

You've referenced the info.php file as a foreign file, not as a virtual
local file.  I've seen where the reference for something
"file:/something.ext" for local stuff but never can remember when to use
what.  All I recall is that using the "http" sent me nowhere.

Hope this will can helps your problem a bit.
