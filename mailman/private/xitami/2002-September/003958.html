> xitami running on xp, which i use only for serving large mp3 files
> of about 80-200 megabytes in size. Is there a way of configuring
> to xitami handle these files more so than normal use. Or will
> xitami work weell out the box?

Any machine that can run XP will serve big files via Xitami VERY
well.

I had Xitami running on a 386SX-16Mhz, with a 15 Meg file for test
downloads, and people outside campus could not tell it from a
Pentium server. Xitami is very efficient, and the network is usually
the bottleneck.
