1) Go to Ericsson's web site and join their developer group.

2) Download the FULL WAP/WML SDK (any one will do).

3) When you install, you will notice a Xitami WAP server app.

4) It was included to aid in testing of apps.

This is how I found out about Xitami in the first place. If they

no longer include it, I'm sure that I can find the "old" copy.


Here are a few links to editors(free):

http://www.wapdrive.com/DOCS/wap_software/waptor.html

http://www.inetis.com/santanabuilder.asp

There are quite a few out there, but the best ones come with

the SDK's(unless you pay (OracleMobile, etc.)).


Hope this helps,
