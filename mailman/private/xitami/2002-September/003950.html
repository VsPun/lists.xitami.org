On Mon, 9 Sep 2002 04:32:04 -0700, Latishia Anders wrote:

>I would like to use xitami to host to websites, I tried installing a second
>version on Xitami, but I receive an error about the port it is trying to
>use.  I would like to know how to set up vhost so that I can have the two
>sites.  Can anyone give me  the long version of how this is done, I'm new to
>all this.  Thanks

don't install multiple versions of Xitami.

xitami.cfg/defaults.cfg

[Virtual-Hosts]
  www.myhost.com=c:\webpages\myhost.com\vhost.cfg

vhost.cfg contains only the settings from xitami.cfg as needed.

like

[Server]
  webpages=c:\webpages\mhost.com\htdocs\
  log-dir=c:\webpages\mhost.com\logfiles\
