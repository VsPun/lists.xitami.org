Don't install a second copy of Xitami, since both programs will try to use
the same ports.

STEP 1:
In the file default.cfg add: (virtual host name=config file name)

[Virtual-hosts]
     Mail.verfaillie.com=mail.cfg

STEP 2:
Create a mail.cfg file containing the path to the root directory of the VH:

[Server]
     webpages=webpages/mail                   #   Relative or full path

All these steps can also be performed from within the webadmin, if enabled.
Go to Xitami administration, click on virtual hosts, click new, enter
identifier name, the webpages root dir, save and that's it for simple VH.
