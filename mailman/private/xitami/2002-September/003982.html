Hi,

It's really easy to do this.
Just edit your Xitami.aut file and add some entries like this:

[page1_folder]
	team1=passw1
	realm=page1_folder

[page2_folder]
	team2=passw2
	realm=page2_folder

etc.

Then create the corresponding folders page1_folder, page2_folder, etc....
Optionally you can put an index.htm in these folders so when going to

http://your-site.com/page1_folder you'll get a default page after the
username team1 and the password passw1 have been entered.


There are some default entries in xitami.aut which you might want to delete
(#[Private], Jacky=robusta#    Jonas=realtime  Sarah=arabica etc.).


Good luck

Stefan
