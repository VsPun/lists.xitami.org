The 'lcc' error sounds like the configuration is set up incorrectly for
Visual Studio and is looking for a Linux compiler of some sort.  'cl.exe'
is the command-line compiler for MSVC++.  You might also need 'ml.exe'
which can be found in the Windows 98 DDK (the DDK is free).
