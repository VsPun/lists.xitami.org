Most of the info on this site is new to me also.  Yes, it is that simple to
run your own website with xitami.  It really does install and run with no
additional configuration.

I advise a persistent connection, like cable modem or ADSL.  In my case I
have ADSL thru mindspring.  Because my IP address is dynamic (changes all
the time) I subscribe to TZO, a dynamic DNS service, to make sure my domain
name can always be resolved with the constantly changing IP address.  I
believe Xitami may offer this service also, although I'm not sure.  Anyhoo,
once you have a simple website functioning you can begin to mire yourself in
the config and admin details.  Hope this helps.

Jill
