When you say "infected" I assume that you consider it a virus - code red is
not a virus, it is just a simple command line (as seen in the log) sent to
exploit a particular security hole that exists (that is if you have not
applied the patch/fix) for Microsoft Internet Information Server (IIS) 3 and 4
I believe. So nothing has been "infected" -- at least not from the Code Red.
But watch out for the W32.Sircam.Worm@mm virus. Supposedly it only gets 98 and
ME boxes via mail readers like Outlook Express, or Outlook. You can read about
that and Code Red at http://www.sarc.com

-J
