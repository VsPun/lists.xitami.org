I think that local LANs are expected to have the IP range 192.168.0.xxx
by default. you might try that. for the router/firewall I use the IPs
are obtained automatically with the DNS disabled, WINS using DCHP, and
no gateway defined. The router does all the work and pulls the numbers
from the ISP.
