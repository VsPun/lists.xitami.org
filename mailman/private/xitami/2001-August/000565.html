Hi Andreas,

   I've found http://www.kessels.com/Upload/index.html to be a very
   simple one to integrate. I've been using the Windows version on W2K, but
   Mr. Kessels also provides the UNIX source for download. I just prefer
   executable versions to the general PERL CGI HTTP upload versions that
   can be found at any PERL CGI distribution site as I generally like
   single purpose tools rather then general scripting engines which seem
   to me more prone to general hacking if not properly secured, though
   allowing any CGI execution adds risk.
