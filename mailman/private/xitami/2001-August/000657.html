It's not real difficult to run your own basic Web server for serving up
pages. Xitami runs with no configuration. Be careful on what machine you run
it on though, for security reasons. TZO, will work for you if you have
dynamic IP addressing. If you have static, you could use zoneedit.com and
edit your own zone (domain). You can run you own mail server too. I use
Mercury/32, www.pmail.com .

-Rich
