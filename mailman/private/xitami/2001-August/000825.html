In message <3.0.6.32.20010828131842.00b64900@137.82.27.14>, Gordon Wong writes:
>For CGI-BIN directory I want to use the common
>cgi-bin directory in c:\xitami\cgi-bin so I entered "/cgi-bin".

If you are configuring the "cgi-bin" setting (which is the actual
directory you want to use), and you enter "/cgi-bin", then Xitami will
expect that you mean "/cgi-bin", which under Windows translates to
"c:\cgi-bin" (assuming you're on C: drive).

Since that wasn't what you wanted, I'd suggest you enter either the
directory you wanted "c:\xitami\cgi-bin"; or leave it at the default
setting of "./cgi-bin", which means that providing you run Xitami from
the Xitami directory.

The cgi-url should be set to "/cgi-bin", if you want "..../cgi-bin/..." 
in your CGI URLs; this too is the default.

The defaults really are intended to be sensible, ya know :-)
