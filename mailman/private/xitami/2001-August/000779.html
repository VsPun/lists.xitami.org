>I'm trying to get my Perl scripts to work with the xitami web server.
>I can get the test one to work (testcgi.exe) but nothing else.

testcgi.exe is an executable program, not a perl script.   If you're
having trouble running perl scripts (a) make sure you have Perl
installed, (b) make sure Perl is in the PATH, (c) make sure your
perl scripts start with something like #! perl, and (d) check cgierr.log
for any error messages.
