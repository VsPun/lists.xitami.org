The browser assumes no such thing.

The web-server has a default action if a URL points to a Directory 
instead of a File.

Xitami comes pre-loaded to look for the usual default.htm and index.html.

You can add index.php to that list.

Then a call for "../bulbs/" (a directory) is fulfilled with
"../bulbs/default.htm" or whatever action you have set up.
