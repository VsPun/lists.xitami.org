As I mentioned previous - the only way another support forum (usenet, irc, 
web forum) will work is if its endorsed by iMatix.  Everyone has made great 
contributions to the discussion, but without Xitami Support group backing I 
don't think its going to matter a whole lot.  A web forum probably makes 
MOST sense (though others would disagree) and the iMatix folks can download 
free ones from various locations.  It doesn't have to or not to be hosted 
anywhere else besides the Xitami site.

But what ever the case, I think whats important is the iMatix folks finally 
take the initiative to improving the format of this Discussion Group.

IMHO  :-)

PS: to the twit who moaned about "not using xitami anymore" due to this 
current topic of discussion - I have been running Windows versions of Xitami 
for over 6 years now.  the only times it has crashed has been 1) a perl 
error which caused an infinite loop and hosed the xitami cgi and 2) I 
somehow corrupted the Xitami config file that prevented to from running at 
all.  So as you can see, it has been user errors that have caused Xitami to 
crash, not Xitami.
