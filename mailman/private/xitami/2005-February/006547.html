FTP now working!   BUT!

First note, I have a dynamic IP.
To get FTP server operating I had to put the external IP in the default.cfg.

In addition Ports 8020, 8021, and range 8020 - 8030 were openned in the 
router.

manually edit the default.cfg every time my dynamic IP changes.  Not a good 
solution.
Here are my default.cfg

[Ftp]
    enabled = "1"
    Limit = "25"
    Pipe = "64k pipe A"
    User-file = "ftpxxxxts.aut"
    ipaddress = "63.163.33.138"
    force-ip = "1"
    passiveip = "63.163.33.138"

Please commit if you have a solution for dynamic IP other than the current 
hard IP I have to use.
