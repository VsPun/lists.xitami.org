Steps;<br>
1: download the zip from <a class="moz-txt-link-freetext" href="http://www.xitami.com/download.htm">http://www.xitami.com/download.htm</a><br>
2:&nbsp; unzip the downloaded file,<br>
3: Move the xitami-25&nbsp; folder &amp; all it's contents to the C:/ drive.
I put a link to the folder on the desktop for easy access<br>
4:&nbsp; Put a shortcut to xigui32.exe on your desktop.<br>
5:&nbsp; run it &amp; go through the test pages,<br>
6: Right click the green circle with the X on the right side of your
tool bar to stop the server.<br>
7: Rename the "webpages" folder to "oldwebpages", don't delete it, the
help file are in there.<br>
8: make a new webpages folder &amp; put your new index.html file &amp;
any new webpages &amp;&nbsp; folders in this folder.<br>
9: the help file are in the xitami-25/app/oldwebpages/xitami&nbsp; folder.<br>
<br>
You're in business have fun.<br>
