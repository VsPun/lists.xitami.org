Hi, I'm currently using the following setup for my logfiles:

[Accesslog]
    Filename = "xyz_de_access.log"
    Cycle-arg = xyz_de_access_%yy%mm%dd.log
    Format = CLFX

So that I will get a logfile for every day. That works fine but
increases the number of files for several domains quite fast. Perhaps it
would be better to have one logfile for each domain where new entries
are concated too. So I tried:

[Accesslog]
    Filename = "xyz_de_access.log"
    Cycle-how = concat
    Cycle-arg = xyz_de_access.archive
    Format = CLFX

But this doesn't work. Any idea where the problem is? Robert
