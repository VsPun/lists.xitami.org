>Hi! I want to protect a directory with a user/password.
>I've tried the .htaccess / .htpasswd method but it don't work.
>If anybody knows how to use this method or an other one, answer me....

Very easy;

First go to the webadmin interface and "security"
Check the password filename (assumed default.aut)

Second
In the root directory of Xitami, open the file default.aut

Each directory to be protected has an entry

[directory_name]
     User_name=user_pasword

You can have more than one directory to be protected, and more than one 
user with different password per directory.
