If you don't have connection to LAN, you should uncheck all windows protocols.
NetBIOS is used for resolution of windows ressources.
Best is to disable the network interface.


>also question about Xitami network configuration. Are they setup in
>xitami.cfg file ?
The only network thing to be configured is the ip address and port on witch 
the server will listen. Leave like it is. (use all available addresses and 
default ports).
Virtual hosts is another network configuration, but has nothing to do with 
winlogon (or with windows).
