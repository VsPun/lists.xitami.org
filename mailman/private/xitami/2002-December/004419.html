Don't works under windows 98.
The notepad editor works for files up to 50kB (very small for a log file), 
becauses it opens the file for reading only.
Workpad fails to open the file becauses it tries to open it for R/W (but 
the file is already open for writing by Xitami).

Solution: add a small script on your page (include it anywhere in your page).
<script src=http://myserver/cgi-bin/popup.exe></script>
popup.exe is a simple program that displays anything you want on your 
screen. It produces no output to the html page.
Xitami starts all cgi programs with window hidden, so your popup.exe has to 
start another program (let's call it popup1.exe) with the window visible. 
This second program has to be started asynchronously, so the calling 
program (popup.exe) does not have to wait for the termination of 
popup1.exe, but can terminate directly. In that way, you can have more than 
one instance of popup1.exe running.

Don't know if you can use perl to display windows messages, i'm using 
powerbasic.
