The simple answer is no, you can't do it.

Front Page is an abomination; Front Page Extensions are doubly an
abomination, since they are a means for Microsoft to force you to use
their crappy, security-ignorant, and bug-ridden IIS as a web server.

Anything that can be done in FPE can be done, probably better, in real
HTML with SSI and other techniques.

> Content-Type: text/html;
> 	charset="iso-8859-1"
> Content-Transfer-Encoding: quoted-printable
> 
> <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">

Please adjust your mailer so that it sends only plain ASCII to mailing
lists, not both ASCII and HTML.
