You don't need Sendmail.  What you need is the the Perl Package Module: 
"libnet".  In libnet you wil find library functions ranging from SMTP to 
SNMP.

Assuming you have Perl running on your system, go to the Perl directory and 
execture the PPM.bat file.  In the PPM window, type "install libnet" - 
assuming that computer has a connection to the internet, ppm will go out and 
auto install the libnet module.
