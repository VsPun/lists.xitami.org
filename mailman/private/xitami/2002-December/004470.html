Some time ago I wrote an url file (Internet shortcut) parser to use with
Xitami.
http://home.fortech.lv/~andris/redirect/

To use - add in [Filter] section 
    .url=url.exe
I see it usefull to add in [Server] section default for index.url files as last
one.

Do not use hyperlinks to clean .url file because it does not work in Netscape
4.xx browsers.
