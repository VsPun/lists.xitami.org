I hated 2.5b5 because it kept crashing on me, and wouldn't recover.

This is what I found:

- The configuration files from 2.4d9 are not the same format for 2.5b5
In 2.4d9 you see in the config files:
      Autostart=1

In 2.5b5 you get:
      Autostart = "1"


- Check that all hosts are using the same setting for "Use standard 
I/O?"  I found that some of my virtual hosts were one way, some were 
another.  Find the correct one for your CGI-script, and make them 
consistent.  It was the biggest source of my crashes.


Now that I've discovered those two problems, Xitami has yet to crash on me.

Hope this helps,
