straightforward.

install xitami.
install mysql.
install php.
find php.ini and customize the mysql section accordingly.
run xitami, goto the admin screens, enter a filter like .php pointing to

/path/to/php.exe.
restart xitami.
run your board setup scripts.

apart from the time needed to download the binaries (which depends on
your 
bandwidth) the whole setup will take you no more than 30 minutes.
