I'm running Xitami 2.4d9 on Windows XP Home.  I don't
think that matters, though, I think its that I'm
running Java 1.41.  I couldn't get Sokrates to run out
of the box as I had once before, I believe on Windows
2k with Java 1.3.  The runner.cmd batch file has a
line that begins by trying to invoke "jre".  jre is an
unrecognized command under Java 1.41.  See the docs
at:
http://java.sun.com/j2se/1.4.1/docs/tooldocs/tools.html

Editing runner.cmd, and substituting "java" for "jre"
did the trick though.
