Hello !

For those interested in using CGI, there is a little informative site located

<http://web.verfaillie.be?webserver> or <http://web.verfaillie.com/webserver>

(both url point to the same server, but using different connections: cable 
versus adsl)

It is mostly basic stuff for those who never used CGI, but want to give it 
a try.
All the examples are in Powerbasic <http://www.powerbasic.com> but can 
easily be modified to Liberty Basic, Visual basic and so on.

Some topics:
* a basic "hello world" program
* configuring Xitami
* writing fast programs as back end
* talking to the user: escaping the input and html-coding the output
* files and security considerations
* some working examples: a guestbook

All comments appreciated to make this site more informative !
