In message <OE38zwCMVSCxhq2pNdX000061ec@hotmail.com>, "Just" writes:
>1. Is there a way NOT to show the page ext.? ie to show: 
>www.abc.com/home instead of www.abc.com/home.htm ?

If you make a directory called "home", then you can use:

www.example.com/home/

instead.  (And at the expense of the webserver having to tell the
browser, "hey that's a directory" you can in practice get away without
the final "/" too; there'll be a redirect to the version with the slash.)

>2. Is there a way to prevent showing the files in a directory? ie when a =
>visitor type www.abc.com/pages when "pages" is a directory?

A.  Create an index.html file, or similar, in the directory
B.  Set [Security] dirlist=0  to disable automatic directory listings
