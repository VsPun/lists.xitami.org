I've set this up as follows:

1) open an account with zoneedit.com (or similar service) and copy the
"nameservers" from the site that holds them now (namesecure, or whoever you
bought the domain name from)

2) a service like "DirectUpdate" (directupdate.net) once installed on your
machine will notify zoneedit whenever your IP changes.

Has been working fine for me on my cable modem since I set it up a couple
weeks ago, but my IP address hasn't changed in that time either.
