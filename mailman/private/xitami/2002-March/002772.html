Still the best way to work around this is to use name based virtual hosts. You may want to take a look into the manual on how to set them up.

The problem is that Xitami doesn't pass PHP the translated ("real") path to the files in the alias-dir but the alias path. Of course this cannot work as PHP doesn't know of the servers alias handling.
