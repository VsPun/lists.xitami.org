>I have set up a webserver with Xitami some time ago.
>Since then (I think) my PC has started to crash aprox 1-2 a week.
>Can this be a Xitami problem?

If the _PC_ crashes (eg, Windows blue screen) it's unlikely to be Xitami
-- Xitami doesn't do anything low level enough to be causing crashes
like that.  (Although make sure that your operating system patches are
up to date, because ISTR bugs in Windows 95/98 that would cause certain
types of file requests to crash the operating system, and these can
sometimes come in from a URL.)

Generally I'd suggest making sure you've got the latest patches, making
sure there's nothing else likely to be causing the problem running, and
then perhaps reinstalling if that doesn't work.  (I've also seen PCs
"suddenly" start crashing a lot if, eg, the CPU is overheating, or the
memory has gone bad or something, so don't completely rule out hardware
issues.)
