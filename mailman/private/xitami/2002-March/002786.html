I can repeatably get a Xitami server to crash when I hit it continuously 
with WGET running at max and hitting an XML page that runs a GSL script. 
If anyone (Ewen?) is interested in the code I can provide it.

returns a specific text block in a webpage, with the exact file and text 
block depending on the query section of the URL. Each page is on average

actually crashes. In neither case does anything go into the log file.
