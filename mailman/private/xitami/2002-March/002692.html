>Im trying to post 40 megs of data through http to lrwp.  Is there a limit
>in the XItami server that limits the size of posts to lrwp processes???

It _should_ work with 2.4d9 and 2.5b5, and I'd be interested in reports
if it doesn't.  There are bugs in all the previous 2.4d and 2.5b
versions which will cause issues with return amounts > 16KB (size where
it changed from all-in-memory to spool-to-disk).

FWIW, returning 40MB from a LRWP (or CGI or whatever) isn't all that
efficient; a 40MB temporary file will be written.  It's perhaps better
to issue a redirect to the location of the file.
