There are API's for dealing with programs that run in the tray or
as a service. 

You can deal with the Windows API directly ( what fun ! ),
or find some after market OCX type product.

Check out www.desaware.com or www.winfiles.com
You will get better results on this quesiton on a VB list.
The technique you will use will work on all services or 
programs in the 'tray'.
