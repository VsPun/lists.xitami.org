In message <5.1.0.14.2.20020303194245.00c5a458@m1.520.telia.com>, John Lawson writes:
>When I do a securityscan on my PC, port 80 is open.
>Is that correct or is ther any way ot hiding it and still have the Xitami 
>webserver running?

Yes.

You can run Xitami's HTTP server on another port using the portbase
option.

Then a different port will be open, and everyone who accesses your site
will need to use:

http://yoursite.example.com:portno/....

which wouldn't be terribly useful.
