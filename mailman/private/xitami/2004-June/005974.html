Hi Nick

Thomas gave you good advice for a lone pc. If  you are useing a peer to
peer
network and need the sites to be seen by all network peers. you should
use
the ip for your network interface card not the 127.0.0.1 loopback ip. I
copy
a master host file to all the pc's in our network.

sample..................................................................
....
.......

#IP Address     Name                  Comments

127.0.0.1          localhost              #Loop back IP/Name

192.168.0.0     desk1                    #Default IP & Names
192.168.0.1     desk2                    #Default IP & Names
192.168.0.2     desk3                    #Default IP & Names
192.168.0.3     desk4                    #Default IP & Names
192.168.0.4     desk5                    #Default IP & Names
192.168.0.5     desk6                    #Default IP & Names

192.168.0.0     company.frim       #In house IP & Names
192.168.0.0     company.com       #In house IP & Names
192.168.0.5     company.web       #In house IP & Names
127.0.0.1         testthis.test           #Local test site

End
