Hi,

I'd like to introduce myself as the maintainer of the Xitami port on
FreeBSD.

Being the maintainer means, I'll have to see to it that Xitami compiles,
installs and runs well on the FreeBSD platform. Apart from patching some
sources and providing a proper makefile, this basically boils down to
watching the xitami web site for announcements, and the rest is waiting
for problem-mails from users.

Hence, if you have any problems with or questions concerning Xitami on
FreeBSD, I'd appreciate a mail about it.

I can state that for me, Xitami runs just fine, at least the HTTP and
CGI part of it (I don't use the FTP part because FreeBSD has a FTP
server of its own). I'd like the log to be written via the syslog daemon
though, rather than directly into files.

In case you wonder what FreeBSD is, it's an open-source Unix, closely
related to NetBSD and OpenBSD; please see http://www.freebsd.org/ for
details. You find the Xitami port listet on
http://www.freebsd.org/ports/www.html .

Thanks,
	Bernd
