This is how I got Perl running in Xitami
it allows me to use,

<!--#exec cgi="/cgi-bin/test.pl" -->

Running Windows98
Xitami v2.4d6

in your
autoexec.bat

SET PATH=C:\PERL\BIN;%PATH%
SET SSI_INSECURE=YES

defaults.cfg
[Filter]
    .shtm=perlssi
    .shtml=perlssi
    .ssi=perlssi
[Ssi]
    Enabled=1
    Exec=1

hope this helps
