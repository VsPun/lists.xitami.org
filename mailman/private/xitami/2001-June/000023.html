In message <004001c0eb1a$99bc37a0$8cd4e2d1@daniel>, "Daniel" writes:
>Hello, I would like to use Xitami as well as war-ftp server.  

You can either tell Xitami not to run the FTP service at all (since it
sounds like you don't need it as you have a separate FTP program):

[FTP]
  enabled = "0"

Or you can change the FTP control port to something other than 21:


[FTP]
  port=XXX

Probably disabling the FTP service is the most appropriate way to go.
