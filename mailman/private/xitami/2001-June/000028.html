At 07:49 PM 02/06/2001 -0700, Dragonballz Insane wrote:
>I would like to know where I can get the program that
>tells the server what to do with a perl or cgi script.
>  I have been told and have experenced that when you
>get ready to run a cgi through an http it comes up to
>be an Internal Server Error can someone help me?

You don't need an external program to tell Xitami how to
handle scripts.  All you need is Perl (or the interpreter for
whatever scripting language you use) and make sure that
Xitami is setup properly.  I've been running perl scripts with
Xitami as my web server for a little under two years with no
problems.  If you're sure you have those things installed
correctly, make sure the script has no syntax errors in it;
check your cgierr.log file to make sure.  Hope this helps.
