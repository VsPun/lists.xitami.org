In message <3B1FB90D.B1AC52B@evernet.net>, david writes:
>:-) Very Small Error: Try this instead, notice the forward slash
>infront of "private"
> 
>[Alias]
>#   /private=e:\                         #   Directory/drive aliases
>#                                            #   E.g "http://xxxxx/mandy/"

I trust y'all realise that if you have a comment ("#") character at the
start, Xitami is going to ignore it as a comment?  You need something
like:

[Alias]
   /private=e:\

in order for Xitami to read it :-)
