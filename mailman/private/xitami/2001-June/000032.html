In message <002501c0ebba$7ff0b620$74b6d1d8@daniel>, "Daniel" writes:
>Hello, Is it possible to tell Xitami server to use another ftp server?

You can tell Xitami not to start its FTP server, and then run another:

[FTP]
  enabled = "0"

But you'll have to run the other one yourself; Xitami won't do it
automatically.
