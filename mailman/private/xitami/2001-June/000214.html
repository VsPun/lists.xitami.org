| it allows me to use,
|
| <!--#exec cgi="/cgi-bin/test.pl" -->
|
| This SSI command
| which is a potential security risk for your system
| is only accepted if the environment variable
| SSI_INSECURE is set to 'yes'
| Under Windows NT

This is not true for me on W2Ksp2 Xitami2.5b5.  <!--#exec
cgi="/cgi-bin/test.pl" --> will run pretty well without ssi_insecure=yes.
The latter only needed to run   #exec cmd   constructs, which certainly can
expose you to risk much more than your own scripts. Therefore, I would NOT
recommend to set SSI_INSECURE=YES
