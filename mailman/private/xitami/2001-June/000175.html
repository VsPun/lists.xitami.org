Hi!

Perhaps that info can help:

Before I started exploring Xitami on w2k I used to run GUI version of Xitami
on Win9x,ME (xiwin32.exe). There is the way to start Xitami at pre-login as
a service.
You simply hack a registry a bit. Go to
HKLM\Software\Microsoft\Windows\Current Version\RunServices. Create new
REG_SZ there. Moify it to contain C:\Xitami\xiwin32.exe (or whatever your
path will be). Bingo. Now your Xitami will start early before anyone logs in
and stay running after logout till you shut down.
Limitations: you will have no tray icon, you'll have to ctrl-alt-del/end
task the server if you wish to stop or restart (except wba) the server.

Mike.
