Many have answered the need to find your machine via the DNS with static ip's or another service.  I will just mention that I have a static ip and that the isp that registered my domains can point them to my static ip. (Yes thats right I point half a dozen at the same ip).  I then successfully use the virtual hosts in Xitami to pick up the domain name used in the url and serve up different web sites.  If anyone actually types in the numeric ip address then they get the default host pages.  It will work and Xitami has proved an excellant choice as I can now easily add my newly registered domains and have them all on the one server - even separtate log files to count the few people that visit !!!!

Yes I run it all on a Win98 pentium 160 box with 48MB ram and it stays up quite well on this platform.
