> the documentation is fairly thin on how to actually set this up. 

Open http://www.yourhost.com/admin

It wants the name and password you picked when you installed
Xitami (the name is probably "admin"). Proceed to the Admin
screen. 

Fourth button down is "VHost Wizard". Click. 

 "Virtual Host Wizard

 Create virtual host profile:  ### make up a name like "test"

 Select IP address:  ### leave blank in this case
                                                      
 Or, enter DNS host name:  ### your second domain name
                                                                                
 Web page root directory:  ### where the second set of files are
                                                                                
You can leave the rest blank. 

This Wizard really does set it all up for you: making CFG files
that point to the right place. 

Stop and re-start Xitami after the change: it is sometimes loathe
to take on a new VHost until you whack it.
