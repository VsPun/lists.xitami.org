The 501 (not implemented) error message indicates that there is no
"xisetup" that can be run.  AFAICT this is because there is no
"xisetup"; the web based admin is "xiadmin", usually at "/admin", and is
set up by the default configuration.

You can set up virtual hosts through the web based admin (at /admin).
