> Does anyone know of a stable counter I can run on my server - that I can trigger from Javascript?

There are a gazillion counters that return a graphic of a number.
I'm not sure what you mean "trigger from Javascript" -- you can 
document.write almost anything. 

I have one counter, a Windows NT EXE, that I trigger from static
HTML. It returns a graphic of a number. 

I have another counter that I call from Javascript (with a
NOSCRIPT option to catch lame or hobbled browsers). It is written
in Perl and returns a static image (generally a 1x1 transparent
GIF to satisfy the browser without showing on the page) but keeps
detailed logs of hits. Another Perl script summarizes the hits,
but that isn't made public. That's for pages where I want to know
what kind of surfers are hitting, but don't want them to see how
(un)popular the site is. 

Look on the top page of Music.rutgers.edu , low on the left is
the static-call public graphic counter. I'm looking for where I
got it from... the name in it is Henn Saar, 1995.
