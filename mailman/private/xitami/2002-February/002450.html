Alvaro E. Siman wrote:

>Can Xitami behave as a proxy server to an IIS server?
>
No. If you need a proxy install squid. A linux box can quite successfuly 
be both a Xitami web server and a Squid proxy server. It can even (if 
you so desire) transparently proxy and run Xitami at the same time.

Although they get rather stroppy with people modifying the config 
because its really a firewall, smoothwall (www.smoothwall.org) is 
everything you need except the Xitami webserver.
