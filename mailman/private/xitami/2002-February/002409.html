> The filter in xitami .php points to f:\internet\php

> test page  c:\my documents\alan\html\alan-lam\phptest.html

The ".php" filter only applies to files with a ".php" extension, but
your test file is called ".html".

The server returns the un-processed file. Because your example has
no visible elements, it looks blank.

Change that file's name to phptest.php and see what you get.
