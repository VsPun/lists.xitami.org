I dont't think you will be able to "get to" 192.168.0.50 
from the internet as this is a private non-routable ip 
address. I also don't think this is a Xitami problem. My 
guess is that you would have to define a directory alias 
for your external path or just use another port from your 
24.190.xxx.xxx address forwarded to the 192.168.0.50 
machine. For example use 24.190.xx.xxx:9090 forwarded to 
192.168.0.50. HTH
