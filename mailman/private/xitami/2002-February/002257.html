Mike,

You can go to the Linksys website http://www.linksys.com "services and
support", "knowledge base" and search for "mac address clone"

Basically, you run "winipcfg" from the start/run menu (on the server) and
you will see you your ip config data.  Select the dropdown to point to the
card your accessing by, and you should see the "adapter address" listed
first.  Write that 12 digit number down. (if your running NT, you'll need to
go dos prompt and type "winipcfg /all"

Now, call up the Linksys and go to the "setup" area  see where it says WAN
IP address?  Write that down in case you have to revert.  Now select the
"advanced" tab then the "mac addr clone" tab.  Type in the 12 digit from the
winipcfg session and confirm it.  When you call up the setup tab again, you
should see the hardware address of the netcard as the wan ip address.  You
may have to do a reset on the router, I'm not sure.

Root around on linksys' web sight and be patient.  Work through logically
and use "ping" to tell you if your getting through at all.
