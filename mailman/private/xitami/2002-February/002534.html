kewl, rebuilding the config file took two seconds....Xitami discovers the
individual *.cfg files for each domain.  So after wiping out the
defaults.cfg and then going to the browser based admin page, Xitami still
saw all my different domains.  All I had to do was recopy the Xitami.cfg
file over to a new version of the defaults.cfg file and then edit the VH
section.  17 individual lines and its all back online now.

I never had to go through these processes before so some of the above
revelations you guys have probably already had - first time for me.
