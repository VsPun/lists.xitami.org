    I have an internal Mandrake/Apache behind a Linksys. No DMZ, just port
forwarding (port 80).

    I also use www.dhs.org to have a symbolic name that I can give to
people, which points to my Linksys IP. So, when they use http://.... it will
go to my internal Linux.

    Of course the symbolic name does not work internally (intranet) because
it will point to teh Linksys, and I end up in teh Linksys admin. So, it will
only work from teh outside. Internally I use teh IP http://192.168.3.200

    I never cfg'ed DNS etc.

marcio
