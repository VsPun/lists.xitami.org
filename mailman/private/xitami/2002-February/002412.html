At 04:24 PM 2/12/2002 -0800, Gunnar Swan writeth:
>	
> Good Day,
>
> Assume I run an uninstall on a service (which was shut down).
> The uninstall does not remove the service from the Control Panel.
> How can I remove this service. The EXE has gone away.
> I'm ending up with two services, one of which I disable (the old one I
want to go away)

Start a Command Prompt (Start->Run, type 'cmd', press enter).
Go to the Xitami directory:

C:\>_
C:\>cd progra~1\xitami

C:\PROGRA~1\Xitami>_

Now, launch Xitami and tell it to remove itself as a service:

C:\PROGRA~1\Xitami>xiwinnt -uninstall
Xitami web server v2.4d9 (c) 1991-99 iMatix
Xitami is free software and comes with ABSOLUTELY NO WARRANTY.
You may redistribute this software under certain conditions;
read the file COPYING for details.  Type xiwinnt -h for help.

xiwinnt: stopping service 'Xitami'... Ok
xiwinnt: service 'Xitami' removed

C:\PROGRA~1\Xitami>_

That's it!

Hope this helps!
