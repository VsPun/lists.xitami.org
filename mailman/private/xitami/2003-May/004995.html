Ok! I got it figured out.
The way to start a service from the command line in win2k is "
net start Servicename
Stopping is:
net stop Servicename

In this case, that is
net start "Xitami Web Server"

Throw the delayer in there and you have
delayer.exe "net start "Xitami Web Server"" /D20

BUT! Double quotes don't work. So I wrote a batch file called xitami.bat
which contains
net start "Xitami Web Server"

So my final shortcut which I placed in Startup reads
delayer.exe "xitami.bat" /D20

Our course with appropriate paths.
Finally got it!
