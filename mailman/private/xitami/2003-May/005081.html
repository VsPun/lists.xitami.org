What operating system are you running?

If it's Windows <something> then you probably need to set up a HOSTS file in
your c:\windows directory  (there should be a sample hosts.sam there
already).

Create an entry in the file that identifies the name of your server and it's
internal IP address, for example:

192.168.0.100   MyIntranetServer

Reboot your system and try pointing your web browser to
http://MyIntranetServer and see what happens.

Good luck.
