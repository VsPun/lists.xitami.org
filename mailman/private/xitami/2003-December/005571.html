"Charles Duval" <ch_duval@hotmail.com> wrote:

> I am trying to use a CGI to copy content on one directory to another
> on my Web server to update information on a ramdisk...

There are many potential problems, the trick is to solve them one by 
one.

Start with the simplest CGI you can make (like testcgi) and make sure 
that works.

Now add functionality, step by step, testing each time.

It is probably something simple: from your description, I guess the 
executable CGI assumes something (like current directory) that is not 
correct when running under Xitami.  Try running the CGI by hand from 
the Xitami directory, and check the Xitami configuration with respect 
to the CGI working directory.
