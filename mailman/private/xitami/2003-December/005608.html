If you don't mind me offering an alternative suggestion, I prefer to use 
ImageMagick for graphic work.  You need to download the 
ImageMagick library (available as an executable for Win32), then install 
either imagemagick or perlmagick from PPM.  The module's name has 
changed, and I can't remember what it is called now, but I do know it is 
one of those.

I found GD to be weak for my purposes, mainly to create graphs on 
demand.  Now, I use ImageMagick and OLE to create the graphs.  

Check out imagemagick.org.  In retrospect, I think the Win32 executable 
also includes the perl module; so, you probably won't even need to 
install it from PPM.

Sincerely,
Brad Smith
