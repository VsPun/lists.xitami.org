Hi Bill,

I had the same problem a week or so ago whereby I had problems with my V-Host.
Im behind a NAT myself and am not sure what Cable modem users have but it
should be same or similar. (We sortof have a STATIC IP in other words.)

Ive managed to create several virtual-host by using a system that's free'ly available
called a Dynamic DDNS service here: http://dyndns.org/
(According to some KZey or ZKey site that test these type systems, the one above is best.)

If you own a registered DNS name from InterNic try using (Simple DNS Plus) here: 
http://www.jhsoft.com/

I don't know what level you're at so I'll try to start from the beginning

Start Xitami, click setup and at the Xitami Administration Window that comes up,
click Virtual Host. At the top of this screen there's a new button click that one and
add-in the account names you created at http://dyndns.org/ or with Simple DNS Plus.
Make certain that your .cfg and .aut files that correspond to these V-Host exist in
the main Xitami directory.
 
If you need further assist let us know but this should get you started.

regards,
Euman
