Assuming that you have the appropriate wildcard CNAME in the DNS for
your domain so that all addresses point to the same place then this
should be as simple as writing a (perl) script to copy a blank virtual
host directory to the one for the name and edit defaults.cfg to add
the new vhost.

In other words it should be trivial with the hardest bit being
controlling who is allowed to create the subdomain and to ensure that
only valid and sensible names are used (at least I assume your client
would prefer it if obscene subdomains were not allowed for example)
