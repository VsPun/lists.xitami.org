after running the installer you need to enable php in xitami itself.
Open up your default.cfg and create 2 new sections like or copy / paste.
The path for the php directory has to be set to yours. :)
Stop the server / start the server so the changes will take effect.
If it's not working, you maybe have to put backslashes into the php path.Mine is only working this 
way. ( xitami 2.5c2)

[Filter]
    .php = "D:/PHP/php.exe"

[Mime]
    Php = "application/x-httpd-php"
