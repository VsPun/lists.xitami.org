Make sure your xitami.cfg is set to the root of your sites. Mine is
different than the defaut xitami/webpages.

(this is the default if you haven't changed it)

	[Server]

	Webpages=./webpages

Now if you enter http://computer_name/site1.com/

site1.com is whatever you have set the directory to (in site.cfg)
