Well, don't let the name fool you.  It actually stands for Personal Home
Page.  It was originally intended to be a simple cut down version of perl so
that home page users could put some scripting into their regular html pages.
But, the implementation of it grew in popularity over the years, and
webmasters started using it.  Since it was open source, it was contributed
to quickly. And, some will argue that it is at least as powerful as Perl
itself if not more now.

At least that is what my SAMs book says about it.
