In message <200112271429125.SM00191@there>, Bob Paddock writes:
>I just buld Xitami on my Mandrake 8.1 linux box.
>When I try to start it up it tells me that port 80 and 81 are already in use, 
>which is news to me.  

If you're running a webserver on a linux system and want to use the
standard ports (80, etc), then you need to start the webserver as root
-- only root has the ability to open those ports. 

I suspect you'll find that's what's going on, rather than that there is
some other program using those ports.
