>I was wondering if someone could post examples of their defaults.cfg etc.
>that have been optimized for maximum security.  Any other tips on enhancing
>security or other places to find information would be greatly appreciated.

It's hard to be specific about what to do without knowing which
operating system you are using.  The basic rules are: turn off
everything you don't need, try to run as a user that has the smallest
number of privileges necessary to do everything you need to, and make
sure that all the files are readable only by the user that needs access
to them.

There are other things to do if you're using scripting langauges (perl,
PHP, etc), but at that point it starts getting moderately specific to
your installation.
