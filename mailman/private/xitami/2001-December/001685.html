Xitami tries to use port 81 for LRWP; but it's never actually had that
port reserved.  Perhaps somthing else is using port 81?  If so you might
want to try running Xitami with LRWP disabled, by adding:

[lrwp]
  enabled=0

to your defaults.cfg.
