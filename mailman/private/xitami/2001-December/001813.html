iHTML used to be called MivaMia and is very similar to cgi scripting. I'm
fairly sure you can get it to work with xitami (I did it back in the day when
it was called by its original name), however, iHTML is not well supported
everywhere as a programming language. There are plenty of other languages out
there that work like iHTML (but better supported) for example: C or Perl CGI
scripting, JSP, PHP,ASP, Python and ColdFusion.
