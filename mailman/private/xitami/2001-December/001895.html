The easiest way is with windows explorer (and you have all the windows 
explorer functionalities, like drag-and-drop).
(sorry for those who are using *nix)

1
Start windows explorer

2
in the address bar type ftp://aaa.bbb.ccc
if you don't have an address bar in we, go to View, Toolbars, and click 
address bar

3
now the right part of the screen is the remote location

4
If you have to log in with login and password, click on File, Login...

It's that easy!
