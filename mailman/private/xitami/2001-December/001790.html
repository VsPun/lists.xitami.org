In message <005101c1891c$91df3aa0$4abefea9@f2k7o6>, "Steve Cox" writes:
>I connect with my FTP client to 127.0.0.1 as admin with all rights, if I try
>to chmod my perl cgi file in the cgi-bin permissions to 755 for example I
>get the error message: "502 'SITE' not implemented ! chmod failed. 

chmod, etc, are not implemented in Xitami's FTP server -- it's a fairly
basic FTP server intended for anonymous-FTP and some basic webpage
uploads.

If you need more advanced features I suggest you disable Xitami's FTP
server and install another one with the extra features you need.
