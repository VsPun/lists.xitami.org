SunOS 4?  (As opposed to Solaris 2.x, 7, 8, or 9)

I'd suggest just trying the "default" gcc options (-O2 -DDEBUG), and see
how you get on.  The main time you're likely to have problems is
linking, where you may need to link in libsocket, libnsl, and other
things a bit like that.  (It's been ages since I've used SunOS 4, so I
don't remember exactly what was required.)

The iMatix "c" build script knows how to compile/link on SunOS 4 (and
various versions of Solaris), but unfortunately it assumes that you are
using the Sun compiler.
