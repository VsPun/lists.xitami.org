>Prior to v2.4d9 I was using the non-service version. Something that seems to
>have changed is that Dir-script = seems to be completely ignored now. You
>can't assign a template and  you can't use graphics in directory listings.
>Is this a bug or by design? 

As far as I know the Dir-script option was only in the Xitami 2.5 beta
(which includes GSL).  It won't work in Xitami 2.4, because the GSL
support wasn't introduced until the 2.5 development version.  In Xitami
2.4 the directory listings that are generated are hard coded.  You could
perhaps use PHP or similar instead if you want to customise things (ie,
have your own index pages).
