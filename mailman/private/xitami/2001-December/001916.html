Default will be port 21 and your ISP (must allow this on port 21)
The default port address of 21 can be changed using the portbase=3D
in the [Server] section of defaults.cfg if needed. =20

In the defaults.cfg file:   (this is the minimum requirement)
[FTP]
enabled=3D1                        #enable ftp
user-file=3Dftpdirs.aut        #point to password/authorization file

In the ftpdirs.aut file:
[MyFtp]                                    #folder name
user1=3Dpassword1              #user1 password is password1
users2=3Dpassword2            #user2 password is password2
