What do you mean by "...something fancier than CGI forms..."?

While Javascript is (in almost all cases) executed inside the browser
(client side), PHP is executed on the server (server side). So the browser
"talks" to the server in Javascript and the server "replies" in PHP (which
is a somewhat misleading description because the server only returns HTML
code which has been generated by parsing the PHP upon the browsers request
generated by Javascript on in another way).
Inside a document, PHP and Javascript are distinguishable by their enclosing
tags. So they can both, coexist and interact. It works as well (or even
better) as communication by a mixture of letters, email, fax, phone and
personal meetings.

Go to www.php.net and read the tutorial for beginners. This will help you
much more because then you can actually see PHP.

Good luck,
