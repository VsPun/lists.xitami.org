>I failed to mention that when I use www.mitchtemple.com I am directed to the 
>default xitami web directory. So yes, dns is configured correctly. 

Have you changed the webroot in defaults.cfg?  If so, there's a bug in
some versions of Xitami (including, I think, the released versions) that
means that this overrides _every_ webroot.  (It's fixed in the
development version in CVS.)

The Xitami server debugging is useful in tracking this down; to turn it
on set:

[server]   
  debug=1

then try again, then look at the debug logs to see what Xitami is basing
its returned things on.
